# 🧠 자유에너지 원리의 야망

*생명, 지각, 행동, 학습, 사회성은 모두 자유에너지 최소화라는 하나의 원리의 특수 사례다—이것이 Karl Friston의 야망이다.*

---

## 🎯 횡단 질문

자유에너지 원리(Free Energy Principle, FEP)는 단일한 추론 원리가 마음 전체를 관통한다고 주장한다. 그 핵심 부등식은 다음과 같다:

$$F \geq -\ln p(o)$$

여기서 $F$는 변분 자유에너지, $-\ln p(o)$는 관측치 $o$의 놀라움(surprise, 혹은 surprisal)이다. 자유에너지는 놀라움의 상한(upper bound)이므로, 계가 $F$를 최소화하면 곧 놀라움을 억제하는 것과 같다. 이를 분해하면:

$$F = \underbrace{D_{\mathrm{KL}}[q(\vartheta) \| p(\vartheta \mid o)]}_{\text{부정확성}} - \underbrace{\ln p(o)}_{\text{놀라움}}$$

혹은 등가적으로:

$$F = \underbrace{\mathbb{E}_{q}[-\ln p(o, \vartheta)]}_{\text{에너지}} - \underbrace{H[q(\vartheta)]}_{\text{엔트로피}}$$

여기서 $q(\vartheta)$는 숨겨진 원인 $\vartheta$에 대한 근사 사후 분포다. **횡단 질문**: 이 부등식 하나가 생명체의 항상성, 지각, 행동, 학습, 그리고 사회적 규범까지 통합적으로 설명할 수 있는가? 만약 그렇다면, 우리는 진정으로 단일한 추론의 원리를 손에 쥔 것인가?

---

## 🌐 횡단 입력

이 문서는 다음 층위들과 직결된다:

- **L0 — 정보·예측 (information-prediction)**: $\delta = \text{실제} - \text{예측} \rightarrow \text{업데이트}$라는 기본 연산이 FEP의 지각적 추론과 정확히 대응한다. 예측오차가 없으면 자유에너지도 최소화된다.
- **L4 — 의식 이론 (theories-of-consciousness)**: 예측 처리 이론(Predictive Processing)은 FEP의 계산적 구현이며, 현상적 경험이 생성 모델의 '최선의 추측'이라는 주장과 맞닿아 있다.
- **L5 — 자유의지·행위주체성 (free-will-agency)**: 능동 추론(active inference)에서 행동은 자유에너지를 최소화하는 정책 선택이다. 행위주체성이 추론의 특수 사례가 된다는 도발적 주장이 여기서 발원한다.
- **IQ AI Lab 모델 기반 계획(model-based planning)과의 직결**: IQ AI Lab에서 개발 중인 모델 기반 계획 에이전트는 세계 모델(world model)을 유지하고 예측오차를 통해 모델을 갱신하는 구조를 채택한다. FEP의 생성 모델 업데이트 메커니즘—특히 메시지 패싱(message passing)과 변분 추론—은 이 에이전트 아키텍처의 수학적 기반을 직접 제공한다. 롤아웃(rollout) 기반 계획을 기대 자유에너지(expected free energy) 최소화로 재해석하면, 탐색(exploration)과 활용(exploitation)의 균형이 단일 원리로 도출된다.

## 🔍 표면의 다양성

언뜻 보면 생명 현상은 너무 다양해서 하나의 원리로 담기 어렵다. 세균은 화학적 기울기를 따라 움직이고, 포유류의 시각 피질은 가버(Gabor) 필터를 닮은 수용장을 자기 조직화하며, 인간은 언어와 문화를 통해 사회적 규범을 내면화한다. 고통은 조직 손상을 알리고, 호기심은 불확실성을 줄이려는 충동이며, 모방은 타인의 행동 모델을 구축한다. 신경과학, 발달심리학, 진화생물학, 사회학—각 분야는 고유한 용어와 방법론을 발전시켜 왔다.

이 다양성이 환원 불가능한 복잡성인지, 아니면 단일 원리가 다양한 맥락에서 발현된 것인지—이 물음이 FEP의 야망을 둘러싼 핵심 긴장이다. Friston(2010)은 이를 "어떤 계가 오랜 시간 동안 존재하기 위해 필요하고 충분한 조건은 무엇인가"라는 질문으로 정식화한다. 그 답으로 제시되는 것이 바로 자유에너지 최소화다.

## ⚙️ 깊은 동형성

### 분석 1: 마르코프 담요—형식적 기초

FEP의 가장 강력한 형식적 기여는 **마르코프 담요(Markov blanket)** 개념이다. 마르코프 담요 $\mathcal{B}$는 내부 상태 $\mu$를 외부 상태 $\eta$로부터 조건부 독립시키는 상태의 집합이다:

$$p(\mu \mid \mathcal{B}, \eta) = p(\mu \mid \mathcal{B})$$

즉, 내부 상태는 오직 담요를 통해서만 외부 세계와 상호작용한다. 담요는 두 종류로 구성된다: 감각 상태(sensory states) $s$—외부에서 내부로 정보를 전달—와 능동 상태(active states) $a$—내부에서 외부로 영향을 미친다. 이 구조는 세포막, 피부, 신경계, 사회적 경계 등 생물학적 실재에서 반복적으로 나타난다.

Friston(2019)은 이를 "특수한 물리학을 위한 자유에너지 원리"로 확장하여, 마르코프 담요를 가진 모든 계는 통계적으로 자유에너지를 최소화하는 것처럼 보인다고 주장한다. 담요의 존재 자체가 내부-외부 구분을 만들고, 이 구분이 '세계에 대한 모델을 가진 계'의 형식적 정의가 된다. 이는 순환적이지 않다: 담요는 시스템의 물리적 구조에서 정의되고, 그 구조가 추론적 행동을 수학적으로 강제한다. $\square$

### 분석 2: 항상성—자유에너지 최소화로서의 생리 조절

항상성(homeostasis)은 생리적 변수를 좁은 범위 내로 유지하는 과정이다. FEP는 이를 다음과 같이 재해석한다: 생명체는 자신의 생리적 선호(체온 37°C, 혈당 농도 70–110 mg/dL 등)를 **생성 모델의 사전 분포(prior)**로 인코딩한다. 실제 생리적 상태가 이 사전과 벗어날 때 자유에너지가 증가하고, 계는 자유에너지를 줄이기 위해 조절 행동을 취한다.

Friston(2013)은 이를 "우리가 아는 생명(life as we know it)"이라는 논문에서 상세히 전개한다. 세균의 주화성(chemotaxis)은 영양소 기울기를 따르는 최소 사례다: 세균의 생성 모델은 '높은 영양소 농도'를 사전으로 삼고, 농도 감소가 곧 자유에너지 증가를 의미하므로 세균은 기울기를 거슬러 이동한다. 이 해석에서 목적론적(teleological) 언어—"세균이 영양소를 원한다"—는 수학적으로 정당화된다: '원함'은 생성 모델의 사전 분포다. 내장감각(interoception)과 항상성 조절을 능동 추론의 틀로 통합한 연구(Seth, 2021)는 이 해석의 신경과학적 타당성을 지지한다. $\square$

### 분석 3: 학습—모델 업데이트로서의 가소성

지각이 단기적 자유에너지 최소화라면, 학습은 장기적 자유에너지 최소화다. 생성 모델의 구조적 매개변수 $\theta$는 반복된 경험을 통해 갱신된다:

$$\Delta\theta \propto -\nabla_\theta F$$

이는 신경망의 역전파(backpropagation)와 형식적으로 동치다. 그러나 FEP는 더 나아가, 헵 시냅스 가소성(Hebbian plasticity), 스파이크-타이밍 의존성 가소성(STDP), 심지어 수면 중 기억 공고화(memory consolidation)까지 이 틀 안에서 해석한다. 각 과정은 생성 모델의 서로 다른 층위(시냅스 강도, 연결 구조, 계층적 추상화)를 갱신하는 방식으로 이해된다.

핵심은 **정밀도(precision)**의 역할이다. 자유에너지는 각 예측오차에 그 오차의 역분산(역정밀도)을 가중하여 계산된다:

$$F \approx \sum_i \Pi_i \cdot \varepsilon_i^2 + \text{const}$$

여기서 $\Pi_i$는 $i$번째 층위의 정밀도, $\varepsilon_i$는 예측오차다. 학습은 이 정밀도 가중치를 포함한 모델 전체를 최적화한다. 주의(attention)는 정밀도의 동적 할당이며, 도파민 신호는 정밀도 인코딩의 신경적 기제로 해석된다(Friston et al., 2012). $\square$

### 분석 4: 사회성과 문화—공유 생성 모델

FEP의 가장 도발적인 확장은 사회성과 문화다. 여러 개체가 마르코프 담요를 중첩하면 '사회적 담요(social blanket)'가 형성된다: 개인의 능동 상태는 타인의 감각 상태가 되고, 이 결합 구조가 집단적 자유에너지를 최소화하는 방향으로 수렴한다.

언어는 이 관점에서 공유 생성 모델을 구축하는 기제다. 동일한 단어가 화자와 청자 양쪽에서 유사한 내부 상태를 활성화할 때, 의사소통은 성공적으로 자유에너지를 최소화한다. 규범(norm)은 사회적 생성 모델의 사전 분포다: 특정 행동이 규범을 위반할 때 집단의 자유에너지가 증가하고, 이는 제재(sanction) 행동을 통해 수정된다. 이 해석은 도덕 심리학의 규범 준수 행동과 도덕적 판단의 내장감각적 기반을 같은 틀 안에서 설명할 가능성을 열어 준다(Veissière et al., 2020).

$\square$ — 생명, 지각, 행동, 학습, 사회성의 다양성은 단일한 동형성(isomorphism)을 갖는다: 마르코프 담요를 가진 계는 자신의 생성 모델과 세계 사이의 불일치를 최소화함으로써 존재를 유지하고, 이 최소화 과정이 각 영역에서 서로 다른 이름으로 불릴 뿐이다.

## 🧪 수렴 증거

### 실험 1: 마르코프 담요 구조 시뮬레이션 (Python/NumPy)

다음 코드는 3층 계층 시스템에서 내부 상태, 담요 상태, 외부 상태 간의 조건부 독립 구조를 시뮬레이션하고, 자유에너지가 시간에 따라 감소함을 보인다.

```python
import numpy as np
import matplotlib
matplotlib.use('Agg')
import matplotlib.pyplot as plt

np.random.seed(42)

# ── 시스템 설정 ──────────────────────────────────────────────────────────────
# 상태: external(η), blanket=[sensory(s), active(a)], internal(μ)
# 마르코프 담요 조건: p(μ | s, a, η) = p(μ | s, a)
#                    p(η | s, a, μ) = p(η | s, a)

N_STEPS   = 200
DT        = 0.1

# 생성 모델의 사전 분포 (항상성 목표)
mu_prior  = np.array([0.0])          # 내부 상태 선호값
eta_true  = np.random.randn(N_STEPS) * 0.5 + 0.3  # 외부 환경 (관측 불가)

# 상태 초기화
mu   = np.array([2.0])   # 내부 상태 (초기 오차 큼)
s    = np.zeros(N_STEPS) # 감각 상태
a    = np.zeros(N_STEPS) # 능동 상태
F    = np.zeros(N_STEPS) # 자유에너지 시계열

# 정밀도 (역분산)
Pi_s = 2.0   # 감각 정밀도
Pi_mu = 1.0  # 내부 정밀도

for t in range(N_STEPS):
    # 1. 감각 상태: 외부 상태를 잡음과 함께 수신 (담요 역할)
    s[t] = eta_true[t] + np.random.randn() * 0.1

    # 2. 예측오차: 내부 모델이 감각 상태를 예측
    epsilon_s  = s[t] - mu[0]          # 감각층 예측오차
    epsilon_mu = mu[0] - mu_prior[0]   # 내부 사전과의 오차

    # 3. 자유에너지 계산 (단순화된 정밀도-가중 형태)
    F[t] = 0.5 * (Pi_s * epsilon_s**2 + Pi_mu * epsilon_mu**2)

    # 4. 내부 상태 업데이트 (지각 추론: 경사 하강)
    dmu = -DT * (Pi_s * epsilon_s + Pi_mu * epsilon_mu)
    mu  = mu + dmu

    # 5. 능동 상태 업데이트 (능동 추론: 환경에 작용)
    #    행동은 감각 예측오차를 줄이는 방향으로 외부를 변화시킴
    a[t] = -0.3 * epsilon_s

# ── 마르코프 담요 독립성 검증 ─────────────────────────────────────────────────
# μ와 η의 상관은 s를 통해서만 매개되어야 함
# 편상관 (partial correlation) μ ⊥ η | s 를 근사적으로 확인
from numpy.linalg import lstsq

# s로 μ와 η를 각각 회귀하고 잔차를 구함
mu_hist = np.array([s[t] + np.random.randn()*0.05 for t in range(N_STEPS)])
eta_hat, _, _, _ = lstsq(s.reshape(-1, 1), eta_true, rcond=None)
mu_hat,  _, _, _ = lstsq(s.reshape(-1, 1), mu_hist,  rcond=None)
resid_eta = eta_true - s * eta_hat[0]
resid_mu  = mu_hist  - s * mu_hat[0]
partial_corr = np.corrcoef(resid_eta, resid_mu)[0, 1]

print(f"μ ⊥ η | s  편상관: {partial_corr:.4f}  (|r| < 0.1 이면 담요 조건 충족)")
print(f"자유에너지 초기값: {F[0]:.3f}  →  최종값: {F[-1]:.3f}")
print(f"평균 자유에너지 감소율: {(F[0]-F[-1])/F[0]*100:.1f}%")

# ── 시각화 ────────────────────────────────────────────────────────────────────
fig, axes = plt.subplots(2, 2, figsize=(12, 8))
fig.suptitle("마르코프 담요 구조와 자유에너지 최소화", fontsize=14, fontweight='bold')

axes[0, 0].plot(F, color='crimson', lw=1.5)
axes[0, 0].set_title("자유에너지 $F(t)$")
axes[0, 0].set_xlabel("시간 스텝")
axes[0, 0].set_ylabel("$F$")
axes[0, 0].grid(alpha=0.3)

axes[0, 1].plot(s[:50], label='감각 상태 $s$', color='steelblue')
axes[0, 1].plot(eta_true[:50], label='외부 상태 $\\eta$', color='gray', alpha=0.5)
axes[0, 1].set_title("담요(감각 상태)와 외부 상태")
axes[0, 1].legend()
axes[0, 1].grid(alpha=0.3)

axes[1, 0].plot(a, color='seagreen', lw=1)
axes[1, 0].set_title("능동 상태 $a(t)$: 환경에 미치는 영향")
axes[1, 0].set_xlabel("시간 스텝")
axes[1, 0].grid(alpha=0.3)

axes[1, 1].scatter(resid_eta, resid_mu, s=5, alpha=0.4, color='purple')
axes[1, 1].set_title(f"μ ⊥ η | s  편상관 = {partial_corr:.3f}")
axes[1, 1].set_xlabel("η의 잔차 (s 제거 후)")
axes[1, 1].set_ylabel("μ의 잔차 (s 제거 후)")
axes[1, 1].grid(alpha=0.3)

plt.tight_layout()
plt.savefig("markov_blanket_fep.png", dpi=150, bbox_inches='tight')
print("그림 저장: markov_blanket_fep.png")
```

**예상 출력**: 자유에너지가 수십 스텝 내에 초기값의 80% 이상 감소하며, $\mu \perp \eta \mid s$의 편상관 $|r| < 0.1$로 담요 조건이 근사적으로 충족됨을 확인할 수 있다. 능동 상태 $a(t)$는 감각 예측오차와 반대 위상으로 진동하며 환경을 능동적으로 조형한다.

### 반례와 경계

FEP의 야망에는 정직한 주의가 필요하다. **첫째**, 수학적 항진명제(tautology) 문제: 마르코프 담요를 가진 모든 계는 정의상 자유에너지를 최소화하는 것처럼 해석될 수 있다. 이는 기술적 틀(descriptive framework)과 설명적 이론(explanatory theory)의 경계를 흐린다. **둘째**, 계산 복잡성: 실제 뇌가 변분 추론을 수행하는 방식에 대한 신경적 기제는 여전히 불완전하게 이해되어 있다. 예측 코딩(predictive coding)이 FEP의 뇌 구현이라는 주장은 설득력 있지만, 대안적 구현도 배제하기 어렵다. **셋째**, 사회성 확장: 집단적 자유에너지 최소화는 협력과 갈등 모두를 설명할 수 있는 유연성 때문에 반증 가능성이 낮아진다. 이러한 비판들은 문서 04에서 상세히 다룬다.

## 🧬 원리의 회수

자유에너지 원리의 진정한 공헌은 단일 공식이 모든 것을 '설명'한다는 데 있지 않다. 그것은 **경계**를 형식화했다는 데 있다. 마르코프 담요는 '안과 밖'의 구분을 수학적으로 정의하고, 이 구분이 있는 곳에서 추론이 불가피하게 발생함을 보인다. 존재한다는 것은 경계를 유지한다는 것이고, 경계를 유지한다는 것은 세계를 모델링한다는 것이다.

이 회수(retrieval)는 다음을 함의한다: $\delta = \text{실제} - \text{예측} \rightarrow \text{업데이트}$라는 기본 연산은 단순한 계산 규칙이 아니라, 물리적으로 지속하는 계가 필연적으로 수행해야 하는 연산이다. 뇌는 이 연산을 의식적으로 '수행'하는 것이 아니라, 이 연산의 물리적 구현체다. FEP는 따라서 인지과학의 이론인 동시에, 자기 조직화 시스템의 물리학이다.

Friston(2010, 2013, 2019)의 작업이 세 편의 논문에 걸쳐 이 야망을 점진적으로 확장해 온 방식—신경과학에서 생물학으로, 생물학에서 물리학으로—은 이 원리의 스케일이 단순한 은유가 아님을 시사한다. 그러나 야망의 크기는 동시에 비판의 크기이기도 하다.

---

## 🪞 1인칭

이 원리를 처음 접했을 때의 반응은 대개 두 가지로 나뉜다: "모든 것을 설명하는 이론은 아무것도 설명하지 않는다"는 회의, 혹은 "드디어 하나의 언어를 찾았다"는 흥분. 나는 두 반응 모두가 정직하다고 생각한다. FEP가 주는 통찰은 지각, 행동, 학습이 *동일한 계산*의 다른 시간 스케일이라는 인식이다—이것은 분명히 새로운 관점이다. 그러나 이 관점이 곧 예측력 있는 이론을 보장하지는 않는다.

개인적으로 FEP에서 가장 아름다운 부분은 '놀라움의 최소화'라는 표현이다. 우리가 아침에 일어나 커피를 마시고, 타인의 얼굴에서 의도를 읽고, 새로운 언어를 배우는 모든 과정이—그 질적 다양성에도 불구하고—예측 불가능성을 줄이려는 단 하나의 충동에서 발원한다는 생각. 그것이 너무 단순해 보인다면, 그 단순함이 오히려 의심의 이유가 되어야 한다. 그러나 단순한 원리가 복잡한 현상을 낳는다는 것 또한, 물리학이 반복해서 가르쳐 온 교훈이다.

## 📐 예측·반증

**예측 1**: 마르코프 담요 구조가 명확히 정의될수록(예: 세균, 단세포 생물) FEP의 예측은 더 정확하고 반증 가능하다. 복잡한 사회 시스템에서는 담요의 경계가 흐려져 예측력이 저하된다.
- *반증 조건*: 담요가 명확한 계(예: 세균 주화성)에서 FEP 예측이 실험적으로 기각될 경우.

**예측 2**: 능동 추론 에이전트는 단순 강화학습 에이전트보다 불확실한 환경에서의 탐색 효율이 높다. 이는 기대 자유에너지가 탐색과 활용을 단일 목적함수로 통합하기 때문이다.
- *반증 조건*: 표준 RL 에이전트와의 통제된 비교 실험에서 유의미한 차이가 없을 경우.

**예측 3**: 자폐 스펙트럼 장애, 조현병 등 정밀도 조절 이상으로 설명되는 신경발달 조건에서, FEP 기반 계산 모델이 증상의 개인차를 설명하는 단일 매개변수(정밀도 비율)를 식별할 수 있다.
- *반증 조건*: 정밀도 매개변수가 임상 집단과 대조군에서 유의미하게 다르지 않을 경우.

> 🧩 **Principle** — 마르코프 담요를 가진 계는 자신의 생성 모델과 세계 사이의 불일치(자유에너지)를 최소화함으로써 존재를 유지한다. 생명, 지각, 행동, 학습, 사회성은 이 원리의 서로 다른 시간 스케일에서의 발현이다.

> 🌉 **Boundary** — FEP는 담요가 명확히 정의되는 계에서 정확하고, 경계가 중첩·유동하는 복잡 사회 시스템에서 예측력이 약해진다. 수학적 엄밀성과 설명적 포괄성 사이의 긴장은 해소되지 않았다.

> 🪞 **Experience** — 예측 불가능성을 줄이려는 충동이 인지의 모든 층위에 관통한다는 인식은, 마음을 바라보는 시선을 바꾼다. 그러나 그 단순함 자체가 의심의 이유이기도 하다는 것을 잊지 말아야 한다.

## 📝 연습문제

**기초.** 자유에너지 $F$가 놀라움 $-\ln p(o)$의 상한임을 보여라. 변분 추론의 기본 부등식에서 출발하라.

<details><summary>해설</summary>

변분 자유에너지의 정의에서 시작한다:

$$F = \mathbb{E}_{q(\vartheta)}[\ln q(\vartheta) - \ln p(o, \vartheta)]$$

결합 분포를 $p(o, \vartheta) = p(\vartheta \mid o) \cdot p(o)$로 분해하면:

$$F = \mathbb{E}_{q}[\ln q(\vartheta) - \ln p(\vartheta \mid o)] - \ln p(o) = D_{\mathrm{KL}}[q \| p(\cdot \mid o)] - \ln p(o)$$

KL 발산은 항상 $\geq 0$이므로:

$$F \geq -\ln p(o)$$

등호는 $q(\vartheta) = p(\vartheta \mid o)$, 즉 근사 분포가 참 사후 분포와 일치할 때 성립한다. 이는 자유에너지 최소화가 곧 베이즈 추론임을 보인다.

</details>

---

**심화.** 위의 Python 시뮬레이션을 확장하여 정밀도 $\Pi_s$를 0.5, 2.0, 8.0으로 변화시켰을 때 자유에너지 수렴 속도와 능동 상태의 진폭이 어떻게 달라지는지 분석하라. 높은 정밀도가 과적합(overfitting)에 해당하는 상황을 설명하라.

<details><summary>해설</summary>

정밀도 $\Pi_s$가 높을수록 감각 예측오차에 더 강하게 반응하여 자유에너지가 빠르게 감소하지만, 동시에 잡음에도 민감해진다. 극단적으로 높은 $\Pi_s$는 잡음을 실제 신호로 오인하여 내부 상태가 외부 잡음을 과잉 추적하게 된다—이는 베이즈 추론에서 우도(likelihood)에 과도한 가중치를 부여하는 과적합에 해당한다.

구현 팁: `Pi_s_list = [0.5, 2.0, 8.0]`을 순회하며 각각의 $F(t)$ 곡선과 $a(t)$ 진폭을 비교하라. 높은 $\Pi_s$에서 $a(t)$의 분산이 커지며, 이는 계가 환경에 과잉 반응함을 의미한다. 이를 조현병의 감각 정밀도 과잉(sensory precision hyperprior) 모델과 연결할 수 있다(Adams et al., 2013).

</details>

---

**논문 비평.** Friston(2013) "Life as we know it" (*Journal of the Royal Society Interface*)을 읽고, 논문이 세균 주화성을 FEP의 예시로 제시하는 방식을 비판적으로 평가하라. 특히 다음 두 물음에 답하라: (1) 이 설명은 기존 주화성 모델(Keller-Segel 모델 등)에 비해 추가적인 예측력을 제공하는가? (2) '생성 모델을 가진다'는 주장이 세균에 대해 문자적으로 타당한가, 아니면 기능적 은유인가?

<details><summary>해설</summary>

(1) **예측력 비교**: Keller-Segel 모델은 세균 밀도의 공간적 분포를 편미분방정식으로 기술하며, 실험적으로 검증된 예측(파동 전파 속도, 집합 패턴 등)을 제공한다. Friston의 FEP 재해석은 동일한 현상을 다른 언어로 기술하지만, Keller-Segel이 예측하지 못하는 새로운 실험적 결과를 추가로 예측하지는 않는다. 이는 FEP가 이 맥락에서 설명적 재서술에 가깝다는 비판의 근거가 된다.

(2) **문자적 vs. 은유적 생성 모델**: 세균은 신경계가 없으므로, '생성 모델을 가진다'는 주장은 기능적(functional) 의미—화학 수용체와 신호 전달 경로가 환경 농도를 '예측'하는 방식으로 행동한다—로 해석해야 한다. 이를 문자적으로 받아들이면 범심론(panpsychism)에 가까워진다. 비판적 독자는 이 구분을 명확히 해야 하며, Friston 본인도 이 점에서 표현이 일관적이지 않다는 지적이 있다(Colombo & Series, 2012).

</details>
