# 🧠 정책 예측 — 능동 추론

*능동 추론에서 행동은 세계를 변화시키는 것이 아니라 세계가 선호 상태와 일치하도록 예측을 실현하는 과정이다 — 정책 선택은 기대 자유에너지를 최소화하는 행동 시퀀스에 대한 예측이다.*

## 🎯 횡단 질문

신호 예측은 "무엇이 올 것인가"를 묻고, 상태 예측은 "세계가 지금 어떤 상태인가"를 묻는다. 정책 예측은 세 번째 질문을 묻는다: "내 행동이 어떤 결과를 낳을 것인가, 그리고 나는 어떤 행동을 선택해야 하는가." 이것은 예측의 시제가 달라진 것이다 — 과거/현재에서 미래로.

능동 추론(active inference)에서 정책(policy) $\pi$는 행동 시퀀스 $a_1, a_2, \ldots, a_T$에 대한 예측이다. 에이전트는 각 가능한 행동 시퀀스가 어떤 미래 상태를 낳을지 예측하고, 가장 선호하는 미래를 실현하는 시퀀스를 선택한다. 핵심 수식은 기대 자유에너지(expected free energy, EFE)다:

$$G(\pi) = \mathbb{E}_{q(o,s|\pi)}\left[\log q(s|\pi) - \log p(o,s)\right]$$

이를 분해하면:

$$G(\pi) \approx \underbrace{-\mathbb{E}_q[\log p(o|\tilde{o})]}_{\text{도구적 가치 (선호 상태 달성)}} + \underbrace{\mathbb{E}_q[H[p(o|s)]]}_{\text{인식론적 가치 (불확실성 감소)}}$$

첫 번째 항은 행동이 선호하는 결과(욕구 충족)를 낳을 것으로 기대되는 정도이고, 두 번째 항은 행동이 세계에 대한 지식을 증가시킬 것으로 기대되는 정도(호기심·탐색)이다. 능동 추론은 이 두 동기를 통합한 단일 목적 함수를 제시한다.

## 🌐 횡단 입력

- **reinforcement-learning-distilled (L3)**: 기대 자유에너지의 도구적 항이 TD 학습의 기대 보상과 형식적으로 대응한다. 차이: EFE는 보상을 최대화하는 것이 아니라 *기대 감각*이 선호 분포와 일치하도록 한다.
- **decision-making-distilled (L3)**: 인식론적 항이 정보 획득 행동(탐색, 호기심 주도 탐험)을 설명한다. 베이즈 최적 실험 설계(BOED)와 능동 추론의 연결.
- **perception-distilled (L2)**: 상태 예측 모델이 능동 추론의 기반. EFE를 계산하려면 행동이 어떤 상태를 만드는지 예측해야 하므로, 세계 모델 없이 정책 예측은 불가능하다.
- **consciousness-distilled (L4)**: 의지(volition)와 능동 추론의 관계 — 행동이 자유에너지 최소화의 결과라면 "선택"의 경험은 어디서 오는가?
- **IQ AI Lab (RL·LLM 직결)**: AlphaZero의 MCTS가 정책 예측의 공학적 변주 — 미래 시뮬레이션으로 행동 가치를 추정하고 최선 행동을 선택. LLM의 chain-of-thought가 EFE 인식론적 항의 텍스트 변주일 수 있다.

## 🔍 표면의 다양성

**차이 1: 시간 지평선.** 신호 예측은 현재 순간의 오차를 처리한다. 상태 예측은 현재 감각을 기반으로 현재 세계를 추론한다. 정책 예측은 미래에 걸친 여러 상태를 시뮬레이션한다 — 시간 지평선이 훨씬 길고, 분기적(branching)이며, 가상적이다. 이것은 계획(planning)과 상상(imagination)의 신경적 기반이다.

**차이 2: 행동이 포함된다.** 신호 예측과 상태 예측은 세계를 표상하는 수동적 예측이다(지각). 정책 예측은 세계를 변화시키는 능동적 예측이다 — 행동 자체가 예측의 구현이다. "세계가 이렇게 되어야 한다"는 예측이 행동을 통해 실현된다는 관점에서, 능동 추론은 지각과 행동을 동일한 자유에너지 최소화 원리 아래 통합한다.

**차이 3: 선호가 필요하다.** 신호 예측과 상태 예측은 선호 없이 작동한다 — 어떤 상태든 추론할 수 있다. 정책 예측은 선호 분포 $p(\tilde{o})$(바람직한 미래 관찰)를 필요로 한다. 이 선호는 어디서 오는가? 능동 추론에서 선호는 모델의 구성 요소로 주어진다 — 이는 강화학습의 보상 함수와 유사하지만, 결과가 아니라 미래 감각 분포로 표현된다는 점에서 다르다.

## ⚙️ 깊은 동형성

**분석 1: 정책도 예측이다.** TD 학습에서 가치 $V(s)$는 미래 보상의 예측이다. 능동 추론에서 정책 $\pi$는 미래 감각 시퀀스의 예측이다. 두 경우 모두 현재에서 미래를 예측하고, 예측을 기반으로 행동을 선택한다. 차이는 무엇을 예측하는가(보상 합 vs 감각 분포)와 어떻게 선호를 표현하는가(스칼라 보상 vs 선호 분포)다. $\square$

**분석 2: 도구적 가치 = 음의 예상 놀라움.** EFE의 도구적 항 $-\mathbb{E}_q[\log p(o|\tilde{o})]$는 미래 관찰이 선호 분포와 얼마나 다를지의 기대값이다. 이를 선호에 대한 예상 놀라움(expected surprise)이라 할 수 있다 — 낮을수록 선호가 실현될 가능성이 높다. 이것은 경로 전체에 걸친 예측 오차의 기대 합을 최소화하는 것과 동형이다. $\square$

**분석 3: 인식론적 가치 = 행동의 정보 이득.** EFE의 인식론적 항 $\mathbb{E}_q[H[p(o|s)]]$는 행동을 취했을 때 세계에 대해 얼마나 더 알 수 있는지의 기대값이다. 높을수록 그 행동이 불확실성을 감소시킨다. 이것이 탐색(exploration)과 호기심의 원리다 — 세계를 더 잘 예측하게 해주는 행동을 선호한다. $\square$

**분석 4: 행동은 감각을 예측에 맞추는 방법.** 지각에서 예측 오차는 표상을 변화시켜 감각에 맞춘다(인식). 능동 추론에서 예측 오차는 행동을 통해 감각 자체를 변화시켜 예측에 맞춘다(행동). 같은 자유에너지 최소화가 두 방향으로 작동한다 — 안에서 바깥을 바꾸거나(지각), 바깥을 안에서 예측한 대로 만들거나(행동). $\square$

## 🧪 수렴 증거

### 실험 1: 기대 자유에너지로 정책 선택 시뮬레이션

```python
import numpy as np
from scipy.special import softmax

np.random.seed(42)

# 간단한 능동 추론 정책 선택 시뮬레이션
# 에이전트: 2개 상태 (A, B), 2개 행동 (탐색, 착취)
# 선호: 상태 B를 선호 (음식이 있는 곳)

# 상태 전이 모델: P(s'|s, a)
# 행동 0(탐색): 랜덤 이동
# 행동 1(착취): 상태 B로 이동
n_states = 2
n_actions = 2

# 전이 행렬: transition[a, s, s'] = P(s'|s, a)
transition = np.array([
    [[0.5, 0.5], [0.5, 0.5]],  # 행동 0: 탐색 (불확실)
    [[0.1, 0.9], [0.1, 0.9]],  # 행동 1: 착취 (상태 B로 이동)
])

# 관찰 모델: P(o|s)
observation = np.array([
    [0.9, 0.1],  # 상태 A: 감각 A가 0.9 확률
    [0.1, 0.9],  # 상태 B: 감각 B가 0.9 확률
])

# 선호 분포: 감각 B(인덱스 1)를 강하게 선호
log_preference = np.array([-2.0, 2.0])  # 로그 선호 분포 (정규화 전)
preference = softmax(log_preference)

# 현재 믿음 (prior): 현재 상태 A에 있을 가능성이 높음
current_belief = np.array([0.8, 0.2])

def compute_efe(action, belief, transition, observation, preference, n_steps=3):
    """기대 자유에너지 계산 (단순화: 1스텝)"""
    # 다음 상태 믿음
    next_belief = transition[action].T @ belief  # P(s'|a) = Σ_s P(s'|s,a)P(s)
    
    # 기대 관찰 분포
    expected_obs = observation @ next_belief  # P(o) = Σ_s P(o|s)P(s)
    
    # 도구적 가치: -KL(P(o) || preference) 근사
    instrumental = np.sum(expected_obs * log_preference)  # 선호와의 일치도
    
    # 인식론적 가치: 정보 이득 (상태 불확실성)
    state_entropy = -np.sum(next_belief * np.log(next_belief + 1e-10))
    # 탐색 행동이 상태를 더 균등하게 만들면 정보 이득 가능
    obs_entropy = np.mean([
        -np.sum(observation[:, s] * np.log(observation[:, s] + 1e-10))
        for s in range(n_states)
    ])
    epistemic = state_entropy - obs_entropy  # 단순화
    
    return instrumental, epistemic, instrumental - epistemic  # EFE (낮을수록 선호)

print("=== 능동 추론: 정책(행동) 선택 ===")
print(f"현재 믿음: 상태A={current_belief[0]:.1f}, 상태B={current_belief[1]:.1f}")
print(f"선호 분포: 감각A={preference[0]:.3f}, 감각B={preference[1]:.3f}")
print()

for action, name in [(0, "탐색 (행동 0)"), (1, "착취 (행동 1)")]:
    inst, epist, efe = compute_efe(action, current_belief, transition, observation, preference)
    next_belief = transition[action].T @ current_belief
    print(f"{name}:")
    print(f"  다음 상태 믿음: A={next_belief[0]:.3f}, B={next_belief[1]:.3f}")
    print(f"  도구적 가치 (선호 달성): {inst:.3f}")
    print(f"  인식론적 가치 (정보 이득): {-epist:.3f}")
    print(f"  기대 자유에너지 G(π): {efe:.3f}")
    print()

# 정책 선택: softmax(-G(π))
efe_values = np.array([compute_efe(a, current_belief, transition, observation, preference)[2] 
                        for a in range(n_actions)])
policy_probs = softmax(-efe_values)  # 낮은 EFE = 높은 확률

print(f"정책 선택 확률:")
print(f"  탐색: {policy_probs[0]:.3f}")
print(f"  착취: {policy_probs[1]:.3f}")
print()
print("해석: 도구적 가치가 높은 착취가 선택되지만, 인식론적 가치에 따라")
print("탐색도 일정 확률로 선택됨 — 호기심과 착취의 자동 균형")
```

### 실험 2: 탐색-착취 균형의 EFE 조절

```python
import numpy as np
from scipy.special import softmax

np.random.seed(1)

# 탐색-착취 딜레마: 3개 선택지, 선호 강도 변화에 따른 정책 변화
n_arms = 3

# 각 팔(arm)의 불확실성과 기대 보상
expected_rewards = np.array([0.3, 0.7, 0.5])   # 기대 보상
uncertainties = np.array([0.8, 0.1, 0.5])       # 불확실성 (높을수록 정보 이득 큼)

print("=== 탐색-착취 균형: EFE 분해 ===")
print(f"팔 1: 기대보상={expected_rewards[0]}, 불확실성={uncertainties[0]} (저보상·고불확실)")
print(f"팔 2: 기대보상={expected_rewards[1]}, 불확실성={uncertainties[1]} (고보상·저불확실)")
print(f"팔 3: 기대보상={expected_rewards[2]}, 불확실성={uncertainties[2]} (중간)")
print()

preference_strengths = [0.5, 2.0, 5.0]  # 선호 강도 (높을수록 착취 편향)

for strength in preference_strengths:
    # EFE = -도구적가치 + 인식론적가치 (낮을수록 선호)
    # 도구적: 기대보상 × 선호 강도
    # 인식론적: 불확실성 감소 기대
    instrumental = expected_rewards * strength
    epistemic = uncertainties  # 불확실한 팔일수록 정보 이득 큼
    
    efe = -instrumental + (1.0 / strength) * epistemic  # 선호 강도에 반비례하는 탐색
    policy = softmax(-efe)
    
    print(f"선호 강도 = {strength}:")
    for i in range(n_arms):
        print(f"  팔 {i+1}: EFE={efe[i]:.3f}, 선택확률={policy[i]:.3f}")
    print(f"  선택: 팔 {np.argmax(policy)+1} 가장 선호됨")
    print()

print("해석: 선호 강도가 낮으면 탐색(불확실한 팔) 선호,")
print("선호 강도가 높으면 착취(높은 보상 팔) 선호.")
print("EFE 하나의 수식이 탐색-착취 균형을 자동으로 조절한다.")
```

**반례와 경계.** 능동 추론은 이론적으로 우아하지만 실용적 한계가 있다. 첫째, 정확한 EFE 계산은 조합론적으로 폭발한다 — 가능한 정책 수가 행동 수의 시간 지평선 거듭제곱이므로, 긴 계획에서는 근사가 필수다. 둘째, 선호 분포 $p(\tilde{o})$의 출처가 불명확하다 — 이것이 고정된 생물학적 상수인지, 학습되는 것인지, 사회적으로 구성되는 것인지에 대한 답이 아직 없다. 셋째, 능동 추론과 표준 RL의 행동 예측이 실제로 다른지에 대한 실험적 구분이 어렵다.

## 🌉 간극의 위치

✅ 도달: 탐색과 착취를 단일 목적 함수(EFE)로 통합하는 계산적 틀  
✅ 도달: 지각(상태 추론)과 행동(정책 선택)이 같은 자유에너지 최소화 원리 하에 있음  
✅ 도달: 호기심·탐색 행동의 정보이론적 설명 (인식론적 가치)  
✅ 도달: 다양한 신경 질환(강박, 중독, 우울)의 능동 추론 모델 (비정상적 선호 또는 EFE 계산 이상)  

❌ 도달 못 함: 어떻게 "하고 싶다"는 주관적 의지 경험이 EFE 최소화에서 나오는가 — 계산적 과정과 경험적 의도 사이의 간극.  
❌ 도달 못 함: 선호 분포가 어디서 오는가 — 능동 추론은 선호를 주어진 것으로 취급하며, 선호 자체의 형성 메커니즘은 다른 이론(강화학습, 사회화, 진화)이 필요하다.  
❌ 도달 못 함: 장기 계획과 가상적 시뮬레이션의 신경적 기반 — 해마의 역할, 전전두엽-기저핵 상호작용이 EFE 최소화로 완전히 설명되는가.  
❌ 도달 못 함: 능동 추론과 표준 강화학습의 행동 수준 구분 가능한 예측.

## 🧬 원리의 회수

정책 예측에서 $\delta$는 기대 감각과 선호 분포의 차이로 표현된다. 행동이 선호를 실현하지 못할 것으로 예측되면(도구적 가치 낮음), 다른 정책이 선택된다 — 이것이 "행동 수준의 예측 오차"다.

신호 예측(01)과의 연결: EFE의 도구적 항 $-\mathbb{E}[\log p(o|\tilde{o})]$은 미래 시점의 기대 보상 합과 형식적으로 연결된다. 상태 예측(02)과의 연결: EFE 계산은 세계 모델(전이 함수, 관찰 함수)에 의존하며, 이것이 바로 상태 예측 층위의 출력이다. 통합(04)으로의 연결: 세 층위의 예측 오차를 하나의 자유에너지 $F$로 묶는 것이 자유에너지 원리의 주장이다.

Friston(2010, 2017)의 능동 추론은 "살아있는 시스템은 자신의 마르코프 블랭킷을 유지하기 위해 자유에너지를 최소화한다"는 주장으로 확장된다. 이는 생물학적 자기조직화, 다세포 발생, 사회적 행동까지 포함하는 매우 광범위한 원리다 — 이 확장이 정당한 일반화인지 아니면 형식적 유비에 불과한지는 04-three-levels-unified.md에서 비판적으로 검토한다.

## 🪞 1인칭

커피와 차 중 무엇을 마실지 고민하는 순간을 떠올려보자. 두 선택지 각각에 대해 미래의 느낌을 상상한다 — 커피의 각성감, 차의 따뜻함. 이 상상이 EFE 계산의 경험적 대응물이다. 그러나 이 고민을 "자유에너지 최소화"라고 부르면 무언가 빠진 것 같다. 선택의 순간에는 무게감이 있다 — 어떤 것을 더 원하는가, 지금 내가 필요한 것은 무엇인가, 이 두 느낌이 동시에 존재한다. 이것이 EFE의 도구적 항(선호 달성)과 인식론적 항(정보 이득)의 경험적 질감일 것이다.

그러나 능동 추론이 말하지 않는 것이 있다. 선택이 이루어진 *이후의* 느낌 — 결단을 내렸다는 느낌, 혹은 고민을 내려놓은 느낌. EFE가 최소화되는 정책이 선택되었을 때, 그 수렴이 주관적으로 어떻게 경험되는가? 자유에너지 이론은 행동의 계산적 이유를 훌륭하게 기술하지만, 결정의 *현상학* — 선택하는 것이 어떤 느낌인지 — 은 여전히 이 틀 밖에 있다.

## 📐 예측·반증

**예측 1**: 능동 추론에서 탐색 행동은 인식론적 가치(정보 이득)로 설명된다. 따라서 환경이 완전히 예측 가능해진 후에도(모든 상태 불확실성 = 0) 에이전트가 새로운 자극을 탐색한다면, 이는 순수 인식론적 가치를 초과하는 탐색 동기를 시사한다.  
*반증 조건*: 환경 불확실성이 0에 도달하면 탐색이 즉시 중단된다면 인식론적 가치 가설을 지지한다. 탐색이 계속된다면 능동 추론만으로 호기심을 설명할 수 없다.

**예측 2**: EFE는 도구적 가치와 인식론적 가치를 더한 단일 수치로 정책을 선택한다. 따라서 인식론적 가치가 매우 높은 위험한 행동(예: 절벽 끝 탐색)과 인식론적 가치가 낮은 안전한 착취 행동 사이의 선택이, EFE의 상대적 크기에 의해서만 결정된다는 예측이 나온다.  
*반증 조건*: 피험자들이 EFE 합산이 예측하는 것보다 훨씬 강하게 위험한 탐색을 회피한다면, 안전성(생존)이 별도의 독립적 목적 함수로 작용함을 시사한다.

**예측 3**: 강박장애(OCD)가 과도하게 높은 선호(예상 감각과 현실의 차이에 대한 극단적 민감도)에서 비롯된다면, EFE의 도구적 항 가중치를 줄이는 개입(예: 인지 재구조화로 선호를 완화)이 강박 행동을 감소시켜야 한다.  
*반증 조건*: 선호의 강도를 직접 조작해도 강박 행동이 유지된다면, OCD의 원인이 선호가 아닌 다른 EFE 구성 요소(인식론적 불확실성 내성 등)에 있음을 시사한다.

## 🤔 열린 질문

**선호는 어디서 오는가?** 능동 추론은 선호 분포 $p(\tilde{o})$를 주어진 것으로 취급한다. 그러나 선호 자체가 발달 과정, 진화, 사회화를 통해 형성된다. 이 형성 과정이 더 상위의 자유에너지 최소화인가, 아니면 완전히 다른 원리가 필요한가?

**행동의 경험적 연속성을 어떻게 설명하는가?** EFE 최소화는 이산적 시간 단계에서 정의된다. 그러나 행동 경험은 연속적이고 유동적이다 — 계획하고, 실행하고, 수정하는 과정이 매끄럽게 연결된다. 이 연속성을 이산적 EFE 최소화로 어떻게 설명하는가?

→ Ch1-04: 세 수준의 통합에서 신호·상태·정책 예측이 하나의 자유에너지 원리로 통합될 수 있는지, 그리고 이 통합의 수학적·개념적 한계가 무엇인지 다룬다.

---

> 🧩 **Principle** — 정책 예측은 행동 시퀀스에 대한 기대 자유에너지 $G(\pi)$ 최소화로 표현되며, 선호를 실현하는 도구적 가치와 세계를 더 잘 알게 해주는 인식론적 가치가 단일 목적 함수 안에 통합된다.
> 🌉 **Boundary** — EFE 최소화가 행동의 계산적 이유를 설명하지만, "선택하는 것" 자체의 주관적 의지 경험 및 선호의 기원은 이 틀 밖에 있다.
> 🪞 **Experience** — 선택지를 상상하고 비교하는 그 고민의 무게감 — 어떤 미래가 더 원해지는가는 EFE가 기술하는 것이지만, 결단을 내린 이후의 해방감은 아직 그 수식에 없다.

## 📝 연습문제

**기초.** 기대 자유에너지(EFE) $G(\pi)$의 두 항 — 도구적 가치와 인식론적 가치 — 을 각각 일상 언어로 설명하고, 탐색(exploration)과 착취(exploitation)의 딜레마가 어떻게 EFE 하나로 통합되는지 설명하라.
<details><summary>해설</summary>

**도구적 가치**: "이 행동을 했을 때, 내가 원하는 결과(선호 감각)가 실현될 것으로 기대되는가?" 높을수록 원하는 것을 얻을 가능성이 높다. **인식론적 가치**: "이 행동을 했을 때, 세계에 대한 불확실성이 줄어들어 더 잘 예측할 수 있게 되는가?" 높을수록 그 행동이 학습에 유익하다. **딜레마 해결**: 착취는 도구적 가치가 높고(이미 아는 좋은 선택지), 탐색은 인식론적 가치가 높다(새로운 정보). EFE = -(도구적) + (인식론적)을 최소화하면, 두 가치의 합이 자동으로 최적화된다. 환경이 불확실할수록 인식론적 항이 크고 탐색이 이득, 선호가 강할수록 도구적 항이 크고 착취가 이득 — 별도의 탐색률(ε)을 설정할 필요 없이 자동으로 균형이 맞춰진다.

</details>

**심화.** 우울증(depression)을 능동 추론의 관점에서 해석하라. "미래가 선호하는 상태와 항상 멀 것으로 예측된다"는 가정에서 어떤 행동 패턴이 EFE 최소화로 나타나는가? 이 모델이 예측하는 치료 개입은 무엇인가?
<details><summary>해설</summary>

우울증의 능동 추론 모델: 선호 분포 $p(\tilde{o})$와 기대 감각 $p(o|\pi)$의 만성적 불일치. **EFE 최소화의 귀결**: 모든 행동이 높은 EFE(실현 불가능)를 가진다고 추정되면, 행동 회피(akinesia)가 EFE를 최소화하는 전략이 된다 — 행동하지 않으면 실망(높은 도구적 비용)을 피할 수 있다. 또한 세계를 탐색할 인식론적 가치도 감소한다(어차피 좋아지지 않는다는 믿음). **예측되는 행동**: 무쾌감증(anhedonia), 행동 억제, 사회적 회피. **치료 함의**: (1) 선호 조정 — 인지치료를 통해 비현실적 선호를 조정. (2) 인식론적 가치 회복 — 작은 탐색 행동을 통해 "세계가 변할 수 있다"는 믿음 복원(행동 활성화 치료와 대응). (3) 정밀도 재조정 — 항우울제가 감각-보상 정밀도를 높여 긍정 오차에 더 민감하게 만드는 것으로 해석 가능.

</details>

**논문 비평.** Friston et al.(2017) "Active inference and epistemic value"에서 제안된 능동 추론 틀과 표준 RL(Sutton & Barto) 사이의 핵심 차이를 평가하라. 두 이론이 행동적으로 구분 가능한 예측을 하는가? 만약 구분 가능하다면 어떤 실험이 이를 검증할 수 있는가?
<details><summary>해설</summary>

**핵심 차이**: (1) 표현: RL은 보상을 최대화하고, 능동 추론은 선호 감각 분포와의 KL 발산을 최소화한다. 스칼라 보상 vs 분포적 선호. (2) 탐색: RL은 ε-탐욕이나 볼츠만 탐색 등 외부 메커니즘 필요, 능동 추론은 EFE의 인식론적 항이 내재적 탐색 동기를 자동 생성. (3) 생성 모델: RL(특히 모델 프리)은 환경 모델 없이 작동 가능, 능동 추론은 세계 모델이 필수. **행동 구분 가능성**: 어렵지만 가능. 핵심 실험: 정보는 있지만 보상이 없는 행동을 에이전트가 선택하는가? 능동 추론은 인식론적 가치만으로도 행동을 예측하므로, 보상 없는 정보 탐색을 예측한다. RL(모델 프리)은 이를 예측하지 않는다. **실험 설계**: 쥐에게 보상 없이 새로운 공간 정보만 제공되는 선택지를 제공. 능동 추론 예측: 정보 탐색. 모델 프리 RL 예측: 무관심. 결과 해석에서 모델 기반 RL과의 구분이 남은 과제.

</details>
