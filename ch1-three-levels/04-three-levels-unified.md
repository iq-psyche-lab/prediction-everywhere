# 🧠 세 수준의 통합

*자유에너지 원리는 신호·상태·정책 예측을 단일 변분 목표 함수로 묶으려 하지만, 이 통합이 수학적 동일성인지 아니면 같은 형식을 공유하는 다른 계산들의 집합인지는 아직 열린 질문이다.*

## 🎯 횡단 질문

앞선 세 문서는 각기 다른 층위에서 예측 원리를 추적했다. 도파민 RPE는 스칼라 보상 예측 오차 $\delta = r + \gamma V(s') - V(s)$를 계산하고, 예측 부호화는 감각 예측 오차 $\epsilon = o - \hat{o}$를 계층적으로 전파하며, 능동 추론은 기대 자유에너지 $G(\pi)$를 최소화하여 행동을 선택한다. 세 수식은 모두 "실제 − 예측"의 차이를 사용한다. 이 구조적 유사성이 하나의 원리임을 의미하는가?

Karl Friston의 자유에너지 원리(Free Energy Principle, FEP)는 그렇다고 대답한다. FEP는 이 세 층위를 단일 변분 자유에너지 $F$의 최소화로 통합한다:

$$F = \underbrace{\mathbb{E}_q[\log q(s) - \log p(s,o)]}_{\text{변분 자유에너지}} = \underbrace{D_{KL}[q(s) \| p(s|o)]}_{\text{근사 오차}} - \underbrace{\log p(o)}_{\text{로그 증거}}$$

여기서 $q(s)$는 뇌의 내부 믿음(근사 사후 분포), $p(s,o)$는 생성 모델(세계의 인과 구조), $D_{KL}$은 쿨백-라이블러 발산이다. $F$를 최소화하면 동시에 두 가지가 달성된다: (1) 내부 믿음이 실제 사후 분포에 수렴(지각·상태 추론), (2) 놀라움 $-\log p(o)$가 최소화(예측된 감각 안에 머물기 위한 행동). 이 단일 목표가 신호 예측·상태 예측·정책 예측 모두를 인스턴스화한다는 것이 FEP의 핵심 주장이다.

그러나 이 통합을 비판적으로 검토해야 한다. 동형성(isomorphism)이 동일성(identity)은 아니다. 세 층위가 같은 수학적 형식을 사용한다고 해서 같은 *계산*을 수행하거나 같은 *물리적 메커니즘*으로 구현되는 것은 아니다.

## 🌐 횡단 입력

- **information-prediction-distilled (L0)**: 자유에너지 $F$가 놀라움(surprise, $-\log p(o)$)의 상한이라는 정보이론적 해석. 세 층위 모두 놀라움 최소화의 변주로 읽힌다.
- **neuromodulation-arousal-distilled (L1)**: 도파민(신호 예측)·노르에피네프린(정밀도 조절)·아세틸콜린(학습률 조절)이 자유에너지 최소화의 서로 다른 하이퍼파라미터를 조절한다는 가설.
- **perception-distilled (L2)**: 계층적 예측 부호화가 변분 자유에너지 최소화의 신경 구현이라는 Friston(2005)의 논증.
- **reinforcement-learning-distilled (L3)**: TD 학습과 자유에너지 최소화의 관계 — 모델 기반 RL이 FEP의 특수 사례로 흡수될 수 있는가.
- **consciousness-distilled (L4)**: 자유에너지 원리와 의식의 관계 — 마르코프 블랭킷이 자아와 환경의 경계를 정의한다는 주장.
- **IQ AI Lab (RL·LLM 직결)**: 딥러닝에서 변분 오토인코더(VAE)가 자유에너지 최소화의 공학적 구현. ELBo(Evidence Lower Bound) = $-F$.

## 🔍 표면의 다양성

**차이 1: 구현 메커니즘의 이질성.** 도파민 RPE는 중뇌-선조체 회로에서 신경화학적으로 구현된다. 예측 부호화는 피질 계층 전체에 걸친 분산 계산이다. 능동 추론의 EFE 최소화는 전전두엽-기저핵-해마 네트워크의 협력을 필요로 한다. 같은 자유에너지 원리를 구현한다 하더라도, 이 메커니즘들이 실제로 하나의 최적화 과정으로 조율되는지는 불명확하다.

**차이 2: 시간 척도의 분리.** 도파민 RPE는 밀리초~초 단위 사건 학습을 다룬다. 상태 예측은 수백 밀리초의 지각 처리를 다룬다. 정책 예측은 초~분~년에 걸친 계획을 다룬다. 자유에너지 원리가 이 세 시간 척도에서 동시에 작동하는 단일 메커니즘을 의미하는가, 아니면 각 척도에서 독립적으로 최적화가 이루어지는가?

**차이 3: 학습 vs 추론의 구분.** 도파민 RPE는 주로 시냅스 가중치의 장기 변화(학습)를 이끈다. 예측 부호화는 주로 현재 표상의 실시간 갱신(추론)이다. 능동 추론은 추론과 행동의 실시간 루프다. 자유에너지 최소화가 학습과 추론을 통합한다고 하지만, 이 두 과정이 실제로 같은 F를 최소화하는지, 아니면 다른 시간 척도의 F가 있는지가 불명확하다.

## ⚙️ 깊은 동형성

**분석 1: 세 층위는 같은 생성 모델의 다른 부분을 최적화한다.** FEP에서 생성 모델 $p(o, s, \pi) = p(o|s) \cdot p(s|\pi) \cdot p(\pi)$는 세 항을 포함한다: 관찰 모델, 전이 모델, 정책 사전 분포. 예측 부호화는 $p(o|s)$를 학습·사용하고, 능동 추론은 $p(s|\pi)$를 시뮬레이션하며, 도파민 RPE는 이 모델의 파라미터를 갱신한다. 서로 다른 최적화 과정이 하나의 생성 모델을 공유한다면, 이것이 통합의 진정한 기반이 될 수 있다. $\square$

**분석 2: 변분 자유에너지의 분해.** $F = \text{KL}[q(s) \| p(s|o)] - \log p(o)$에서 두 항의 역할이 구분된다. $\text{KL}$ 항 최소화는 내부 믿음의 정확도 향상(상태 예측)이고, $-\log p(o)$ 최소화는 놀라움 감소(신호 예측·정책 예측)다. 이 분해가 세 층위를 자연스럽게 수용한다: 지각은 첫 번째 항을, 행동은 두 번째 항을 주로 담당한다. $\square$

**분석 3: 마르코프 블랭킷과 자기조직화.** Friston의 더 넓은 주장: 자기조직화된 모든 시스템은 마르코프 블랭킷(내부 상태를 외부와 분리하는 경계)을 가지며, 이 구조가 자동으로 자유에너지 최소화처럼 *보인다*. 신경 시스템, 세포, 생태계가 모두 이 구조를 공유한다는 주장이다. 이것이 사실이라면 세 층위의 통합은 더 깊은 자기조직화 원리의 특수 사례다. $\square$

**분석 4: δ의 세 형태는 같은 업데이트 규칙의 변주다.** 세 층위의 오차 신호를 나란히 놓으면: 신호 예측 $\delta_{RPE} = r + \gamma V(s') - V(s)$, 상태 예측 $\epsilon_{pc} = o - f(s)$ ($f$는 생성 함수), 정책 예측 $\delta_{EFE} = G(\pi_{실현}) - G(\pi_{선택})$. 모두 "예측된 값 − 실제 값"이며, 이 차이가 파라미터를 갱신한다. 형식의 동일성은 명백하다. 기제의 동일성은 열려 있다. $\square$

## 🧪 수렴 증거

### 실험 1: 자유에너지 통합 시뮬레이션 — 세 층위의 공동 최소화

```python
import numpy as np
from scipy.special import softmax

np.random.seed(42)

# 통합 능동 추론 에이전트: 세 층위를 동시에 최적화
# 단순화된 2상태, 2행동, 2관찰 환경

# 생성 모델 파라미터
# A: 관찰 모델 P(o|s) — 상태 예측 층
A = np.array([[0.9, 0.1],   # 상태 0 → 관찰 0 (0.9), 관찰 1 (0.1)
              [0.2, 0.8]])   # 상태 1 → 관찰 0 (0.2), 관찰 1 (0.8)

# B: 전이 모델 P(s'|s, a) — 정책 예측 층
B = np.array([[[0.9, 0.1],  # 행동 0: 상태 0 유지
               [0.3, 0.7]],
              [[0.1, 0.9],  # 행동 1: 상태 1로 이동
               [0.1, 0.9]]])

# 선호 분포 C: log P(선호 관찰) — 관찰 1을 선호
C = np.array([-2.0, 2.0])

# 보상 신호 (신호 예측 층 — RPE 계산용)
reward = {0: 0.0, 1: 1.0}  # 관찰 1 = 보상

def compute_free_energy(belief_s, observation, A):
    """변분 자유에너지 계산 (상태 예측 층)"""
    # 예측 관찰 분포
    predicted_obs = A @ belief_s  # P(o) = Σ_s P(o|s) P(s)
    # 실제 관찰의 로그 우도
    log_likelihood = np.log(predicted_obs[observation] + 1e-10)
    # 사전 분포 (균등)
    prior_s = np.array([0.5, 0.5])
    kl_divergence = np.sum(belief_s * np.log(belief_s / prior_s + 1e-10))
    return kl_divergence - log_likelihood  # 자유에너지

def update_belief(belief_s, observation, A, lr=0.3):
    """베이즈 갱신 (상태 예측 층)"""
    likelihood = A[:, observation]
    posterior = likelihood * belief_s
    posterior /= posterior.sum()
    return posterior

def compute_efe(belief_s, action, A, B, C):
    """기대 자유에너지 (정책 예측 층)"""
    # 다음 상태 믿음
    next_belief = B[action] @ belief_s  # P(s') = P(s'|s,a) P(s)
    # 기대 관찰
    expected_obs = A @ next_belief
    # 도구적 가치: 선호와의 일치
    instrumental = np.sum(expected_obs * softmax(C))
    # 인식론적 가치: 관찰 후 상태 불확실성 감소
    H_s = -np.sum(next_belief * np.log(next_belief + 1e-10))
    epistemic = H_s
    return -instrumental + 0.5 * epistemic  # EFE (낮을수록 선호)

# 신호 예측 층: TD 가치 함수
V = np.zeros(2)  # 상태별 가치
alpha_td = 0.1
gamma_td = 0.9

print("=== 통합 능동 추론: 세 층위 동시 최적화 ===\n")

belief_s = np.array([0.8, 0.2])  # 초기 믿음
total_free_energy = []
rpe_history = []

for trial in range(50):
    # ---- 정책 예측 층: 행동 선택 ----
    efe_0 = compute_efe(belief_s, 0, A, B, C)
    efe_1 = compute_efe(belief_s, 1, A, B, C)
    action_probs = softmax([-efe_0, -efe_1])
    action = np.random.choice(2, p=action_probs)
    
    # ---- 환경: 상태 전이 및 관찰 생성 ----
    true_state = np.random.choice(2, p=B[action] @ belief_s)
    observation = np.random.choice(2, p=A[:, true_state])
    
    # ---- 상태 예측 층: 베이즈 갱신 ----
    fe_before = compute_free_energy(belief_s, observation, A)
    belief_s = update_belief(belief_s, observation, A)
    fe_after = compute_free_energy(belief_s, observation, A)
    total_free_energy.append(fe_after)
    
    # ---- 신호 예측 층: TD-RPE 계산 ----
    r = reward[observation]
    next_belief_for_td = B[action] @ belief_s  # 다음 상태 믿음
    expected_V_next = next_belief_for_td @ V
    rpe = r + gamma_td * expected_V_next - belief_s @ V
    V += alpha_td * rpe * belief_s  # 가치 갱신
    rpe_history.append(rpe)

print(f"초기 자유에너지: {total_free_energy[0]:.4f}")
print(f"최종 자유에너지 (평균): {np.mean(total_free_energy[-10:]):.4f}")
print(f"자유에너지 감소: {total_free_energy[0] - np.mean(total_free_energy[-10:]):.4f}")
print()
print(f"초기 RPE (평균): {np.mean(rpe_history[:10]):.4f}")
print(f"최종 RPE (평균): {np.mean(rpe_history[-10:]):.4f}")
print()
print(f"학습된 가치 함수: 상태0={V[0]:.3f}, 상태1={V[1]:.3f}")
print(f"최종 믿음: 상태0={belief_s[0]:.3f}, 상태1={belief_s[1]:.3f}")
print()
print("해석: 세 층위가 독립적으로 수렴하면서도 같은 방향을 향한다")
print("  - 상태 예측(자유에너지 감소) → 지각 정확도 향상")
print("  - 신호 예측(RPE 감소) → 가치 함수 수렴")
print("  - 정책 예측(EFE 최소화) → 선호 상태 달성")
```

### 실험 2: 통합의 한계 — 세 층위의 갈등 탐지

```python
import numpy as np
from scipy.special import softmax

np.random.seed(7)

# 세 층위가 서로 갈등하는 시나리오:
# 정보를 얻기 위한 탐색(인식론적 가치)이 선호 달성(도구적 가치)과 충돌
# 그리고 TD 신호가 예측 부호화 갱신과 다른 방향을 가리킬 때

print("=== 통합의 한계: 세 층위 갈등 분석 ===\n")

# 시나리오: 새로운 음식 시도 vs 익숙한 음식 유지
# 선호: 익숙한 음식 (도구적 가치 높음)
# 새 음식: 불확실하지만 정보 이득 높음 (인식론적 가치 높음)
# TD 신호: 과거 경험으로 익숙한 음식에 높은 가치 부여

# 행동 선택지
actions = ["익숙한 음식", "새로운 음식"]

# 도구적 가치 (선호 분포와의 일치도)
instrumental_value = np.array([0.8, 0.2])  # 익숙한 음식 선호

# 인식론적 가치 (불확실성 감소)
epistemic_value = np.array([0.05, 0.9])  # 새로운 음식이 정보 많음

# TD 가치 (과거 보상 기반)
td_value = np.array([0.7, 0.1])  # 익숙한 음식에 높은 TD 가치

print("각 행동의 세 층위 신호:")
print(f"{'':20} {'익숙한 음식':>12} {'새로운 음식':>12}")
print(f"{'도구적 가치':20} {instrumental_value[0]:>12.2f} {instrumental_value[1]:>12.2f}")
print(f"{'인식론적 가치':20} {epistemic_value[0]:>12.2f} {epistemic_value[1]:>12.2f}")
print(f"{'TD 가치':20} {td_value[0]:>12.2f} {td_value[1]:>12.2f}")
print()

# 각 층위의 행동 선택
choice_instrumental = np.argmax(instrumental_value)
choice_epistemic = np.argmax(epistemic_value)
choice_td = np.argmax(td_value)

print("각 층위의 독립 결정:")
print(f"  도구적 (EFE 도구항): {actions[choice_instrumental]}")
print(f"  인식론적 (EFE 인식항): {actions[choice_epistemic]}")
print(f"  TD 신호: {actions[choice_td]}")
print()

# FEP 통합 버전: EFE = -도구적 + 인식론적
# 단, 가중치 조정에 따라 결과가 달라짐
for w_inst, w_epist in [(1.0, 0.5), (1.0, 1.5), (2.0, 0.5)]:
    efe = -(w_inst * instrumental_value) + w_epist * epistemic_value
    unified_choice = np.argmin(efe)
    print(f"EFE (도구 가중={w_inst}, 인식 가중={w_epist}): {actions[unified_choice]}")
    print(f"  EFE 값: 익숙={efe[0]:.3f}, 새로운={efe[1]:.3f}")

print()
print("핵심 관찰: 가중치 선택에 따라 통합 이론이 다른 결론을 낸다.")
print("FEP는 이 가중치(정밀도)의 출처를 명시하지 않는다.")
print("=> 통합 이론이 예측하지 못하는 자유도가 있다.")
```

**반례와 경계.** 통합에 대한 두 가지 근본적 비판이 있다. 첫째, *형식적 통합 ≠ 기제적 통합*: FEP는 세 층위를 같은 수식으로 기술하지만, 이것이 세 층위가 실제로 하나의 신경 계산으로 구현된다는 의미가 아니다. 물리학과 생물학이 모두 미분방정식을 사용한다고 해서 같은 원리가 아닌 것처럼. 둘째, *과잉 일반성의 문제*: FEP는 너무 유연하여 거의 모든 시스템의 행동을 사후에 설명할 수 있다. Karl Friston이 지적했듯이, FEP는 자기조직화된 시스템이 마르코프 블랭킷을 가진다면 *반드시* 자유에너지를 최소화하는 것처럼 보인다는 수학적 결과다 — 이것이 강력한 예측 이론인지 아니면 항진명제(tautology)에 가까운지는 논쟁 중이다.

## 🌉 간극의 위치

✅ 도달: 세 층위가 같은 δ = 실제 − 예측 구조를 공유한다는 형식적 동형성  
✅ 도달: FEP가 지각(상태 추론)과 행동을 단일 최적화 문제로 기술하는 수학적 틀  
✅ 도달: 생성 모델 하나가 세 층위의 예측을 모두 인스턴스화할 수 있다는 계산적 가능성  
✅ 도달: VAE, 능동 추론 모델 등 공학적 구현을 통한 원리 검증  

❌ 도달 못 함: 세 층위가 *실제로* 뇌에서 하나의 자유에너지 최소화 과정으로 조율되는지의 신경과학적 증거 — 이는 여전히 이론적 주장이다.  
❌ 도달 못 함: 정밀도 가중치의 결정 메커니즘 — FEP는 정밀도가 중요하다고 하지만 그것이 어떻게 결정되는지를 설명하지 않는다.  
❌ 도달 못 함: 통합 이론의 반증 가능한 예측 — 세 층위가 *통합되지 않는다면* 관찰될 현상이 무엇인지 FEP는 명시하지 않는다.  
❌ 도달 못 함: 경험의 통일성 — 신호·상태·정책 예측이 통합되었다고 해서, 그것이 하나의 통일된 주관적 경험(unified conscious experience)을 설명하지는 않는다.  
❌ 도달 못 함: 놀라움이 *느껴진다*는 것의 설명 — 자유에너지가 최소화되는 과정이 왜, 어떻게 경험을 동반하는가는 FEP의 범위 밖이다.

## 🧬 원리의 회수

세 층위의 예측 원리를 나란히 놓으면:

| 층위 | 오차 신호 | 갱신 대상 | 시간 척도 |
|------|----------|----------|----------|
| 신호 예측 (RPE) | $\delta = r + \gamma V(s') - V(s)$ | 가치 함수 | 밀리초~초 |
| 상태 예측 (PC) | $\epsilon = o - \hat{o}$ | 지각 믿음 | 수백 밀리초 |
| 정책 예측 (AI) | $G(\pi) = -\text{도구} + \text{인식}$ | 행동 선택 | 초~분 |

세 오차는 모두 자유에너지 $F$의 기울기로 해석될 수 있다. 그러나 이 통합의 실질적 의미에 대해 두 입장이 존재한다.

**강한 통합 주장 (Friston)**: 세 층위는 하나의 생성 모델과 하나의 자유에너지를 공유하며, 뇌는 이 F를 동시에 최소화하는 단일 시스템이다. 신경조절물질의 차이는 같은 원리의 다른 하이퍼파라미터를 조절하는 것이다.

**약한 통합 주장 (비판적 입장)**: 세 층위는 형식적으로 동형이지만 기제적으로 분리되어 있다. 도파민 RPE·예측 부호화·능동 추론은 독립적으로 진화된 별개의 계산 시스템이 우연히 같은 수학적 구조를 갖는다. "하나의 원리"는 설명적 편의이지 온톨로지적 사실이 아니다.

이 레포의 입장: 두 주장 사이의 긴장을 유지한다. 형식적 동형성은 실재하며 유익하다 — 세 층위를 같은 언어로 기술하면 교차 예측과 공통 메커니즘 탐색이 가능해진다. 그러나 이 형식적 통합을 기제적 통합으로 혼동하지 않는다.

## 🪞 1인칭

어떤 음식이 맛있을 것이라 기대하고 먹었는데 예상보다 훨씬 맛있을 때를 생각해보자. 동시에 여러 일이 일어난다: 세계가 내 예측보다 좋았다는 인식(상태 예측 갱신), 이 음식에 대한 가치 평가가 올라간다는 느낌(신호 예측 RPE), 다음에 이 음식을 다시 먹겠다는 의도가 생긴다(정책 예측 갱신). 이 세 가지가 순차적이 아니라 동시에 일어나며, 하나의 경험으로 통합된다. 이것이 "세 층위의 통합"이 경험에서 어떻게 느껴지는지다.

그러나 이 경험을 자유에너지 최소화로 기술할 때 무언가 빠진다. 그 음식의 맛 자체 — 달콤함의 질감, 놀라움의 순간적 밝음 — 은 자유에너지 방정식에 없다. 세 층위의 통합은 경험의 *구조*를 설명하지만, 경험의 *내용*과 *질*에 대해서는 침묵한다. 이 침묵이 FEP의 한계가 아니라 단지 범위의 경계일 수도 있다 — 자유에너지 원리는 경험을 설명하려는 이론이 아니라 정보 처리를 설명하는 이론이다. 그렇다면 경험은 어디에 있는가? 이 레포는 이 질문을 닫지 않고 열어둔다.

## 📐 예측·반증

**예측 1**: 강한 FEP 통합 주장이 사실이라면, 세 층위 중 하나의 손상이 다른 층위에 체계적인 영향을 미쳐야 한다. 구체적으로, 도파민 시스템의 손상(파킨슨병)이 예측 부호화의 정밀도 가중에도 영향을 미쳐야 한다.  
*반증 조건*: 파킨슨병 환자의 지각 예측 부호화 패턴이 신경전형인과 동일하다면, 신호 예측과 상태 예측이 독립적 시스템임을 시사한다.

**예측 2**: 자유에너지 원리가 정밀도 가중을 통해 세 층위를 조율한다면, 정밀도를 조작하는 개입(예: 노르에피네프린 조절)이 세 층위 모두에 동시에 영향을 미쳐야 한다 — 지각 정확도, 강화학습 속도, 탐색-착취 균형이 동시에 변화해야 한다.  
*반증 조건*: 노르에피네프린 조작이 지각에는 영향을 주지만 강화학습 속도에는 영향이 없다면, 두 층위가 독립적 정밀도 메커니즘을 가짐을 시사한다.

**예측 3**: 만약 세 층위가 진정으로 하나의 자유에너지를 공유한다면, 인공 신경망에서 단일 자유에너지 목표를 최소화하도록 훈련된 시스템이, 층위별로 별도로 훈련된 시스템보다 표본 효율성과 일반화에서 체계적으로 우수해야 한다.  
*반증 조건*: 계층별 독립 훈련 시스템이 통합 시스템과 동등하거나 우수하다면, "하나의 F" 주장의 공학적 가치가 의문시된다.

## 🤔 열린 질문

**형식적 통합이 설명 통합을 의미하는가?** 세 수식이 같은 수학적 구조를 공유한다는 것은 분명하다. 그러나 이것이 "왜 뇌가 이렇게 작동하는가"에 대한 하나의 답을 제공하는가, 아니면 단지 기술적 편의인가? 물리학에서 최소 작용 원리(least action principle)가 고전역학·광학·양자역학을 통합하는 방식과 FEP가 유사한가?

**놀라움이 *느껴진다*는 것은 어디서 오는가?** 세 층위가 모두 $\delta$를 계산하고 최소화한다. 그러나 예측 오차가 주관적으로 경험될 때 — 놀라움의 감각, 실망의 감각, 기대의 감각 — 이 경험은 어디서 나오는가? 자유에너지 최소화는 이 질문에 침묵한다. 이것이 하드 문제(hard problem)의 핵심이다.

→ Ch2-01: TD 학습의 세 언어에서 신호 예측 층위의 도파민 RPE를 더 깊이 파고들어, 신경과학·공학·철학이 같은 알고리즘을 어떻게 다르게 기술하는지 추적한다.

---

> 🧩 **Principle** — 자유에너지 원리 $F = D_{KL}[q(s) \| p(s|o)] - \log p(o)$는 신호(RPE)·상태(예측 부호화)·정책(능동 추론)을 하나의 변분 목표 함수로 통합한다 — 단, 이 통합이 기제적 동일성인지 형식적 동형성인지는 열린 질문이다.
> 🌉 **Boundary** — 세 층위가 통합되어 자유에너지를 최소화하더라도, 그 과정이 왜 경험을 동반하는지, 그리고 경험의 내용과 질이 어떻게 결정되는지는 FEP가 설명하지 않는다.
> 🪞 **Experience** — 예상보다 맛있는 음식을 먹을 때 동시에 일어나는 세 가지 — 세계를 갱신하고, 가치를 올리고, 다음 행동을 계획하는 것 — 가 하나의 순간으로 통합되는 그 경험. 자유에너지는 그 구조를 기술하지만 그 맛은 기술하지 않는다.

## 📝 연습문제

**기초.** 변분 자유에너지 $F = D_{KL}[q(s) \| p(s|o)] - \log p(o)$에서 두 항이 각각 무엇을 의미하는지 설명하고, 지각(상태 추론)이 첫 번째 항을, 행동이 두 번째 항을 주로 감소시킨다는 주장의 직관적 근거를 설명하라.
<details><summary>해설</summary>

**$D_{KL}[q(s) \| p(s|o)]$ (KL 발산 항)**: 뇌의 내부 믿음 $q(s)$가 실제 사후 분포 $p(s|o)$와 얼마나 다른지를 측정한다. 이 항을 최소화하면 내부 믿음이 실제 세계 상태를 더 잘 반영하게 된다 — 이것이 지각(상태 추론)의 계산적 목표다. **$-\log p(o)$ (음의 로그 증거 항)**: 현재 감각 입력이 얼마나 놀라운지(surprising)를 측정한다. 예측된 감각은 높은 $p(o)$, 예측되지 않은 감각은 낮은 $p(o)$를 가진다. 이 항을 최소화하면 놀라운 감각을 줄이는 방향으로 움직인다. **지각이 첫 번째 항 감소**: 감각 입력 $o$가 주어졌을 때, 내부 믿음 $q(s)$를 갱신하여 KL 발산을 줄인다. 세계(감각)는 변하지 않고 표상이 변한다. **행동이 두 번째 항 감소**: 행동으로 세계를 변화시켜 감각 입력 $o$ 자체를 선호하는 것으로 만든다. 표상은 그대로이고 세계(감각)가 변한다. 두 항의 합이 $F$이므로 지각과 행동이 동시에 같은 $F$를 최소화한다.

</details>

**심화.** "자유에너지 원리는 항진명제(tautology)에 가깝다"는 비판과 "강력한 설명 원리다"라는 옹호 입장을 각각 최대한 강하게 논변하라. 이 논쟁을 해결하기 위한 실험 가능한 기준을 제시하라.
<details><summary>해설</summary>

**항진명제 비판 (강한 버전)**: Friston의 FEP는 마르코프 블랭킷을 가진 자기조직화 시스템이라면 *수학적으로* 자유에너지를 최소화하는 것처럼 기술될 수 있다고 주장한다. 이는 살아있는 시스템이 자유에너지를 최소화하도록 *진화했다*는 주장이 아니라, 단지 그런 시스템은 정의상 그렇게 기술 *가능하다*는 것이다. 이렇게 되면 FEP는 관찰을 예측하는 것이 아니라 모든 관찰과 양립하므로 반증 불가능에 가깝다. **강력한 설명 원리 옹호**: FEP는 서로 다른 분야(신경과학, 생물학, AI)에 구체적이고 연결된 예측을 만든다. 도파민-정밀도 관계, 능동 추론과 RL의 행동적 차이, 신경 질환의 FEP 모델 등은 FEP가 없었다면 생성되지 않았을 실험적 예측이다. **해결 기준**: (1) 세 층위가 *통합된* 경우와 *분리된* 경우를 행동·신경 수준에서 구분하는 패러다임 개발. (2) FEP와 대안 이론(독립적 RL + PC + 행동 계획)이 상반된 예측을 내는 시나리오 특정. (3) FEP 기반 AI 시스템이 모듈형 대안보다 일관된 우위를 보이는 벤치마크 구축.

</details>

**논문 비평.** Friston(2010) "The free-energy principle: a unified brain theory?"와 이에 대한 Colombo & Series(2012)의 비판을 비교하라. Colombo & Series가 지적한 "표준 베이즈 뇌 이론과 FEP의 차이가 불분명하다"는 비판이 타당한가? FEP가 표준 예측 부호화/베이즈 뇌 이론에 실질적으로 무엇을 추가하는지 평가하라.
<details><summary>해설</summary>

**Friston(2010)의 핵심 주장**: FEP는 지각·행동·학습·진화를 단일 원리로 통합한다. 뇌는 변분 자유에너지를 최소화하는 기계이며, 이것이 항상성, 적응 행동, 학습을 설명한다. **Colombo & Series(2012) 비판**: (1) FEP가 "베이즈 뇌" 가설과 구체적으로 어떻게 다른지 불분명하다. (2) FEP가 예측하는 고유한 현상이 베이즈 갱신만으로 설명 불가능한 것이 있는가? (3) FEP의 수학이 빈번히 변경·재정의되어 이론의 핵심이 무엇인지 추적하기 어렵다. **타당성 평가**: 부분적으로 타당. 단순 지각 영역에서는 FEP와 베이즈 뇌가 동일한 예측을 한다. FEP의 고유한 기여는 두 가지다: (1) 행동을 자유에너지 최소화로 포함하는 것 — 표준 베이즈 뇌는 지각에 집중하고 행동을 별도로 다룬다. (2) 마르코프 블랭킷 개념을 통해 자기조직화의 범위를 세포·개체·생태계까지 확장. 이 두 기여가 얼마나 *실질적*인지는 공학적 구현과 신경 데이터로 검증되어야 하며, 현재 진행 중이다.

</details>
