# 🧠 신호 예측 — 도파민 RPE

*도파민 뉴런은 보상 자체가 아니라 보상과 예측의 차이(δ)를 부호화한다 — 이것이 예측 원리의 가장 구체적이고 측정 가능한 신경 변주다.*

## 🎯 횡단 질문

"예측"이라는 단어가 뇌과학 문헌에서 처음으로 측정 가능한 형태를 얻은 곳은 중뇌 도파민 시스템이다. Schultz, Dayan & Montague(1997)의 고전적 실험은 원숭이의 복측피개부(VTA) 도파민 뉴런이 조건 자극(CS) 이전에는 보상(US)에 발화하다가, 학습이 진행되면서 보상 예측 신호인 CS에 발화하고, 예측된 보상이 생략되면 보상 시점에 오히려 발화가 억제됨을 보였다. 이 발화 패턴은 시간차 학습(Temporal Difference Learning, TD)의 보상 예측 오차(Reward Prediction Error, RPE)와 수학적으로 대응한다.

TD-RPE의 핵심 수식은 다음과 같다:

$$\delta_t = r_t + \gamma V(s_{t+1}) - V(s_t)$$

여기서 $r_t$는 시점 $t$의 즉각 보상, $\gamma \in [0,1]$는 할인 계수, $V(s_t)$는 상태 $s_t$의 가치 추정치다. $\delta > 0$이면 예측보다 좋은 결과(양성 RPE), $\delta < 0$이면 예측보다 나쁜 결과(음성 RPE), $\delta = 0$이면 예측과 일치한다. 도파민 뉴런의 발화율은 이 $\delta$에 선형적으로 비례하며, 이는 "뇌가 세계의 예측 오차를 계산하고 있다"는 강력한 증거다.

이 층위의 예측은 *신호 예측(signal prediction)*이라 부를 수 있다. "무엇이 올 것인가"라는 질문에 대한 스칼라 확률 추정이며, 결과가 도착하면 즉각 오차로 전환된다. 이것은 상태 공간을 추론하는 지각 예측이나 행동 시퀀스를 선택하는 정책 예측과 구별된다. 그러나 세 예측 모두 δ = 실제 − 예측 구조를 공유한다는 점에서 같은 원리의 변주다.

## 🌐 횡단 입력

- **information-prediction-distilled (L0)**: 보상 예측 오차의 정보이론적 해석 — 예기치 않은 사건의 정보량 $-\log p(r)$과 RPE의 대응. 놀라운 보상일수록 더 큰 학습 신호를 생성한다.
- **neuromodulation-arousal-distilled (L1)**: 도파민의 위상성(phasic) 발화 vs 긴장성(tonic) 발화의 구분. 위상성 RPE는 학습, 긴장성 도파민은 동기·각성의 기저선을 조절한다.
- **perception-distilled (L2)**: 선조체(striatum)가 피질 예측 신호를 수신하여 RPE를 계산하는 해부학적 루프. 기저핵-피질 회로가 신호 예측과 상태 예측을 연결한다.
- **reinforcement-learning-distilled (L3)**: Sutton & Barto(1998) TD(λ) 알고리즘의 신경 구현. 자격 흔적(eligibility trace)이 시냅스 변화의 시간적 귀속 문제를 해결한다.
- **IQ AI Lab (RL·TD 직결)**: 딥러닝 기반 RL에서 도파민 RPE와 동형인 TD 오차가 가치 네트워크를 훈련. AlphaGo/AlphaZero의 가치 헤드가 이 원리의 공학적 변주다.

## 🔍 표면의 다양성

**차이 1: 스칼라 vs 분포.** 고전적 도파민 RPE는 단일 스칼라 $\delta$로 모델링된다. 그러나 최근 분산 강화학습(distributional RL) 연구(Dabney et al., 2020)는 도파민 뉴런 집단이 보상 분포 전체를 부호화할 수 있음을 시사한다. 이 경우 동일한 RPE 원리가 훨씬 풍부한 통계적 구조를 담는다.

**차이 2: 시간 척도.** 도파민 위상성 발화는 밀리초~초 단위의 사건 예측을 처리한다. 반면 세로토닌·노르에피네프린 같은 다른 신경조절물질은 분~시간 단위의 더 긴 기간 예측 오차를 처리할 수 있다. "신호 예측"이 단일 시스템이 아니라 다중 시간 척도의 앙상블일 수 있다.

**차이 3: 모델 기반 vs 모델 프리.** 순수한 TD 학습은 모델 프리(model-free)다 — 환경의 전이 모델 없이 직접 경험에서 가치를 학습한다. 그러나 전전두엽-해마 회로는 모델 기반(model-based) 예측도 수행하며, 이 두 시스템이 같은 도파민 신호를 공유하면서도 서로 다른 방식으로 예측을 구성한다.

## ⚙️ 깊은 동형성

**분석 1: Rescorla-Wagner에서 TD로의 확장.** Rescorla-Wagner(1972) 모델은 연합 학습의 오차 수정 규칙 $\Delta V = \alpha(r - V)$를 제시했다. TD 학습은 이를 시간적으로 확장하여 즉각 보상뿐 아니라 미래 가치 추정치를 포함하는 $\delta_t = r_t + \gamma V(s_{t+1}) - V(s_t)$를 사용한다. 두 규칙 모두 "예측과 실제의 차이가 학습을 이끈다"는 구조를 공유한다. $\square$

**분석 2: 도파민 발화율의 선형 코딩.** Schultz(1997)의 데이터는 도파민 뉴런 발화율이 $\delta$에 선형 비례함을 보인다. 기저 발화율(~3-5 Hz)이 $\delta = 0$에 대응하고, 양성 오차는 발화율 증가($\sim$20 Hz), 음성 오차는 발화 억제($\sim$0 Hz)로 표현된다. 이 선형 인코딩은 RPE가 이진 신호가 아니라 연속 오차 신호임을 의미한다. $\square$

**분석 3: 가치 갱신의 방향성.** 가치 함수 업데이트 $V(s_t) \leftarrow V(s_t) + \alpha \delta_t$에서 $\alpha > 0$는 학습률이다. 양성 오차($\delta > 0$)는 가치 추정치를 올리고, 음성 오차($\delta < 0$)는 낮춘다. 이 방향성이 전전두엽-선조체 시냅스 가소성(LTP/LTD)과 대응한다는 신경과학적 증거가 누적되어 있다. $\square$

**분석 4: 조건 자극으로의 역전파.** 학습이 진행되면서 도파민 발화가 보상(US)에서 조건 자극(CS)으로 이동한다. 이는 TD 학습에서 예측 신호가 시간적으로 역전파되는 과정과 동형이다. $V(s_{CS})$가 점차 $V(s_{US})$에 수렴하면서, CS 자체가 보상 예측의 부호화 지점이 된다. 결과적으로 오차 신호는 예측이 가장 먼저 시작되는 지점으로 이동한다. $\square$

## 🧪 수렴 증거

### 실험 1: TD-RPE 시뮬레이션 — 도파민 발화 패턴 재현

```python
import numpy as np
import matplotlib.pyplot as plt

# TD 학습으로 Schultz(1997) 도파민 발화 패턴 시뮬레이션
np.random.seed(42)

n_trials = 300
n_timesteps = 10  # CS at t=3, US at t=7
gamma = 0.95
alpha = 0.1

# 가치 함수 초기화
V = np.zeros(n_timesteps + 1)

# 발화 패턴 기록
rpe_early = []   # 초기 학습
rpe_late = []    # 학습 후
rpe_omit = []    # 보상 생략

for trial in range(n_trials):
    delta_trial = np.zeros(n_timesteps)
    
    for t in range(n_timesteps):
        # CS = t=3, US(보상) = t=7
        reward = 1.0 if t == 7 else 0.0
        
        # TD-RPE 계산
        delta = reward + gamma * V[t+1] - V[t]
        delta_trial[t] = delta
        
        # 가치 함수 업데이트
        V[t] += alpha * delta
    
    # 초기/후기 시행 기록
    if trial < 20:
        rpe_early.append(delta_trial.copy())
    elif trial > 280:
        rpe_late.append(delta_trial.copy())

# 보상 생략 시행 (학습 완료 후)
V_trained = V.copy()
delta_omit = np.zeros(n_timesteps)
for t in range(n_timesteps):
    reward = 0.0  # 보상 생략
    delta_omit[t] = reward + gamma * V_trained[t+1] - V_trained[t]

rpe_early_mean = np.mean(rpe_early, axis=0)
rpe_late_mean = np.mean(rpe_late, axis=0)

print("=== Schultz(1997) RPE 패턴 재현 ===")
print(f"초기 시행 — CS(t=3): {rpe_early_mean[3]:.3f}, US(t=7): {rpe_early_mean[7]:.3f}")
print(f"학습 후 — CS(t=3): {rpe_late_mean[3]:.3f}, US(t=7): {rpe_late_mean[7]:.3f}")
print(f"보상 생략 — CS(t=3): {delta_omit[3]:.3f}, US(t=7): {delta_omit[7]:.3f}")
print()
print("예측:")
print("  초기: US에서 큰 양성 RPE, CS에서 약한 RPE")
print("  학습 후: CS에서 양성 RPE, US에서 ~0 RPE")
print("  생략: CS에서 양성 RPE, US에서 음성 RPE")

# 해석:
# - 초기 시행: 보상이 예측되지 않으므로 US 시점에 큰 δ > 0
# - 학습 완료: CS가 보상을 예측하므로 CS 시점에 δ > 0, US는 이미 예측됨 → δ ≈ 0
# - 보상 생략: CS 시점 δ > 0 (기대 활성화), US 시점 δ < 0 (예측 위반)
```

이 시뮬레이션은 Schultz(1997)의 세 핵심 조건(초기 학습/학습 완료/보상 생략)을 재현한다. 도파민 발화가 보상 자체에서 예측 오차로 이동하는 과정이 TD 알고리즘의 가치 전파와 수치적으로 일치한다.

### 실험 2: 분산 RPE — Dabney et al.(2020) 확장

```python
import numpy as np

# 분산 강화학습: 여러 분위수(quantile) RPE를 병렬로 학습
np.random.seed(0)

n_atoms = 21          # 도파민 뉴런 집단 크기 (다양한 분위수)
quantiles = np.linspace(0.05, 0.95, n_atoms)  # τ ∈ (0,1)
gamma = 0.95
alpha = 0.05
n_trials = 500

# 각 뉴런이 다른 분위수의 가치를 학습
# τ < 0.5인 뉴런: 비관적 (음성 오차에 민감)
# τ > 0.5인 뉴런: 낙관적 (양성 오차에 민감)
V_atoms = np.zeros(n_atoms)  # 단순화: 단일 상태

# 보상 분포: 0.5 확률로 1, 0.5 확률로 0
for trial in range(n_trials):
    r = 1.0 if np.random.rand() < 0.5 else 0.0
    
    for i, tau in enumerate(quantiles):
        delta = r - V_atoms[i]
        # 분위수 회귀: τ > 0.5이면 양성 오차에 더 크게 반응
        lr = alpha * tau if delta > 0 else alpha * (1 - tau)
        V_atoms[i] += lr * delta

print("=== 분산 RPE: 분위수별 가치 추정 ===")
print(f"이론적 기댓값: 0.500")
print(f"추정 범위: {V_atoms.min():.3f} ~ {V_atoms.max():.3f}")
print(f"중앙값 (τ=0.5): {V_atoms[n_atoms//2]:.3f}")
print()
print("해석: 각 도파민 뉴런이 보상 분포의 다른 분위수를 부호화")
print("비관적 뉴런(τ<0.5)은 낮은 값, 낙관적 뉴런(τ>0.5)은 높은 값 인코딩")
print("이 집합이 보상 분포 전체의 정보를 담는다")
```

**반례와 경계.** 도파민-RPE 대응은 강력하지만 완전하지 않다. 첫째, 도파민은 혐오 자극에도 반응하며 이는 순수 보상 RPE로 설명하기 어렵다(Matsumoto & Hikosaka, 2009). 둘째, 선조체 외 영역(전두엽, 해마)에서도 독립적 오차 신호가 관찰된다. 셋째, 도파민 조작 실험에서 RPE만이 아니라 이동·동기·살리언스도 함께 변화한다 — 도파민이 RPE의 순수 캐리어인지 아니면 더 넓은 신호인지는 미해결이다.

## 🌉 간극의 위치

✅ 도달: 도파민 발화율과 TD-RPE의 수치적 대응 (Schultz 1997, Bayer & Glimcher 2005)  
✅ 도달: 학습 과정에서 예측 신호가 CS로 역전파되는 메커니즘  
✅ 도달: 가치 갱신 방향성과 시냅스 가소성(LTP/LTD)의 대응  
✅ 도달: 분산 RPE 가설의 행동 및 신경 증거 (Dabney et al. 2020)  

❌ 도달 못 함: 왜 $\delta > 0$가 *좋게* 느껴지는가 — 오차의 발생적 메커니즘은 설명하나, 그것이 긍정적 경험으로 전환되는 이유는 불명.  
❌ 도달 못 함: RPE의 크기가 어떻게 주관적 놀라움의 강도로 변환되는가 — 스칼라 신호와 풍부한 경험적 질감 사이의 간극.  
❌ 도달 못 함: 도파민 없이도 인간이 강화학습을 한다는 증거 — 인지적 RPE와 신경화학적 RPE의 분리 문제.  
❌ 도달 못 함: 메타 예측 — "나는 잘 예측하고 있는가"에 대한 예측 오차의 신경적 기반.

## 🧬 원리의 회수

신호 예측에서의 $\delta = r_t + \gamma V(s_{t+1}) - V(s_t)$는 "예측 오차 최소화 δ = 실제 − 예측"의 가장 구체적인 구현이다. 이 층위에서 예측은 스칼라 가치 추정 $V(s)$로 압축되고, 오차는 뉴런 발화율로 부호화되며, 갱신은 시냅스 가중치의 변화로 구현된다.

다른 층위와의 연결: 상태 예측(L2, 02-state-prediction.md)에서 $\delta$는 감각 예측 오차로 재등장하며, 이는 역방향 계층적 연결을 통해 상위 표상을 갱신한다. 정책 예측(L3, 03-policy-prediction.md)에서 $\delta$는 기대 자유에너지의 도구적 항으로 흡수된다. 자유에너지 원리는 이 세 $\delta$를 단일 변분 목표 함수로 통합하려 하지만, 그것이 수학적 통합인지 아니면 개념적 은유인지는 04-three-levels-unified.md에서 다룬다.

Rescorla-Wagner(1972) → Sutton-Barto TD(1998) → Schultz et al.(1997) → Dabney et al.(2020)의 계보는 같은 오차 수정 구조가 심리학·공학·신경과학을 가로질러 독립적으로 발견되었음을 보여준다. 이 수렴이 "하나의 원리"의 증거인지 아니면 "오차 기반 학습"이라는 매우 넓은 범주의 다중 구현인지가 레포 전체의 핵심 질문 중 하나다.

## 🪞 1인칭

예상치 못한 보너스가 들어왔을 때의 그 감각을 떠올려보자. 단순히 기쁜 것이 아니다 — 순간적으로 세계가 예상보다 더 좋았다는 인식이 먼저 오고, 그 인식이 감정의 색을 칠한다. 반대로 기대했던 메시지가 오지 않았을 때, 허전함은 무언가가 *없는* 것의 느낌이다. 도파민 RPE는 이 두 경험이 같은 메커니즘의 양극임을 말한다 — 예측과 실제의 차이가 느껴지는 방향성.

그러나 $\delta$는 스칼라다. 숫자는 방향과 크기를 가지지만, 질감을 갖지 않는다. "예상보다 좋다"는 기분의 특정한 밝음, 가슴이 약간 들뜨는 감각, 시간이 잠깐 확장되는 느낌 — 이것들은 $\delta$에 없다. 신호 예측의 메커니즘은 정확히 무엇이 일어나는지 기술하지만, *어떻게* 느껴지는지에 대해서는 침묵한다. 이 침묵이 설명의 한계인지 아니면 아직 채워지지 않은 빈칸인지, 이 레포는 정직하게 미해결로 표시한다.

## 📐 예측·반증

**예측 1**: 도파민 차단제(할로페리돌 등)를 투여하면 조건 자극에 대한 반응 학습이 저해되나, 이미 학습된 가치에 기반한 행동 실행은 온전히 유지된다.  
*반증 조건*: 도파민 차단 후에도 새로운 CS-US 연합 학습이 정상적으로 이루어진다면 도파민-RPE 가설의 학습 역할이 부정된다.

**예측 2**: 분산 RPE 가설에 따르면, 낙관적(τ > 0.5) 도파민 뉴런과 비관적(τ < 0.5) 뉴런을 선택적으로 자극하면 동물의 위험 감수 행동(risk preference)이 분리적으로 조작된다.  
*반증 조건*: 낙관적/비관적 뉴런 자극이 위험 감수에 동일한 방향으로 영향을 주거나 전혀 영향이 없다면 분산 코딩 가설은 기각된다.

**예측 3**: 만약 RPE가 도파민에 의해서만 전달된다면, 도파민 특이적 신경 사멸(6-OHDA 병변) 이후 보상 학습이 완전히 중단되어야 한다.  
*반증 조건*: 6-OHDA 병변 동물도 새로운 보상 연합을 (느리더라도) 학습한다면 대체 RPE 경로의 존재를 지지하며, 도파민의 필요충분 역할이 약화된다.

## 🤔 열린 질문

**RPE는 있지만 경험은 어디서 오는가?** $\delta$ 신호가 선조체에서 계산되고 전전두엽으로 전달되는 과정을 추적할 수 있다. 그러나 이 정보 전달이 어느 지점에서 "느껴지는" 놀라움으로 전환되는지는 불명이다. 신호 예측은 메커니즘을 제공하지만 경험의 발생을 설명하지 않는다.

**신호 예측과 상태 예측은 어떻게 상호작용하는가?** 선조체의 RPE가 피질의 감각 모델을 갱신하는가, 아니면 두 시스템이 독립적으로 작동하는가? 이 질문은 다음 문서에서 베이즈 갱신의 관점으로 다룬다.

→ Ch1-02: 상태 예측 — 지각 모델에서 신호 예측이 감각 처리의 계층적 예측 부호화와 어떻게 연결되는지 추적한다.

---

> 🧩 **Principle** — 도파민 RPE는 $\delta = r_t + \gamma V(s_{t+1}) - V(s_t)$로 표현되는 예측 오차 최소화 원리의 신경화학적 구현이며, 보상 신호가 아니라 예측 위반 신호가 학습을 이끈다.
> 🌉 **Boundary** — $\delta$의 부호와 크기는 측정 가능하지만, 그것이 주관적 긍정·부정 경험으로 전환되는 이유는 메커니즘 수준에서 설명되지 않는다.
> 🪞 **Experience** — 예상보다 좋은 일이 일어날 때의 그 찰나의 밝음, 예상이 빗나갈 때의 허전함 — 두 경험은 같은 오차 신호의 두 방향이지만, $\delta$라는 숫자가 그 질감을 담지는 않는다.

## 📝 연습문제

**기초.** 도파민 뉴런이 조건 자극(CS) 학습 후 보상(US) 시점에서 발화가 억제되는 이유를 TD-RPE 수식으로 설명하라.
<details><summary>해설</summary>

학습 완료 후 CS가 보상을 완전히 예측하면, CS 시점에 $\delta_{CS} = 0 + \gamma V(s_{US}) - V(s_{CS}) > 0$로 양성 오차가 발생한다. 그러나 이미 $V(s_{CS}) \approx \gamma V(s_{US})$로 수렴했다면 $\delta_{CS} \approx 0$. US 시점에서는 보상 $r = 1$이 예측된 값 $V(s_{US}) \approx 1$과 같으므로 $\delta_{US} = 1 + \gamma \cdot 0 - 1 \approx 0$. 보상이 정확히 예측되면 오차가 0이고 발화 억제는 없다. 보상이 완전히 *예측된* 경우 도파민 기저 발화율만 유지되며, 이는 발화 증가도 억제도 아닌 상태다. 음성 억제는 보상이 *생략*될 때 나타난다: $\delta_{US} = 0 + 0 - V(s_{US}) < 0$.

</details>

**심화.** 분산 강화학습(distributional RL)에서 비관적(τ = 0.1)과 낙관적(τ = 0.9) 뉴런이 동일한 불확실한 보상(0.5 확률로 10, 0.5 확률로 0) 분포에서 학습할 때 최종 가치 추정치가 어떻게 달라지는지 계산하고, 이것이 위험 회피 vs 위험 추구 행동과 어떻게 연결되는지 논하라.
<details><summary>해설</summary>

기댓값은 5. 분위수 회귀에서 분위수 τ의 뉴런은 $\tau$-분위수에 수렴한다. τ = 0.1 뉴런은 보상 분포의 10번째 백분위수에 해당하는 값(≈0)에 수렴하고, τ = 0.9 뉴런은 90번째 백분위수(≈10)에 수렴한다. 비관적 뉴런 기준으로 행동 선택 시 불확실한 보상보다 확실한 작은 보상(예: 2)을 선호하여 위험 회피(risk-averse) 행동을 만든다. 낙관적 뉴런 기준으로 행동 선택 시 확실한 2보다 불확실한 분포를 선호하여 위험 추구(risk-seeking) 행동을 만든다. 이는 인간의 손실 회피와 도박 행동의 개인 차이를 신경 수준에서 설명하는 기전이 될 수 있다.

</details>

**논문 비평.** Schultz et al.(1997) "A Neural Substrate of Prediction and Reward"의 핵심 증거 구조를 평가하라: 단일 뉴런 기록에서 집단 수준 추론으로 가는 귀납적 도약이 정당한가? 방법론적 한계를 두 가지 이상 지적하고, 이후 연구(Bayer & Glimcher 2005, Dabney et al. 2020)가 이를 어떻게 보완하거나 수정했는지 논하라.
<details><summary>해설</summary>

**한계 1 — 선택 편향**: 단일 뉴런 기록은 도파민성으로 분류된 뉴런(발화율·파형 기준)에서만 이루어졌다. 이 기준이 완전하지 않아 다른 세포 유형이 혼입될 수 있다. Bayer & Glimcher(2005)는 더 엄격한 해부학적 위치 확인과 약물 검증을 추가했다. **한계 2 — 행동적 RPE와의 대응**: Schultz(1997)는 발화 패턴이 TD-RPE와 *닮았다*고 주장하지만, 직접적인 행동 RPE를 측정하여 선형 비교를 하지 않았다. Bayer & Glimcher는 행동 선택 데이터에서 독립적으로 추정한 RPE와 도파민 발화의 상관을 직접 계산했다(r ≈ 0.8). **보완**: Dabney et al.(2020)은 fMRI와 단일 뉴런 데이터를 결합하여 뉴런 집단이 분포를 부호화한다는 증거를 제시, 스칼라 RPE 가정 자체를 수정했다. 전체적으로 Schultz(1997)는 연결 구조를 처음 보였다는 점에서 탁월하지만, 해석의 직접성과 대안 가설 배제가 부족하다.

</details>
