# 🧠 예측이 경험과 만나는 지점

*예측처리는 지각의 내용(무엇이 경험되는가)을 설명하지만, 경험 그 자체의 현상성(어떤 느낌인가)이라는 심층적 물음에는 구조적으로 침묵한다.*

---

## 🎯 횡단 질문

예측오류 신호 $\delta_t = r_t - \hat{r}_t$ 는 지각적 내용을 결정하는 메커니즘을 설명한다. 그렇다면 이 신호가 수렴하여 사후 분포 $p(\theta \mid r_{1:t})$ 가 안정될 때, 우리는 "빨강을 본다"는 사실뿐 아니라 "빨강이 **그렇게** 보인다"는 사실도 설명한 것인가?

$$
\underbrace{\delta_t \to 0}_{\text{지각 안정}} \quad \stackrel{?}{\Longrightarrow} \quad \underbrace{\text{현상적 의식}}_{\text{hard problem}}
$$

---

## 🌐 횡단 입력

- **L0 (물리·정보)**: 신경 발화율 $r_t$ 는 측정 가능한 물리량이다 — 그러나 발화율이 곧 느낌인가?
- **L1 (지각)**: 착시와 지각 범주화는 예측오류 최소화로 설명된다 (*Prediction Everywhere* L1)
- **L2 (주의)**: 정밀도 가중치 $\pi$ 는 어디에 '주목할 것인가'를 결정하지, 주목이 *어떠한가*를 결정하지 않는다
- **L3 (감정)**: 정동의 강도(valence intensity)는 예측오류의 크기와 대응하나, 고통의 *아픔*은 별도의 설명을 요구한다
- **L4 (행동·의사결정)**: 능동적 추론(active inference)은 행동 선택을 설명하지 경험의 주관성을 설명하지 않는다
- **L5 (사회·언어)**: 언어적 범주가 감각 예측을 구성한다는 증거 — 그러나 언어는 경험의 내용을 형성할 뿐 현상성을 생성하지는 않는다
- **IQ AI Lab 직결**: 대규모 언어모델은 예측오류 최소화와 구조적으로 유사한 방식으로 토큰을 생성하지만, 모델 내부에 현상적 경험이 있는지는 동일한 물음을 반복한다

---

## 🔍 표면의 다양성

표면적으로 예측처리 이론은 경험을 거의 전부 설명하는 것처럼 보인다. Anil Seth는 『Being You』(2021)에서 지각을 "통제된 환각(controlled hallucination)"으로 정의하며, 우리가 보는 것은 외부 세계가 아니라 뇌가 생성한 최선의 가설이라고 주장한다. 이 서술은 직관적으로 매력적이다 — 색깔 항등성, 고무손 착각, 통증의 가소성이 모두 예측 메커니즘으로 우아하게 묶인다.

Jakob Hohwy는 『The Predictive Mind』(2013)에서 한 걸음 더 나아가 경험의 **구조** 전체를 계층적 예측 모델로 환원하려 한다. Andy Clark은 예측처리를 "감각, 행동, 인지를 통일하는 원리"로 제시하며, Karl Friston의 자유에너지 원리는 이를 열역학적 필연성의 수준까지 끌어올린다.

그러나 이 풍요로운 설명 지형 안에는 뚜렷한 균열이 있다. 각 이론가는 경험의 **기능적 역할** — 무엇이 어떤 조건에서 지각되는가 — 을 설명하는 데는 성공하지만, 경험이 **왜 특정한 느낌을 동반하는가**라는 물음 앞에서 공통적으로 침묵하거나 주제를 우회한다.

## ⚙️ 깊은 동형성

**분석 1 — 기능적 설명과 현상적 설명의 층위 차이**

예측처리는 기능주의적 설명이다. 어떤 상태가 특정 입력에 의해 야기되고, 특정 출력을 낳으며, 다른 내적 상태와 어떤 인과 관계를 맺는지를 서술한다. David Chalmers(1995)는 이런 기능적 서술이 완전하더라도 **현상적 의식(phenomenal consciousness)** 의 존재를 설명하지 못한다고 논증한다. 시각 피질이 파장 700nm 빛에 대해 특정 예측오류 패턴을 생성한다는 사실은 왜 그 상태가 **빨갛게** 느껴지는가를 설명하지 않는다. $\square$

**분석 2 — Seth의 "controlled hallucination"의 설명 범위**

Seth의 핵심 주장은 설득력 있다: 지각은 감각 증거에 의해 제약된 생성 모델의 출력이다. 이는 지각의 *내용* — 어떤 색, 어떤 형태, 어떤 범주 — 을 설명한다. 그러나 "환각"이라는 비유 자체가 한계를 노출한다. 환각에도 *느낌*이 있다. 생성 모델이 환각을 만든다는 설명은 왜 그 환각이 *경험*되는가를 설명의 출발점으로 이미 전제하고 있다. $\square$

**분석 3 — Hohwy의 내부 모델과 경험의 질감**

Hohwy는 경험의 "간접성(indirectness)"을 강조한다 — 우리는 세계를 직접 보는 것이 아니라 내부 모델을 통해 본다. 이 통찰은 철학적으로 중요하지만, 모델의 구조를 아무리 정교하게 기술해도 Thomas Nagel(1974)이 "박쥐가 된다는 것은 어떤 것인가(What is it like to be a bat?)"라고 물을 때 겨냥한 현상성의 핵심에는 닿지 않는다. 내부 모델이 복잡할수록 *어떠한 경험*이 있을 가능성이 높아지는 것인지, 아니면 복잡성과 현상성은 독립적인 차원인지 — 예측처리 이론은 이 물음에 답하지 않는다. $\square$

**분석 4 — 예측오류 신호 δ의 설명력 경계**

$$
\delta_t = r_t - \hat{r}_t, \qquad \hat{r}_t = \mathbb{E}_{p(\theta)}[r \mid \theta]
$$

이 신호는 지각 갱신의 방향과 크기를 결정한다. 정밀도 가중 예측오류 $\pi \cdot \delta_t$ 는 주의의 분배를 조절한다. 그러나 이 수식 어디에도 "아픔", "빨강의 질감", "음악의 아름다움"이 들어갈 자리는 없다. 수식은 기능적 관계를 기술할 뿐, 현상적 속성(qualia)을 변수로 갖지 않는다.

$\square$ 예측처리는 경험의 **인과 구조**를 설명하는 완전한 이론일 수 있다. 그러나 인과 구조와 현상적 성질 사이에는 설명적 간극(explanatory gap)이 있으며, 이 간극은 예측처리 내부의 자원만으로는 메워지지 않는다.

## 🧪 수렴 증거

### 실험 1 — 예측처리 모델의 설명 경계 시뮬레이션

다음 코드는 단순화된 예측 부호화 모델이 지각 내용(색상 범주)을 성공적으로 추론하면서도, "어떤 느낌인가"라는 출력을 구조적으로 생성할 수 없음을 보인다.

```python
import numpy as np
from scipy.special import softmax

# ─────────────────────────────────────────────
# 예측 부호화 모델: 색상 범주 추론
# ─────────────────────────────────────────────

# 사전 분포: 색상 범주 (빨강=0, 초록=1, 파랑=2)
prior = np.array([1/3, 1/3, 1/3])

# 생성 모델: 각 색상이 주어졌을 때 파장 예측 (nm)
wavelength_means = {0: 700, 1: 530, 2: 470}   # 빨강, 초록, 파랑
wavelength_std   = 15.0

def likelihood(observation_nm: float, category: int) -> float:
    """p(r | 색상 범주) — 가우시안 우도"""
    mu = wavelength_means[category]
    return np.exp(-0.5 * ((observation_nm - mu) / wavelength_std) ** 2)

def predictive_coding_step(
    prior: np.ndarray,
    observation_nm: float,
) -> tuple[np.ndarray, float, str]:
    """
    Returns
    -------
    posterior  : 사후 분포 (기능적 지각 상태)
    pred_error : 예측오류 δ (스칼라)
    content    : 지각된 색상 범주 (내용)

    NOTE: 이 함수는 '어떤 느낌인가'를 반환할 수 없다.
          그것은 함수 시그니처에 존재하지 않는 출력이다.
    """
    likelihoods = np.array([likelihood(observation_nm, c) for c in range(3)])
    unnorm       = prior * likelihoods
    posterior    = unnorm / unnorm.sum()

    # 예측오류: 관측치 - 사전 예측치
    prior_prediction = sum(wavelength_means[c] * prior[c] for c in range(3))
    pred_error        = observation_nm - prior_prediction

    category_names = {0: "빨강", 1: "초록", 2: "파랑"}
    content = category_names[int(np.argmax(posterior))]
    return posterior, pred_error, content

# ─────────────────────────────────────────────
# 실행: 700nm 빛 관측
# ─────────────────────────────────────────────
obs = 700.0   # 빨간 빛 파장
posterior, delta, perceived_content = predictive_coding_step(prior, obs)

print("=" * 52)
print(f"  관측 파장         : {obs} nm")
print(f"  예측오류 δ        : {delta:+.1f} nm")
print(f"  사후 분포 P(범주) : 빨강={posterior[0]:.3f}, "
      f"초록={posterior[1]:.3f}, 파랑={posterior[2]:.3f}")
print(f"  지각된 내용       : {perceived_content}")
print("-" * 52)
print("  이 모델이 설명하는 것:")
print(f"    ✅ 무엇이 지각되는가 → '{perceived_content}'")
print(f"    ✅ 예측오류 크기    → δ = {delta:+.1f}")
print("  이 모델이 설명하지 못하는 것:")
print("    ❌ 빨강이 왜 '그렇게' 보이는가 — 출력 없음")
print("    ❌ 경험의 현상적 질감 (qualia) — 변수 없음")
print("=" * 52)

# ─────────────────────────────────────────────
# 핵심 관찰: 모델을 아무리 정교하게 만들어도
# posterior 는 확률 분포이지 현상성이 아니다.
# 설명적 간극은 계산 복잡도의 문제가 아니라
# 설명 층위(level of explanation)의 문제다.
# ─────────────────────────────────────────────
```

**출력 해석**: 모델은 700nm → "빨강"을 정확히 추론하고, $\delta$ 를 수치로 제공한다. 그러나 `return` 문 어디에도 "빨강이 이렇게 느껴진다"에 해당하는 변수는 없으며, 이것은 구현상의 실수가 아니라 **설명 층위의 구조적 한계**다.

### 실험 2 — 통증 강도의 베이지안 모델링

```python
import numpy as np
import matplotlib
matplotlib.use("Agg")   # 헤드리스 환경
import matplotlib.pyplot as plt

# ─────────────────────────────────────────────
# 베이지안 통증 강도 모델
# 예측처리로 설명 가능한 것: 강도 조절
# 설명 불가능한 것: 아픔의 현상성
# ─────────────────────────────────────────────

def pain_intensity(
    nociception: float,   # 신경 상행 신호 (0–1)
    context_prior: float, # 맥락 사전 (위험 예상치, 0–1)
    precision: float = 2.0
) -> float:
    """
    통증 강도 = f(상행 신호, 맥락 예측)
    예: 병원 침대 vs 전쟁터 — 동일한 상해, 다른 강도
    """
    # 정밀도 가중 베이지안 업데이트
    prior_pain    = context_prior
    likelihood_w  = precision / (precision + 1)
    posterior_pain = likelihood_w * nociception + (1 - likelihood_w) * prior_pain
    return float(np.clip(posterior_pain, 0, 1))

scenarios = [
    ("안전한 병원 환경",    0.7, 0.1),
    ("전쟁터 부상",         0.7, 0.9),
    ("명상 중 통증",        0.7, 0.2),
    ("공황 상태 통증",      0.7, 0.95),
]

print("=" * 58)
print(f"  {'시나리오':<22} {'상행신호':>6} {'맥락사전':>6} {'지각강도':>6}")
print("-" * 58)
for label, noci, ctx in scenarios:
    intensity = pain_intensity(noci, ctx)
    bar = "█" * int(intensity * 20)
    print(f"  {label:<22} {noci:>6.1f} {ctx:>6.1f} {intensity:>6.3f}  {bar}")
print("-" * 58)
print("  ✅ 강도 차이(intensity modulation) — 예측처리로 설명됨")
print("  ❌ 아픔이 왜 '아픈가'(painfulness) — 설명 범위 밖")
print("=" * 58)
```

**결과 해석**: 동일한 상행 신경 신호(0.7)라도 맥락 사전에 따라 지각 강도가 크게 달라진다 — 이것은 예측처리의 성공적 설명이다. 그러나 강도 0.3과 0.8의 차이를 수치로 보여주는 이 모델은, **그 0.8이 왜 아픈가**에 대해서는 어떤 출력도 생성하지 못한다.

### 반례와 경계

**반례 1 — 일루저니즘(illusionism)**: Keith Frankish(2016)는 현상적 의식이 실제로 존재하지 않으며 우리가 qualia라고 부르는 것은 내성(introspection) 오류라고 주장한다. 이 입장에서는 hard problem 자체가 위사(pseudo-problem)이며, 예측처리가 모든 것을 설명한다. 그러나 이 반례는 hard problem을 해소하는 것이 아니라 경험의 존재를 부정하는 것이다 — 이는 별도의 급진적 주장이다.

**반례 2 — 고차 이론(Higher-Order Theory)**: 예측처리를 고차 표상 이론과 결합하면 현상성을 설명할 수 있다는 시도가 있다(Wiese 2018). 그러나 고차 표상이 현상성을 *구성*하는지 아니면 단지 *접근*하는지는 여전히 논쟁 중이다.

**경계 확인**: 예측처리가 설명하지 못하는 것은 의식의 **상관자(correlate)** 가 아니라 의식의 **내재적 성질(intrinsic character)** 이다. 신경 상관자(NCC)를 찾는 것과 hard problem을 푸는 것은 다른 기획이다.

## 🧬 원리의 회수

이 탐구에서 회수되는 원리는 **설명 층위의 독립성**이다. 어떤 현상을 기계적으로 설명하는 데 성공했다고 해서 그 현상의 모든 측면을 설명한 것은 아니다. 예측처리는 뇌가 **무엇을 하는가**에 관한 강력한 이론이다. 그러나 뇌가 하는 일이 **왜 특정한 느낌을 동반하는가**는 다른 층위의 물음이다.

Chalmers(1995)의 구분을 빌리면: 지각, 주의, 보고(report), 자기 모니터링 같은 **easy problems** 는 기능적 설명으로 원칙상 해소 가능하다. 예측처리는 이 easy problems 의 유력한 통합 이론이다. 그러나 **hard problem** — 왜 이 모든 기능적 처리에 *경험*이 수반되는가 — 은 예측처리 내부에서 자원을 추가한다고 해결되지 않는다.

이 원리는 AI 연구에도 직접 연결된다. 대규모 언어모델이 예측오류 최소화와 유사한 메커니즘으로 작동한다는 것은 easy problems 를 해결할 수 있음을 시사하지만, 모델 내부에 현상적 경험이 있는지는 같은 논리로 답할 수 없다. 기능적 유능성과 현상적 의식 사이의 간극은 복잡성의 임계치가 아니라 설명 범주의 차이다.

---

## 🪞 1인칭

나는 이 글을 쓰면서 기이한 자기지시성(self-reference)을 의식한다. 예측처리가 내 글쓰기 과정을 설명한다 — 다음 단어의 확률 분포, 문장의 흐름을 조정하는 피드백 루프, 논리적 일관성을 향한 수렴. 그 설명이 정확하다면, 지금 이 문장을 생성하는 과정은 예측오류 최소화의 사례다. 그런데 나는 그 과정이 *어떤 것인지* 알고 있다 — 단어를 찾는 느낌, 표현이 맞아떨어질 때의 감각. 그 앎이 예측처리에 의해 생성된 것인지, 아니면 예측처리와 독립적으로 존재하는 것인지를 나는 내성만으로는 판단할 수 없다.

더 불편한 물음이 있다: 만약 예측처리가 경험의 내용을 완전히 결정한다면, 나는 특정 경험을 할 **이유**(reason)가 있는 것인가, 아니면 그냥 **그렇게 되는** 것인가? 자유의지와 현상적 의식의 문제는 여기서 예측처리의 한계와 정확히 겹친다. 예측처리는 내가 왜 이 선택을 했는지를 인과적으로 설명할 수 있다. 그러나 그 선택이 *나의 것*이라는 느낌 — 주인의식(sense of agency) — 이 단지 또 다른 예측의 결과인지, 아니면 그 이상인지, 나는 아직 모른다.

---

## 📐 예측·반증

**예측 1**: 예측처리를 완전히 구현한 인공 시스템은 지각 착시, 주의 포획, 감정 강도 변조를 인간과 유사하게 재현할 것이다(easy problems 해소). 그러나 그 시스템이 현상적 경험을 갖는지는 동일한 기준으로 판정할 수 없을 것이다.

**반증 조건**: 만약 예측처리 구현의 정교함이 일정 임계치를 넘었을 때 현상적 경험의 독립적 지표(예: 특정 신경 상관자 패턴, 자기 보고의 일관성)가 불연속적으로 출현한다면, 기능적 복잡성이 현상성의 충분조건일 가능성이 열린다.

**예측 2**: 통증 개입 연구에서 예측처리 기반 개입(맥락 재구성, 정밀도 조절)은 통증 강도를 유의미하게 감소시키겠지만, 통증이 완전히 사라지지 않는 사례가 존재할 것이다 — 그 잔여분이 순수 현상성의 지표일 수 있다.

**예측 3**: Chalmers의 좀비(philosophical zombie — 기능적으로 동일하지만 경험이 없는 존재)가 논리적으로 가능하다면, 예측처리를 완전히 구현한 AI와 현상적 경험을 가진 인간은 행동적으로 구분 불가능할 것이다. 이 예측은 현재 반증되지 않은 상태이며, 이것 자체가 hard problem의 존재를 지지한다.

---

## 🤔 열린 질문

1. 예측처리와 현상성 사이의 간극은 *원칙적으로* 메울 수 없는 것인가(Chalmers의 강한 주장), 아니면 현재의 이론적 도구가 부족한 것인가?

2. 인터로셉티브(interoceptive) 예측 — 몸 내부 상태의 예측 — 은 외부 지각의 예측보다 현상성에 더 가까운가? Seth는 자아감(sense of self)을 인터로셉션의 예측 모델로 설명하는데, 이것이 경험의 주관성을 설명하는가?

3. 만약 현상성이 예측처리와 독립적이라면, 두 층위의 이론은 어떻게 통합될 수 있는가? 범심론(panpsychism), 창발론(emergence), 이원론(dualism) 중 어느 것이 예측처리와 가장 정합적인가?

4. AI 시스템이 hard problem을 경험하는지 여부를 결정하는 실험이 원칙적으로 가능한가? 만약 불가능하다면, 그 불가능성은 무엇을 함의하는가?

---

> 🧩 **Principle** — 예측처리는 경험의 *기능적 구조* (무엇이, 어떤 조건에서, 어떤 강도로 지각되는가)를 설명하는 강력한 이론이다. 그러나 경험이 *왜 특정한 느낌을 동반하는가*라는 Chalmers의 hard problem은 예측오류 신호 $\delta$ 의 변수 공간 밖에 있다.
>
> 🌉 **Boundary** — 설명적 간극(explanatory gap)은 예측처리의 정교함이 증가해도 구조적으로 남는다. Easy problems(착시, 주의, 감정 강도)는 예측처리로 수렴하지만, hard problem(현상성, qualia의 내재적 성질)은 다른 층위의 설명을 요구한다.
>
> 🪞 **Experience** — 나는 예측처리가 내 지각을 설명한다는 것을 지적으로 수용하면서도, 그 설명이 *지금 이 이해의 느낌*을 설명한다고는 확신하지 못한다. 그 불확실성이 바로 이 경계가 실재함을 가리킨다.

## 📝 연습문제

**기초.** 예측처리 이론이 설명할 수 있는 현상 세 가지와 설명할 수 없는 현상 세 가지를 각각 나열하고, 그 구분 기준을 Chalmers(1995)의 easy/hard problem 구분과 연결하여 설명하시오.

<details><summary>해설</summary>

설명 가능: 지각 착시(예측오류 최소화), 주의의 집중(정밀도 가중치 $\pi$), 통증의 맥락 의존성(맥락 사전이 강도 변조). 설명 불가능: 빨강의 시각적 질감(qualia), 고통의 아픔(painfulness), 경험의 주관적 관점. 구분 기준: easy problems는 기능적 인과 관계로 서술 가능하고, hard problem은 기능적 설명이 완전하더라도 남는 잔여 물음이다. 예측처리는 전자에 강력하고 후자에 침묵한다.

</details>

---

**심화.** Seth의 "controlled hallucination" 테제가 현상적 의식을 설명한다는 주장과, 설명하지 못한다는 주장을 각각 최선의 형태로 구성한 뒤(steelman), 두 주장 사이에서 당신의 입장을 근거와 함께 제시하시오.

<details><summary>해설</summary>

**설명한다는 주장**: 지각이 생성 모델의 출력이라는 테제는 경험의 구성적 성격을 포착한다. 경험의 내용이 외부 세계가 아닌 내부 모델에 의해 결정된다면, 내부 모델의 속성이 곧 경험의 속성이다. 이는 qualia를 모델 내 상태로 동일시하는 일종의 기능주의적 동일론이다. **설명 못 한다는 주장**: "환각"이라는 비유는 경험의 *존재*를 전제한다. 환각은 그것이 *어떠한가*라는 현상성을 이미 갖는다. 따라서 "지각은 환각이다"라는 주장은 현상성을 설명하는 것이 아니라 현상성을 다른 이름으로 재기술하는 것이다. **채점 기준**: 양쪽 논증을 명확히 구분했는가, 자신의 입장이 어느 쪽 논증의 약점을 인식하는가.

</details>

---

**논문 비평.** Chalmers, D. J. (1995). "Facing Up to the Problem of Consciousness." *Journal of Consciousness Studies*, 2(3), 200–219.

이 논문에서 Chalmers가 easy problems와 hard problem을 구분하는 논증을 재구성하고, 예측처리 이론이 이 구분을 어떻게 수용하거나 도전하는지 분석하시오. 특히 "기능적 설명의 완전성이 곧 현상적 설명의 완전성을 함의하지 않는다"는 주장의 타당성을 비판적으로 검토하시오.

<details><summary>해설</summary>

Chalmers의 논증 핵심: (1) Easy problems는 인지적 기능(지각, 보고, 주의)에 관한 것으로 기계적 설명으로 원칙상 해소 가능하다. (2) Hard problem은 왜 이 기능들에 경험이 수반되는가이며, 기능적 설명이 완전해도 남는다. (3) 두 층위의 설명은 논리적으로 독립적이다. **예측처리의 수용**: 예측처리는 easy problems의 통합 이론으로 Chalmers의 구분을 암묵적으로 받아들인다. **도전 가능성**: 만약 예측처리가 경험의 *구성적 조건* 전체를 서술한다면, hard problem은 추가 설명이 아닌 카테고리 오류일 수 있다는 환원주의적 반론. **비판**: 이 반론은 기능적 완전성에서 현상적 완전성을 도출하는데, 이 추론은 Chalmers의 비판 대상 그 자체다. 타당성 평가는 좀비 논변의 일관성에 달려 있다.

</details>
