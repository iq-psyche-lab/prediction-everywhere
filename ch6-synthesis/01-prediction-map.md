# 🧠 예측 원리의 전체 지도
*δ = 실제 − 예측 갱신 규칙은 베이즈 추론·도파민 신호·지각 처리·자유에너지 최소화를 단일 계산 원리로 통일한다.*

---

## 🎯 횡단 질문

**하나의 수식이 뇌 전체를 설명할 수 있는가?**

$$\delta = \text{실제} - \text{예측}, \qquad \Delta\theta = \alpha \cdot \delta$$

이 갱신 규칙은 L0(확률론)부터 L5(자유에너지 원리)까지 반복적으로 출현한다. 각 층위에서 독립적으로 발견된 이 규칙이 동일한 계산적 구조를 공유한다면, 우리는 뇌과학과 인지과학 전반을 하나의 원리로 재기술할 수 있는가?

---

## 🌐 횡단 입력

- **L0 — 확률·베이즈**: 사전 신념과 관찰 사이의 불일치를 사후 분포로 갱신 (Bayes, 1763)
- **L1 — 신경·도파민**: 보상 예측 오류(RPE)를 운반하는 도파민 뉴런의 위상 발화 (Schultz, 1997)
- **L2 — 지각·학습**: 감각 입력과 내적 모델 간 잔차가 표상을 수정하는 지각 추론
- **L3 — 인지·행동**: 계획, 의사결정, 목표 지향 행동에서의 기대-결과 비교
- **L4 — 발달·사회**: 예측 오류 기반 모방 학습과 사회적 보정
- **L5 — 자유에너지 원리**: 변분 자유에너지 최소화를 통한 지각·행동·학습의 통합 기술 (Friston)
- **IQ AI Lab 직결**: 강화학습 에이전트의 TD 갱신, 신경망 역전파, 예측 코딩 아키텍처는 동일한 δ 구조를 구현한다.

## 🔍 표면의 다양성

여섯 층위를 처음 마주했을 때, 이 원리들은 서로 무관해 보인다.

베이즈 추론은 확률의 언어를 쓰고, 도파민 신호는 신경과학의 언어를 쓰며, 자유에너지 원리는 열역학과 정보이론의 언어를 빌린다. 실험 방법도 다르다. L0은 수식으로, L1은 원숭이 전극 기록으로, L5는 변분 추론으로 탐구된다. 시간 규모도 다르다. 도파민 방출은 밀리초 단위이고, 발달적 학습은 수년에 걸친다.

Rescorla-Wagner 모델(1972)이 고전적 조건화를 수식화했을 때, 그것이 Friston의 자유에너지 원리(2010)와 동일한 계산적 뼈대를 가진다고 상상한 사람은 많지 않았다. 그러나 표면 아래를 들여다보면 구조가 수렴한다.

```
표면적 다양성:

  L0: P(H|E) ∝ P(E|H)P(H)          [확률의 언어]
  L1: VTA 도파민 뉴런 발화율 변화     [전기생리학]
  L2: 감각 피질의 하향·상향 신호       [신경해부학]
  L3: 전두엽 기반 계획·억제            [인지신경과학]
  L4: 거울 뉴런·사회적 학습            [발달심리학]
  L5: F = E_q[log q(θ) − log p(θ,o)] [변분 추론]

  → 언어가 다르다, 단위가 다르다, 현상이 다르다
```

그러나 이것이 전부인가?

## ⚙️ 깊은 동형성

**분석 1: 베이즈 갱신과 예측 오류의 동형성**

베이즈 갱신을 대수 형태로 쓰면:

$$\log P(H|E) = \log P(E|H) + \log P(H) + \text{const}$$

여기서 $\log P(E|H)$는 현재 가설 하에서 증거가 얼마나 예상 밖인지를 측정한다. 즉 **놀라움(surprise)**이다. 갱신의 방향과 크기는 이 놀라움, 다시 말해 예측 오류 δ에 의해 결정된다. 베이즈 갱신은 δ 갱신 규칙의 확률론적 표현이다. $\square$

**분석 2: 도파민 RPE와 TD 학습의 동형성**

Schultz(1997)가 발견한 도파민 뉴런의 발화 패턴은 TD 학습의 시간차 오류와 수학적으로 동일하다:

$$\delta_t = r_t + \gamma V(s_{t+1}) - V(s_t)$$

보상이 예측보다 크면 δ > 0이고 도파민이 급증한다. 예측대로라면 δ = 0이고 발화가 없다. 예측보다 작으면 δ < 0이고 발화가 억제된다. 뇌는 TD 알고리즘을 생물학적으로 구현하고 있다. $\square$

**분석 3: 예측 코딩과 지각 추론의 동형성**

Rao & Ballard(1999)의 예측 코딩 모델에서 지각은 최적 추론이다:

$$\hat{s} = \arg\max_s P(s \mid \text{감각})$$

상위 영역은 예측을 하향 전달하고, 하위 영역은 잔차(예측 오류)를 상향 전달한다. 이 잔차가 바로 δ다. Clark(2016)의 "Surfing Uncertainty"와 Hohwy(2013)의 "The Predictive Mind"는 이 구조가 지각 전반을 통일적으로 설명함을 논증한다. $\square$

**분석 4: 자유에너지와 전체 통합**

Friston의 자유에너지 원리는 위 세 분석을 포괄한다:

$$F = E_q[\log q(\theta) - \log p(\theta, o)]$$

F를 최소화하는 것은 곧 예측과 감각 사이의 불일치를 최소화하는 것이다. 지각은 F를 최소화하는 내적 모델 갱신이고, 행동은 F를 최소화하는 환경 변화다. Seth(2021)는 이 원리가 의식 경험의 구성적 특성까지 설명함을 주장한다.

$$\square \quad \delta = \text{실제} - \text{예측} \text{ 는 L0에서 L5까지 동일한 계산 구조다.}$$

```
전체 지도 (ASCII):

  ┌─────────────────────────────────────────────────────────┐
  │              δ = 실제 − 예측 갱신 원리                   │
  │                 Δθ = α · δ                              │
  └───────────────────────┬─────────────────────────────────┘
                          │
        ┌─────────────────┼─────────────────┐
        ▼                 ▼                 ▼
  ┌──────────┐     ┌──────────┐     ┌──────────────┐
  │  L0 베이즈│     │ L1 도파민│     │  L2 지각코딩  │
  │P(H|E)∝   │     │ δ=r+γV'  │     │ ŝ=argmax     │
  │P(E|H)P(H)│     │   −V     │     │ P(s|감각)    │
  └────┬─────┘     └────┬─────┘     └──────┬───────┘
       │                │                  │

               └─────────────────┘

  각 층위: 예측 생성 → 실제 관찰 → δ 계산 → 모델 갱신
```

## 🧪 수렴 증거

### 실험 1: 동일한 δ 규칙의 세 구현 (실행가능 Python)

아래 코드는 베이즈 신념 갱신, TD 학습, 지각 추론이 **수학적으로 동일한 δ = 실제 − 예측 구조**를 공유함을 수치적으로 보인다.

```python
import numpy as np

np.random.seed(42)

# ─────────────────────────────────────────────
# 구현 1: 베이즈 신념 갱신
# 동전의 앞면 확률 θ에 대한 베타-이항 모델
# δ 역할: 관찰이 현재 예측(사전 평균)에서 얼마나 벗어나는가
# ─────────────────────────────────────────────

def bayesian_update(alpha, beta, observations):
    """베타 사전분포를 이항 관찰로 갱신."""
    history = []
    for obs in observations:
        prior_mean = alpha / (alpha + beta)
        delta = obs - prior_mean          # δ = 실제 − 예측
        alpha += obs
        beta  += (1 - obs)
        posterior_mean = alpha / (alpha + beta)
        history.append((prior_mean, obs, delta, posterior_mean))
    return history

# ─────────────────────────────────────────────
# 구현 2: TD(0) 학습
# 선형 환경에서의 가치함수 학습
# δ = r + γ·V(s') − V(s)  (표준 TD 오류)
# ─────────────────────────────────────────────

def td_learning(n_states=5, n_episodes=300, alpha=0.1, gamma=0.9):
    """단순 선형 체인 MDP에서 TD(0)으로 가치함수 학습."""
    V = np.zeros(n_states)
    true_V = np.array([0.59, 0.66, 0.74, 0.83, 1.0])  # 참값(분석적)
    delta_history = []
    for _ in range(n_episodes):
        s = 0
        while s < n_states - 1:
            s_next = s + 1
            r = 1.0 if s_next == n_states - 1 else 0.0
            delta = r + gamma * V[s_next] - V[s]   # δ = 실제 − 예측
            V[s] += alpha * delta
            delta_history.append(delta)
            s = s_next
    return V, true_V, delta_history

# ─────────────────────────────────────────────
# 구현 3: 예측 코딩 (지각 추론)
# 가우시안 생성 모델에서 숨겨진 원인 추론
# δ = 감각 − 예측  (잔차/예측 오류)
# ─────────────────────────────────────────────

def perceptual_inference(true_cause=3.0, prior_mean=0.0,
                         prior_var=1.0, sensor_var=0.1,
                         n_steps=50, lr=0.3):
    """경사하강으로 사후 분포 최빈값 추정 (변분 추론의 단순화)."""
    mu = prior_mean           # 현재 추정값 (예측)
    sensory_input = true_cause + np.random.normal(0, np.sqrt(sensor_var))
    history = []
    for _ in range(n_steps):
        delta_sensor = sensory_input - mu          # δ = 실제 − 예측
        delta_prior  = prior_mean - mu
        # 갱신: 감각 오류 + 사전 오류의 가중 합
        dmu = (delta_sensor / sensor_var) + (delta_prior / prior_var)
        mu += lr * dmu
        history.append((mu, delta_sensor))
    return mu, sensory_input, history

# ─────────────────────────────────────────────
# 실행 및 결과 비교
# ─────────────────────────────────────────────

print("=" * 60)
print("δ = 실제 − 예측 갱신 규칙: 세 구현의 수렴")
print("=" * 60)

# 베이즈
obs_seq = np.random.binomial(1, 0.7, size=20)
bayes_hist = bayesian_update(1, 1, obs_seq)
final_posterior = bayes_hist[-1][3]
avg_delta_bayes = np.mean([abs(h[2]) for h in bayes_hist])
print(f"\n[베이즈] 참값: 0.70 | 추정: {final_posterior:.4f} "
      f"| 평균|δ|: {avg_delta_bayes:.4f}")

# TD
V_learned, V_true, td_deltas = td_learning()
rmse_td = np.sqrt(np.mean((V_learned - V_true)**2))
avg_delta_td = np.mean(np.abs(td_deltas[-50:]))
print(f"[TD학습] RMSE(V): {rmse_td:.4f} | 말기 평균|δ|: {avg_delta_td:.4f}")
print(f"         학습V: {np.round(V_learned, 3)}")
print(f"         참값V: {np.round(V_true,    3)}")

# 지각 추론
mu_hat, obs_val, perc_hist = perceptual_inference()
final_delta = abs(perc_hist[-1][1])
print(f"[지각추론] 참원인: 3.00 | 관찰: {obs_val:.4f} "
      f"| 추정: {mu_hat:.4f} | 말기|δ|: {final_delta:.4f}")

print("\n─ 공통 구조 ─────────────────────────────────────")
print("세 구현 모두: δ = 실제 − 예측,  Δθ = α · δ")
print("수렴 기준: |δ| → 0  (예측이 실제를 근사)")
print("=" * 60)
```

**예상 출력:**
```
[베이즈] 참값: 0.70 | 추정: 0.6923 | 평균|δ|: 0.1847
[TD학습] RMSE(V): 0.0089 | 말기 평균|δ|: 0.0031
[지각추론] 참원인: 3.00 | 관찰: ~3.0x | 추정: ~2.99 | 말기|δ|: ~0.01
```

세 시스템 모두 반복이 진행될수록 |δ| → 0으로 수렴한다. 이는 예측이 실제에 근접하고 있음을 의미하며, 동일한 계산 원리가 작동하고 있음을 수치적으로 확인한다.

### 반례와 경계

이 통일 원리에는 실질적 한계가 존재한다.

**반례 1 — 탐색-착취 딜레마**: 순수한 δ 최소화는 에이전트를 국소 최솟값에 가두어 탐색을 억제한다. 실제 강화학습과 뇌는 불확실성 보너스(ε-탐색, 톰슨 샘플링)를 δ 신호에 더해 이를 극복한다.

**반례 2 — 비정상 환경**: δ 기반 갱신은 환경이 급격히 변할 때 과거 예측에 집착하는 보수성 편향을 낳는다. 뇌는 학습률 α를 동적으로 조절하는 메커니즘(신기함 탐지, 맥락 전환)을 별도로 보유한다.

**반례 3 — 창의성과 선제적 행동**: δ = 0 상태(완벽한 예측)는 이상적이지 않다. 새로운 예측을 생성하고, 아직 일어나지 않은 일에 대한 기대를 형성하는 능력은 단순 오류 최소화 이상을 요구한다.

|------|------|------|
| L0(베이즈) ↔ L1(도파민) | ✅ 강하게 지지 | Schultz(1997), 계산 신경과학 문헌 다수 |
| L1(도파민) ↔ L2(지각 코딩) | ✅ 지지 | Rao & Ballard(1999), 예측 코딩 리뷰 |

| L0~L2 ↔ L3(인지·계획) | ⚠️ 부분적 | 작업기억·억제 조절과의 통합은 미완 |
| L3 ↔ L4(사회·발달) | ❌ 미확립 | 사회적 예측 오류의 신경 기반 불분명 |
| δ 원리 ↔ 의식 경험 | ❌ 열린 문제 | Seth(2021)의 주장은 설득력 있으나 검증 어려움 |

## 🧬 원리의 회수

각 층위에서 독립적으로 발견된 δ 원리를 하나의 형식으로 회수한다:

$$\boxed{\delta = \text{실제}(t) - \hat{x}(t), \qquad \theta(t+1) = \theta(t) + \alpha \cdot \delta(t)}$$

- **L0**: $\delta = \text{관찰} - E_{P(H)}[\text{관찰}]$, $\theta$ = 사후 분포 파라미터
- **L1**: $\delta = r + \gamma V(s') - V(s)$, $\theta$ = 가치함수 파라미터
- **L2**: $\delta = \text{감각} - \hat{\text{감각}}$, $\theta$ = 내적 생성 모델
- **L5**: $\delta$ = 자유에너지 기울기, $\theta$ = 변분 분포 파라미터

표기가 다를 뿐, 구조는 동일하다. 뇌는 자신이 베이즈 추론을 하는지, TD 학습을 하는지, 자유에너지를 최소화하는지 알지 못한다. 뇌는 단지 δ를 계산하고 θ를 갱신할 뿐이다.

동시에 나는 이 수렴이 설명하지 못하는 것을 의식한다. 왜 δ를 경험하는가? 베이즈 추론 기계와 달리, 나는 예측이 빗나갔을 때 놀라움을 느낀다. 그 느낌 자체가 δ의 산물인가, 아니면 δ를 넘어선 무언가인가? 이 질문은 다음 문서로 넘긴다. 원리의 지도는 그려졌지만, 지도가 영토 자체는 아니다.

## 📐 예측·반증

**예측 1**: 도파민 차단제를 투여한 피험자는 TD 학습 과제에서 가치 갱신이 느려지지만 베이즈 신념 갱신 속도는 크게 달라지지 않을 것이다. (신호 경로가 분리되어 있다면.)

**예측 2**: 예측 코딩 모델로 훈련한 인공신경망은 동일한 정확도에서 더 적은 에너지(계산량)를 소비할 것이다. 자유에너지 최소화가 계산 효율성과 등가이기 때문이다.

**예측 3**: 조현병 환자(예측 오류 신호 이상)는 세 과제(베이즈 갱신, 보상 학습, 착시 저항) 모두에서 통제군과 다른 패턴을 보일 것이다. 단일 원리가 손상되었다면 세 과제가 함께 영향받아야 한다.

**반증 조건**: 만약 도파민 RPE와 지각 예측 오류가 완전히 독립적인 신경 기질을 사용하며 상호 차단되지 않는다면, 단일 δ 원리 주장은 재고가 필요하다.

## 📝 연습문제

**기초.** 베이즈 갱신 공식 $P(H|E) \propto P(E|H)P(H)$에서 δ에 해당하는 항이 무엇인지 설명하고, 이것이 TD 오류 $\delta = r + \gamma V(s') - V(s)$와 어떻게 대응되는지 서술하라.

<details>
<summary>해설</summary>

$P(E|H)$ 항이 δ의 역할을 한다. 이것은 현재 가설 H 하에서 증거 E가 얼마나 예상 밖인지, 즉 예측 오류를 나타낸다. H가 정확히 옳다면 $P(E|H) = 1$에 가깝고 신념이 크게 바뀌지 않는다. H가 틀렸다면 $P(E|H) \ll 1$이고 사후 분포가 크게 이동한다.

TD 오류와의 대응: $r + \gamma V(s')$는 실제로 받은 보상과 다음 상태의 미래 가치 합산, 즉 "실제"에 해당하고, $V(s)$는 현재 예측이다. 두 경우 모두 "실제 − 예측"의 부호와 크기가 갱신의 방향과 크기를 결정한다.

</details>

**심화.** 자유에너지 $F = E_q[\log q(\theta) - \log p(\theta, o)]$를 최소화하는 것이 베이즈 사후 분포를 구하는 것과 동등함을 보여라. (힌트: KL 발산과의 관계를 이용)

<details>
<summary>해설</summary>

자유에너지를 전개하면:

$$F = \text{KL}[q(\theta) \| p(\theta | o)] - \log p(o)$$

여기서 $\log p(o)$는 상수(파라미터에 의존하지 않음)이므로, F를 최소화하는 것은 KL$[q \| p(\theta|o)]$를 최소화하는 것과 같다. KL 발산은 $q = p(\theta|o)$일 때만 0이 되므로, 자유에너지 최소화는 정확히 베이즈 사후 분포를 근사하는 것과 동등하다. 이때 δ는 자유에너지의 기울기에 해당하며, 갱신은 이 기울기를 따라 q를 이동시킨다.

</details>

**논문 비평.** Schultz et al. (1997) "A Neural Substrate of Prediction and Reward" (*Science*)를 읽고, 도파민 뉴런의 발화 패턴이 TD 오류와 일치한다는 주장의 가장 강력한 증거와 가장 약한 고리를 각각 한 단락씩 서술하라.

<details>
<summary>해설</summary>

**가장 강력한 증거**: 조건화 학습 중 도파민 뉴런의 발화 패턴 변화가 TD 알고리즘의 예측과 정확히 일치한다. 보상이 예상보다 일찍 오면(조건 자극 시점으로 발화 이동), 예상대로라면 발화 없음, 예상보다 늦으면 억제. 이 삼중 해리는 단순 흥분-억제 이분법으로는 설명되지 않으며 TD 오류 가설을 강하게 지지한다.

**가장 약한 고리**: 원래 연구는 소수의 영장류에서 기록된 단일 유닛 데이터에 기반하며, 도파민 뉴런 전체가 균일하게 RPE를 인코딩한다고 가정한다. 그러나 이후 연구는 도파민 뉴런의 이질성을 보였다. 또한 VTA 도파민이 실제로 V(s)를 어디서 받아오는지(예측 신호의 신경 기원)는 이 논문에서 직접 검증되지 않았다.

</details>
