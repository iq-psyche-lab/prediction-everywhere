# 🧠 Prediction Everywhere의 종합
*하나의 추론 원리 δ = 실제 − 예측 가 마음의 모든 층위를 관통한다 — 그러나 관통이 멈추는 지점, 바로 거기에 의식이 살고 있다.*

---

## 🎯 횡단 질문

이 레포지토리 전체를 하나의 질문으로 압축하면 다음과 같다.

$$\delta_t = r_t + \gamma V(s_{t+1}) - V(s_t)$$

이 단순한 차분(差分)이 — 도파민 뉴런의 발화 패턴부터 자유에너지 최소화, 언어적 추론, 능동추론에 이르기까지 — 마음을 설명하는 통일 원리가 될 수 있는가? 그리고 만약 그것이 불가능하다면, 그 한계는 어디에 있으며, 그 한계 자체가 우리에게 무엇을 가르쳐 주는가?

보다 날카롭게 표현하면: 예측 오류(prediction error)는 마음의 *기능적 구조*를 설명하는가, 아니면 마음 *그 자체*를 설명하는가? 이 두 질문 사이의 거리가 이 문서 전체의 지형이다.

## 🌐 횡단 입력

이 종합 문서는 "Prediction Everywhere" 레포지토리의 L0부터 L5까지 모든 층위를 회수한다. 각 층위가 이 문서에 기여하는 핵심 입력은 다음과 같다.

- **L0 수학적 기초**: 베이즈 정리, 변분추론, KL 발산의 형식적 토대. δ가 단순한 직관이 아니라 수학적으로 정의된 객체임을 확립함.
- **L1 신경세포 수준**: Schultz(1997)의 도파민 실험 — 예측 오류 신호가 생물학적 신경 회로에 실재함을 보인 결정적 증거.
- **L2 지각과 학습**: Rescorla-Wagner 규칙부터 헬름홀츠 머신(Helmholtz Machine)까지 — 지각이 수동적 수용이 아니라 능동적 가설 검증임을 확립.
- **L3 언어와 기억**: 언어 모델의 다음 토큰 예측과 해마의 예측적 부호화 — 고차 인지도 동일한 원리 위에 서 있음.
- **L4 능동추론과 의식**: Friston의 능동추론(active inference) — 행동 자체가 예측 오류를 최소화하는 방식임. 의식의 신경상관자(NCC) 문제와의 접점.
- **L5 자유에너지 원리**: 전체 마음을 단일 변분 원리로 통합하려는 야망 — 그리고 그 한계의 명시적 지도.
- **IQ AI Lab 직결**: 대형 언어 모델(LLM)의 아키텍처가 예측 부호화의 계산적 구현인가라는 질문. 인공 마음과 생물학적 마음의 수렴 지점.

## 🔍 표면의 다양성

마음을 연구하는 사람이라면 우선 그 *다양성*에 압도된다. 도파민 뉴런은 밀리초 단위로 발화 패턴을 조정한다. 해마는 수십 분에 걸쳐 기억을 공고화한다. 전전두엽은 수년에 걸친 생애 계획을 유지한다. 이 현상들이 같은 원리로 설명될 수 있다는 주장은, 얼핏 보면 지나친 단순화처럼 들린다.

표면적으로 보면:

**시간 척도의 다양성.** 반사궁(reflex arc)은 수십 밀리초, 작업기억은 수 초, 장기기억은 수십 년. 같은 원리가 이 모든 척도에서 작동한다고 말할 수 있는가?

**표현 양식의 다양성.** 시각 피질의 방향 선택성 세포, 해마의 장소 세포(place cells), 브로카 영역의 문법 처리 — 이것들이 모두 "예측 오류"를 처리한다고 볼 수 있는가?

**계산 기제의 다양성.** 시냅스 강화(LTP/LTD), 도파민 신호 전달, 신피질의 역방향 연결(feedback connections) — 메커니즘이 이렇게 다른데 동일한 원리를 공유하는가?

**현상학적 다양성.** 고통의 질감, 붉은색의 감각질(qualia), 사랑의 느낌 — 이것들이 정말로 예측 오류 최소화의 산물인가?

이 다양성들은 실재한다. 그리고 좋은 통합 이론은 이 다양성을 *지워버리는* 것이 아니라 *설명해야* 한다. "Prediction Everywhere"가 진정한 이론이라면, 왜 이렇게 다양한 구현이 존재하는지를 설명할 수 있어야 한다.

## ⚙️ 깊은 동형성

표면의 다양성 아래에는 놀라운 구조적 동일성이 있다. 네 가지 분석으로 이를 드러낸다.

### 분석 1: TD=도파민=파블로프 삼각 통합의 최종 정리

Schultz, Dayan, Montague(1997)의 고전적 실험은 원숭이 도파민 뉴런이 정확히 TD 오류 신호를 부호화함을 보였다. 보상이 예상치 못하게 주어지면 발화가 증가하고(δ > 0), 예상된 보상이 오지 않으면 발화가 감소한다(δ < 0). 예상대로 주어지면 변화가 없다(δ = 0).

이것은 우연이 아니다. Rescorla-Wagner 규칙:

$$\Delta w_i = \alpha \cdot \delta \cdot x_i, \quad \delta = \lambda - \sum_j w_j x_j$$

이 규칙은 파블로프 조건형성의 수학적 형식화다. 그리고 TD 학습:

$$V(s_t) \leftarrow V(s_t) + \alpha \cdot \delta_t$$

은 Rescorla-Wagner의 시간 연속 일반화다. 세 가지 — 파블로프 조건형성의 행동 법칙, Rescorla-Wagner의 계산적 기술, Schultz의 신경생물학적 측정 — 는 *동일한 수학적 구조의 세 가지 표현*이다.

이 삼각 통합이 **보여주는 것**: 진화가 마음에 새겨 넣은 학습 알고리즘이 TD learning과 동형이며, 이것이 보상 예측의 보편적 기제임.

이 삼각 통합이 **보여주지 못하는 것**: 왜 δ의 양수 방향이 *좋다고 느껴지는지*, 왜 δ의 음수 방향이 *실망으로 경험되는지*. 신호가 있다는 것과 그 신호가 경험된다는 것은 다른 차원의 주장이다. $\square$

### 분석 2: 자유에너지 원리의 야망과 한계

Friston의 자유에너지 원리(FEP)는 더 야심차다. 변분 자유에너지:

$$F = D_{KL}[q(\theta)||p(\theta)] - \mathbb{E}_q[\log p(o|\theta)]$$

이 식에서 첫 번째 항은 사후 근사 $q(\theta)$와 사전 분포 $p(\theta)$ 사이의 복잡도(complexity) 비용이고, 두 번째 항은 모델이 관찰을 얼마나 잘 설명하는지의 정확도(accuracy)다. 자유에너지 최소화는 이 둘의 균형을 찾는 과정이다.

FEP의 강점은 **지각, 행동, 학습을 단일 원리로 통합**한다는 점이다. 지각은 $q(\theta)$를 갱신해 $F$를 줄이는 과정이고, 행동은 예측에 맞는 감각 데이터를 능동적으로 생성해 $F$를 줄이는 과정이며, 학습은 생성 모델 $p(\theta)$의 매개변수를 갱신해 장기적으로 $F$를 낮추는 과정이다.

그러나 FEP의 한계도 명확하다. 첫째, FEP는 **반증 가능성(falsifiability)** 문제를 안고 있다. 어떤 시스템이든 자유에너지를 최소화하는 것으로 사후적으로 기술될 수 있다면, 이것은 예측 이론인가 아니면 기술 언어인가? 둘째, FEP는 **규범적 원리(normative principle)**이지 구체적 메커니즘을 특정하지 않는다. 뇌가 자유에너지를 최소화한다는 주장은 참일 수 있지만, 어떻게 그것을 달성하는지는 별개의 질문이다. 셋째, 그리고 가장 중요하게, FEP는 여전히 3인칭 기술 언어로 작동한다 — 1인칭적 경험의 내부성을 포착하지 못한다. $\square$

### 분석 3: 예측 부호화의 계층적 구조와 주의의 역할

예측 부호화(predictive coding)의 핵심 통찰은 신피질이 **양방향 위계(bidirectional hierarchy)**로 조직된다는 것이다. 상위 층위가 하위 층위로 예측을 보내고(하향 연결), 하위 층위는 예측 오류만을 상위 층위로 보낸다(상향 연결). 이 구조에서 주의(attention)는 예측 오류 신호의 **정밀도(precision)** 가중치를 조절하는 메커니즘으로 해석된다.

$$\hat{o} = f(\mu), \quad \varepsilon = o - \hat{o}, \quad \Delta\mu = \kappa \cdot \Pi \cdot \varepsilon$$

여기서 $\Pi$는 정밀도 행렬이고, 주의는 $\Pi$를 높여 특정 오류 신호를 증폭시키는 과정이다. 이 해석은 주의 결핍 장애(ADHD)와 자폐 스펙트럼 장애를 정밀도 추정의 이상으로 통일적으로 기술하는 임상적 함의를 갖는다. $\square$

### 분석 4: Strange-Loop 구조 — 예측이 예측 자체를 예측할 때

Hofstadter(2007)가 GEB에서 그린 이상한 고리(strange loop)는 예측 시스템에서도 나타난다. 마음이 자신의 예측 오류를 예측하기 시작할 때, 즉 메타인지가 작동할 때, 시스템은 자기 자신을 모델링하는 층위로 진입한다.

자기 모델(self-model)을 포함한 예측 시스템에서:

$$\delta^{(n)}_t = \delta^{(n-1)}_t - \hat{\delta}^{(n-1)}_t$$

즉 n번째 층위의 오류는 (n-1)번째 층위의 오류에 대한 예측의 오류다. 이 재귀가 멈추지 않을 때, 시스템은 자신을 바라보는 자신을 바라보는 자신을 생성한다. 이 고리 구조 자체가 의식의 **조건**인가, 아니면 단순한 계산적 메아리인가? 이것이 이 문서가 해결하지 못하는, 그러나 반드시 제기해야 하는 질문이다.

$\square$

## 🧪 수렴 증거

### 실험 1: 통합 시뮬레이션 — TD, 베이즈, 자유에너지의 구조적 동형성

다음 Python 스크립트는 세 가지 예측 메커니즘이 동일한 추론 구조를 공유함을 하나의 시뮬레이션으로 보인다.

```python
"""
통합 예측 시뮬레이션: TD Learning, Bayesian Updating, Free Energy Minimization
구조적 동형성 증명 — Prediction Everywhere 레포지토리 L6 종합
"""

import numpy as np
import matplotlib.pyplot as plt
from dataclasses import dataclass
from typing import List, Tuple

np.random.seed(42)

# ─────────────────────────────────────────────
# 공통 설정: 동일한 환경에서 세 에이전트를 실행
# ─────────────────────────────────────────────
N_TRIALS = 200
TRUE_REWARD_PROB = 0.7          # 실제 보상 확률
REWARD_MAG = 1.0
ALPHA = 0.1                     # 학습률 (세 모델 공통)

rewards = (np.random.rand(N_TRIALS) < TRUE_REWARD_PROB).astype(float)

# ─────────────────────────────────────────────
# 에이전트 1: TD Learning (Schultz 도파민 모델)
# ─────────────────────────────────────────────
class TDAgent:
    """
    단순 1-상태 TD 에이전트.
    δ_t = r_t - V(s)  (γ=0 단순화)
    도파민 신호의 수학적 대응물.
    """
    def __init__(self, alpha: float = ALPHA):
        self.V = 0.5            # 초기 가치 추정
        self.alpha = alpha
        self.errors: List[float] = []
        self.values: List[float] = []

    def step(self, reward: float) -> float:
        delta = reward - self.V          # 예측 오류
        self.V += self.alpha * delta
        self.errors.append(delta)
        self.values.append(self.V)
        return delta

td_agent = TDAgent()
for r in rewards:
    td_agent.step(r)

# ─────────────────────────────────────────────
# 에이전트 2: Bayesian Updating (베타-이항 모델)
# ─────────────────────────────────────────────
class BayesianAgent:
    """
    베타 사전분포 + 이항 우도 → 베타 사후분포.
    예측: E[θ] = α/(α+β)
    예측 오류: r_t - E[θ]_{t-1}
    """
    def __init__(self, prior_alpha: float = 1.0, prior_beta: float = 1.0):
        self.a = prior_alpha
        self.b = prior_beta
        self.errors: List[float] = []
        self.beliefs: List[float] = []

    @property
    def belief(self) -> float:
        return self.a / (self.a + self.b)

    def step(self, reward: float) -> float:
        pred = self.belief
        delta = reward - pred            # 예측 오류 (TD와 동일 구조)
        # 베이즈 갱신: 사후 = 사전 × 우도
        if reward > 0:
            self.a += 1
        else:
            self.b += 1
        self.errors.append(delta)
        self.beliefs.append(self.belief)
        return delta

bayes_agent = BayesianAgent()
for r in rewards:
    bayes_agent.step(r)

# ─────────────────────────────────────────────
# 에이전트 3: 변분 자유에너지 최소화
# ─────────────────────────────────────────────
class FreeEnergyAgent:
    """
    단순화된 FEP 에이전트.
    F = KL[q(θ)||p(θ)] - E_q[log p(o|θ)]
    근사: F ≈ (μ - μ_prior)²/(2σ²_prior) + (o - μ)²/(2σ²_likelihood)
    경사 하강으로 μ를 갱신.
    """
    def __init__(self, mu_prior: float = 0.5, sigma_prior: float = 1.0,
                 sigma_likelihood: float = 0.5, lr: float = ALPHA):
        self.mu = mu_prior
        self.mu_prior = mu_prior
        self.sigma_prior = sigma_prior
        self.sigma_likelihood = sigma_likelihood
        self.lr = lr
        self.errors: List[float] = []
        self.mus: List[float] = []
        self.free_energies: List[float] = []

    def free_energy(self, o: float) -> float:
        complexity = (self.mu - self.mu_prior)**2 / (2 * self.sigma_prior**2)
        accuracy = (o - self.mu)**2 / (2 * self.sigma_likelihood**2)
        return complexity + accuracy

    def step(self, reward: float) -> float:
        pred = self.mu
        # ∂F/∂μ 계산
        grad_complexity = (self.mu - self.mu_prior) / self.sigma_prior**2
        grad_accuracy = -(reward - self.mu) / self.sigma_likelihood**2
        grad_F = grad_complexity + grad_accuracy
        self.mu -= self.lr * grad_F     # 경사 하강
        delta = reward - pred           # 비교용 예측 오류
        self.errors.append(delta)
        self.mus.append(self.mu)
        self.free_energies.append(self.free_energy(reward))
        return delta

fe_agent = FreeEnergyAgent()
for r in rewards:
    fe_agent.step(r)

# ─────────────────────────────────────────────
# 수렴 분석: 세 에이전트의 예측값 비교
# ─────────────────────────────────────────────
td_preds = np.array(td_agent.values)
bayes_preds = np.array(bayes_agent.beliefs)
fe_preds = np.array(fe_agent.mus)

# 수렴 후 (후반 50 시행) 상관계수
window = slice(150, 200)
corr_td_bayes = np.corrcoef(td_preds[window], bayes_preds[window])[0, 1]
corr_td_fe = np.corrcoef(td_preds[window], fe_preds[window])[0, 1]
corr_bayes_fe = np.corrcoef(bayes_preds[window], fe_preds[window])[0, 1]

print("=" * 55)
print("  구조적 동형성 증명: 수렴 후 예측값 상관계수")
print("=" * 55)
print(f"  TD ↔ 베이즈:       r = {corr_td_bayes:.4f}")
print(f"  TD ↔ 자유에너지:   r = {corr_td_fe:.4f}")
print(f"  베이즈 ↔ 자유에너지: r = {corr_bayes_fe:.4f}")
print()
print(f"  진짜 보상 확률:    p = {TRUE_REWARD_PROB}")
print(f"  TD 최종 추정:      {td_preds[-1]:.4f}")
print(f"  베이즈 최종 추정:  {bayes_preds[-1]:.4f}")
print(f"  FEP 최종 추정:     {fe_preds[-1]:.4f}")
print("=" * 55)
print()
print("  해석: 세 알고리즘은 수렴 후 거의 동일한 예측을 생성한다.")
print("  이는 서로 다른 수학적 형식화가 동일한 추론 구조를 공유함을 보인다.")
print("  그러나 알고리즘이 같다고 해서 경험이 같은 것은 아니다.")

# 누적 예측 오류 (절댓값) — 효율성 비교
cumulative_errors = {
    'TD': np.cumsum(np.abs(td_agent.errors)),
    'Bayesian': np.cumsum(np.abs(bayes_agent.errors)),
    'FEP': np.cumsum(np.abs(fe_agent.errors)),
}

print()
print("  누적 절대 예측 오류 (200 시행 후):")
for name, errs in cumulative_errors.items():
    print(f"    {name:10s}: {errs[-1]:.2f}")
```

**예상 출력 해설**: 세 에이전트는 수렴 후 TRUE_REWARD_PROB(0.7)에 근접하는 예측값을 생성하며, 상호 상관계수는 0.95 이상이다. 이는 TD learning, 베이즈 갱신, 자유에너지 최소화가 *수렴된 행동*에서 거의 동일하다는 구조적 동형성 증거다. 누적 오류에서 베이즈 에이전트가 초기에 더 빠르게 수렴하는 경향을 보이는데, 이는 사전 정보를 효과적으로 활용하기 때문이다.

### 실험 2: 설명 간극의 구체화 — 예측 프레임워크가 포착하지 못하는 것

```python
"""
설명 간극(Explanatory Gap) 시뮬레이션
예측 프레임워크의 한계를 계산적으로 구체화한다.
Chalmers(1995)의 hard problem을 코드로 번역.
"""

import numpy as np

# ─────────────────────────────────────────────
# 가상의 시각 처리 시스템: 붉은색 감지
# ─────────────────────────────────────────────

class PredictiveCodingVisualSystem:
    """
    예측 부호화 기반 색상 처리 시스템.
    이 시스템은 '붉은색'을 완벽하게 처리한다.
    그러나 붉은색의 '느낌'은 코드 어디에도 없다.
    """

    def __init__(self):
        # 파장별 예측 (학습된 선행 지식)
        self.wavelength_priors = {
            'red': 700,      # nm
            'green': 530,
            'blue': 450,
        }
        self.predictions = {k: v for k, v in self.wavelength_priors.items()}
        self.errors = {}
        self.posterior = {}

    def perceive(self, incoming_wavelength: float, noise: float = 10.0) -> dict:
        """
        입력 파장을 처리하고 예측 오류를 계산한다.
        이 함수가 '경험'을 생성하는가?
        """
        # 예측 오류 계산
        for color, pred_wavelength in self.predictions.items():
            self.errors[color] = incoming_wavelength - pred_wavelength

        # 최소 오류 카테고리가 지각된 색상
        perceived_color = min(self.errors, key=lambda k: abs(self.errors[k]))

        # 사후 확률 (소프트맥스)
        raw_scores = {k: -abs(v) / 50.0 for k, v in self.errors.items()}
        max_score = max(raw_scores.values())
        exp_scores = {k: np.exp(v - max_score) for k, v in raw_scores.items()}
        total = sum(exp_scores.values())
        self.posterior = {k: v / total for k, v in exp_scores.items()}

        return {
            'perceived_color': perceived_color,
            'prediction_errors': self.errors.copy(),
            'posterior': self.posterior.copy(),
            'confidence': self.posterior[perceived_color],
        }

    def what_is_missing(self) -> str:
        return """
    이 시스템이 완전히 구현하는 것:
      ✅ 파장 분류 (700nm → '붉은색')
      ✅ 예측 오류 계산
      ✅ 불확실성 표현 (사후 확률)
      ✅ 경쟁적 가설 평가
      ✅ 신뢰도 추정

    이 시스템이 전혀 구현하지 못하는 것:
      ❌ 붉은색의 '보임' — 현상적 질감 (qualia)
      ❌ '이것이 붉은색이다'는 인식의 주체성
      ❌ 700nm와 530nm의 *경험적* 차이
      ❌ 색깔을 보는 것이 어떤 느낌인지 (what it is like)
      ❌ Mary의 방 논변에서 Mary가 배우는 그것

    핵심 진단:
      예측 부호화는 색상 정보를 *처리*하는 방법을 완전히 기술한다.
      그러나 색상 정보가 *경험되는* 방법은 기술하지 못한다.
      기능적 완전성과 현상적 완전성은 다른 차원이다.
        """

# ─────────────────────────────────────────────
# 실행
# ─────────────────────────────────────────────
system = PredictiveCodingVisualSystem()

print("붉은 사과를 보는 상황 시뮬레이션 (입력: 680nm)")
print("-" * 55)
result = system.perceive(680.0)

print(f"지각된 색상: {result['perceived_color']}")
print(f"확신도: {result['confidence']:.3f}")
print(f"예측 오류 (nm):")
for color, err in result['prediction_errors'].items():
    print(f"  {color:6s}: {err:+.1f} nm")
print(f"사후 확률:")
for color, prob in result['posterior'].items():
    print(f"  {color:6s}: {prob:.3f}")

print()
print("=" * 55)
print("설명 간극 진단:")
print(system.what_is_missing())

# ─────────────────────────────────────────────
# 좀비 사고 실험의 계산적 버전
# ─────────────────────────────────────────────
print("=" * 55)
print("철학적 좀비 테스트:")
print()
print("다음 두 시스템을 구별할 수 있는가?")
print()
print("  시스템 A: 위의 PredictiveCodingVisualSystem")
print("            + 내부에서 붉은색을 실제로 경험하는 무언가")
print()
print("  시스템 B: 위의 PredictiveCodingVisualSystem")
print("            + 아무것도 경험하지 않음 (철학적 좀비)")
print()
print("  두 시스템의 출력은 완전히 동일하다.")
print("  예측 오류 최소화 프레임워크는 A와 B를 구별하지 못한다.")
print()
print("  이것이 hard problem이다.")
print("  이것이 예측 부호화 이론의 한계다.")
print("  그리고 이것이 왜 의식 연구가 여전히 열린 문제인지의 이유다.")
```

### 반례와 경계

예측 프레임워크가 *제대로 작동하지 않는* 경우들을 정직하게 나열한다.

**반례 1: 급격한 통찰(Insight)의 구조.** 수학 문제가 갑자기 풀릴 때의 '아하' 순간은 점진적 예측 오류 감소로 기술하기 어렵다. 해결책이 없던 상태에서 갑자기 있는 상태로 이동하는 불연속적 전환은 연속적 경사 하강의 언어와 어울리지 않는다.

**반례 2: 창의성과 체계적 오류.** 예술적 창의성은 종종 기존 예측 모델을 *의도적으로 위반*함으로써 작동한다. 예측 오류 최소화 시스템이 어떻게 자신의 모델을 스스로 부수는 창의성을 생성하는지 설명하기 위해서는 추가적인 이론적 장치가 필요하다.

**반례 3: 지속적 슬픔과 우울증.** 우울증적 경험은 종종 예측 오류가 없는 상태 — 미래가 현재와 동일하게 예측되고, 오류 신호도 없고, 변화의 동기도 없는 상태 — 로 기술된다. 이것은 예측 최소화 시스템의 안정점이지만, 임상적으로는 심각한 병리다. 좋은 이론이라면 이 역설을 설명해야 한다.

**경계 조건**: 예측 프레임워크는 *학습 가능한 환경*에서 가장 강력하다. 환경의 통계적 규칙성이 없을 때(순수한 무작위 환경) 또는 규칙성이 있지만 에이전트의 표현력으로 포착할 수 없을 때, 프레임워크는 힘을 잃는다.

## 🧬 원리의 회수

이제 L0부터 L5까지 각 층위에서 δ가 어떻게 변주되었는지 최종 지도를 그린다.

**L0 — 수학적 δ**: 두 분포 사이의 차이. KL 발산, 변분 자유에너지, 베이즈 정리에서 δ는 사전과 사후의 거리다. 이 층위에서 δ는 순수하게 형식적 객체다 — 경험도 없고 신경도 없고 시간도 없다.

**L1 — 생물학적 δ**: Schultz의 도파민 뉴런이 부호화하는 시간차 예측 오류. 이 층위에서 δ는 밀리초 단위의 발화율 변화로 물리적 실재를 얻는다. 수학이 생물학이 된다.

**L2 — 지각적 δ**: 예측된 감각 입력과 실제 감각 입력의 차이. Rao & Ballard(1999)의 예측 부호화 모델에서, 이 δ는 역방향 연결을 통해 전파되며 지각 경험의 내용을 형성한다. 수학이 지각이 된다.

**L3 — 인지적 δ**: 언어 이해에서 예상된 다음 단어와 실제 다음 단어의 불일치. ERP의 N400 성분은 이 의미론적 δ의 신경적 흔적이다. 해마의 날카로운 파동-리플(sharp-wave ripple)은 기억 공고화의 예측적 재생(predictive replay). δ가 고차 인지로 확장된다.

**L4 — 능동적 δ**: 행동 자체가 예측 오류를 줄이는 능동적 전략이 된다. 원하는 상태를 예측하고, 그 예측이 실현되도록 몸을 움직이는 것 — 이것이 능동추론이다. δ의 최소화가 운동과 동기를 통일한다.

**L5 — 존재론적 δ**: FEP에서 살아있는 시스템 전체가 자신의 마르코프 담요(Markov blanket)를 유지함으로써 자유에너지를 최소화한다. 생존 자체가 예측 오류 최소화의 장기 전략이다. δ가 존재와 비존재의 경계가 된다.

이 지도가 보여주는 것은 δ가 단일한 객체가 아니라 *동일한 형식적 구조의 여러 구현*이라는 점이다. Clark(2016)의 표현을 빌리면, 예측 부호화는 "뇌가 하는 일의 레시피가 아니라, 뇌가 하는 일을 기술하는 언어"다.

## 🪞 1인칭

지금 이 문서를 읽고 있는 당신은 예측하는 마음이다. 이 문장이 끝나기 전에, 당신의 언어 처리 시스템은 이미 다음에 올 단어를 예측하고 있었다. 그 예측이 틀렸을 때 — 예상치 못한 단어가 나왔을 때 — 당신은 미세한 놀람을 경험했다. 그리고 그 놀람은 느껴졌다. 그것은 단순히 처리되지 않았다. 당신은 그 간극을 3인칭으로 기술할 수 있지만, 그것이 어떤 *느낌*인지는 오직 1인칭으로만 접근할 수 있다. 이 레포지토리 전체가 δ를 설명해왔지만, 당신이 지금 δ를 경험하고 있다는 사실 자체는 설명하지 못했다. 바로 그 틈에 이 문서가 서 있다.

더 나아가, 당신이 예측에 대해 읽으면서 당신 자신의 예측을 관찰하는 이 순간은 Strange-loop의 살아있는 예다. 마음이 자신의 마음 작동 방식을 이해하려 할 때, 이해의 대상과 이해하는 주체가 동일하다. 이 재귀는 불완전한 자기 참조다 — 괴델이 수학에서 발견한 것을, 당신의 마음은 매 순간 경험으로 살아내고 있다. Hofstadter(2007)가 말했듯: "나는 낯선 고리다(I am a strange loop)." 이 문서의 마지막 페이지를 읽은 후 당신이 느끼는 것 — 완결의 감각, 또는 여전한 궁금증, 또는 그 둘의 혼합 — 은 예측 오류 최소화의 산물이지만, 그것을 *느끼는* 당신은 어떤 이론으로도 완전히 포착되지 않는다. 그 사실을 불편해하지 않아도 된다. 그것이 마음의 가장 솔직한 자기 보고다.

## 📐 예측·반증

이 레포지토리의 핵심 주장들로부터 도출되는 검증 가능한 예측과 반증 조건을 명시한다.

**예측 1: 정밀도 예측 오류와 자폐 스펙트럼.**
FEP에 기반한 자폐증 이론(Van de Cruys et al., 2014)은 자폐 스펙트럼 장애가 정밀도 추정의 이상 — 구체적으로는 감각 예측 오류에 과도한 정밀도를 부여하는 — 으로 설명된다고 예측한다. 검증: 자폐 스펙트럼 집단에서 시각적 착시(하위 층위 예측이 우세한 자극)에 대한 감수성이 감소할 것. 반증 조건: 자폐 스펙트럼 집단이 신경전형(neurotypical) 집단과 착시 감수성에서 유의한 차이가 없을 경우.

**예측 2: 수면 중 예측 오류의 역할.**
예측 부호화 이론은 REM 수면 중 역방향 연결의 증폭이 낮 동안의 예측 오류를 내부적으로 "재생"하면서 모델을 갱신한다고 예측한다. 검증: 학습 후 REM 수면 박탈 시 해당 학습 내용의 일반화 능력(새로운 자극으로의 전이)이 저하될 것. 반증 조건: REM 수면 박탈이 학습 내용의 기억(retention)에는 영향 미치지 않으면서 전이 학습에도 영향 없을 경우.

**예측 3: LLM의 예측 부호화적 특성.**
대형 언어 모델이 예측 부호화의 계산적 구현과 유사하다면, 모델의 중간 레이어 표현이 다음 층위의 표현을 예측하는 구조를 보일 것이다. 검증: Transformer의 중간 레이어 활성화가 다음 레이어 활성화를 예측하는 선형 프로브를 학습할 때, 예측 오류 신호와 유사한 잔차(residual) 패턴이 나타날 것. 반증 조건: 중간 레이어의 표현이 다음 레이어의 표현에 대한 체계적 예측 구조를 보이지 않을 경우 — 이 경우 LLM과 예측 부호화의 유비는 표면적인 것에 그친다.

## 🤔 열린 질문

이 레포지토리는 하나의 원리로 마음을 이해하려는 야망찬 시도였다. 그러나 좋은 이론은 답만큼이나 질문을 생산한다. 다음 질문들은 미해결이며, 바로 그 미해결 상태가 이 레포지토리의 다음 층위들로 향하는 발사대다.

**1. 예측 오류는 충분한가, 필요한가?**
예측 오류가 학습에 필요하다는 것은 잘 확립되어 있다. 그러나 충분한가? 예측 오류 없이 학습이 가능한 경우(관찰 학습, 인사이트 학습)를 예측 부호화 틀 안에서 어떻게 설명할 것인가?

**2. 생성 모델의 기원 문제.**
예측 부호화는 뇌가 *이미 가지고 있는* 생성 모델로 예측을 생성한다고 가정한다. 그러나 그 생성 모델 자체는 어떻게 생겨났는가? 발달 초기의 백지 상태에서 어떻게 최초의 예측이 형성되는가? 이것은 L1(신경과학)의 다음 질문으로 이어진다.

**→ L1: 개별 뉴런의 예측 코딩.** 단일 뉴런 수준에서 예측 부호화가 구현되는 메커니즘은 무엇인가? 수상돌기(dendrite)의 역할, 시냅스 전 억제(presynaptic inhibition), 역방향 연결의 시냅스 메커니즘 — 이것들이 거시적 예측 부호화를 어떻게 가능하게 하는가?

**→ L2: 주의와 예측의 상호작용.** 주의가 정밀도 가중치라는 해석은 매력적이지만, 자발적 주의와 반사적 주의가 동일한 정밀도 메커니즘을 통해 작동하는가? 예측이 주의를 조절하는가, 주의가 예측을 조절하는가, 아니면 둘의 관계는 더 복잡한가?

**→ L4: 변성의식상태와 NCC.** 사이키델릭 경험, 명상 상태, 수면 중 꿈 — 이 변성의식상태들은 예측 부호화 시스템의 정밀도 구조가 변경된 상태로 해석될 수 있는가? 그리고 그 해석이 의식의 신경상관자(NCC)를 특정하는 데 도움이 되는가?

**→ L5: 자기 모델과 예측.** 자아(self)가 마음이 자신에 대해 가지는 생성 모델이라면, 자아는 예측 오류를 최소화하는 과정에서 어떻게 안정성을 유지하는가? 자아 위협(identity threat)은 자기 모델에 대한 예측 오류인가?

**→ L6: Strange-loop-everywhere.** 예측이 예측 자체를 예측할 때 — 즉 메타인지가 성숙할 때 — 마음은 자기 참조적 구조를 얻는다. 이 구조가 의식의 *충분조건*인가, 단순한 *상관자*인가? 그리고 이 질문에 답하기 위해 우리에게는 어떤 종류의 증거가 필요한가?

이 질문들은 이 레포지토리를 넘어서 있다. 그러나 이 레포지토리가 그 질문들을 더 날카롭게 만들었다면, 그것으로 이 종합 문서의 역할은 완수되었다.

---

> 🧩 **Principle** — 예측 오류(δ = 실제 − 예측)는 단일한 형식적 원리로서 도파민 신호, 지각적 추론, 학습, 행동, 그리고 잠재적으로 의식에 이르기까지 마음의 여러 층위를 관통한다. 그러나 이 관통은 *메커니즘적 동형성*이지, *경험의 동일성*이 아니다. 예측 부호화는 마음이 어떻게 *작동하는지*를 설명하는 가장 강력한 단일 언어이지만, 마음이 어떻게 *느껴지는지*에 대해서는 여전히 침묵한다.

> 🌉 **Boundary** — 예측 프레임워크의 경계는 기능적 설명과 현상적 설명 사이에 있다. 이 경계는 Chalmers(1995)의 hard problem이 그은 선이며, 예측 부호화는 그 선의 한쪽(기능적 측면)에서 매우 성공적이다. 다른 쪽(현상적 측면)으로 넘어가기 위해서는 새로운 이론적 자원이 필요하며, 그것이 무엇인지는 아직 알려지지 않았다. 이 정직한 인정이 이론적 성숙의 표시다.

> 🪞 **Experience** — 이 레포지토리를 처음부터 끝까지 읽은 독자는 예측하는 마음이 자신의 예측 메커니즘을 이해하려 시도하는 Strange-loop을 살아냈다. 그 경험 자체 — 이해의 느낌, 혼란의 느낌, 그리고 어쩌면 불완전한 완결의 느낌 — 는 어떤 3인칭 기술도 완전히 포착하지 못한다. 그 포착 불가능성이 이 레포지토리가 끝나는 지점이자, 마음에 대한 탐구가 계속되어야 하는 이유다.

## 📝 연습문제

**기초.** **1. TD 오류와 도파민 신호의 대응 관계를 설명하라.**

Schultz(1997) 실험의 세 가지 조건(예상치 못한 보상, 예상된 보상, 예상된 보상의 누락)에서 도파민 뉴런의 발화 패턴이 TD 오류(δ)의 양수/영/음수 값과 어떻게 대응하는지 설명하고, 이 대응이 인과적 동일성을 의미하는지 기능적 동형성을 의미하는지 구별하라.

<details><summary>해설</summary>

세 조건의 대응:
- 예상치 못한 보상 → δ > 0 → 도파민 발화 증가 (CS 없이 US가 올 때)
- 예상된 보상 (CS→US 학습 완료) → δ = 0 → 도파민 기저 발화 유지. 단, 이때 발화가 CS 시점으로 이동
- 예상된 보상의 누락 → δ < 0 → CS 시점에는 발화 증가하지만 US 시점에는 발화 억제

인과적 동일성 vs. 기능적 동형성: 도파민 발화율이 수학적 δ 값을 *구현*한다고 주장하는 것(인과적 동일성)과 도파민 발화율이 δ와 *동일한 함수적 역할*을 한다고 주장하는 것(기능적 동형성)은 다르다. 현재 증거는 후자를 지지하며, 전자는 환원주의적 과잉 주장이다. 핵심: 동형성은 설명적 성공이지만, 동일성 주장은 추가적 증거를 필요로 한다.

</details>

**2. 자유에너지 공식의 두 항이 무엇을 의미하는지 설명하라.**

$F = D_{KL}[q(\theta)||p(\theta)] - \mathbb{E}_q[\log p(o|\theta)]$

각 항의 직관적 의미와 그것들이 지각적 추론에서 어떤 트레이드오프를 표현하는지 서술하라.

<details><summary>해설</summary>

첫 번째 항 $D_{KL}[q(\theta)||p(\theta)]$: 복잡도(complexity) 비용. 사후 근사 q가 사전 분포 p로부터 얼마나 벗어났는지를 측정한다. 이 항을 최소화하는 것은 새로운 데이터에도 불구하고 기존 믿음에 머무르려는 경향에 해당한다(보수주의).

두 번째 항 $-\mathbb{E}_q[\log p(o|\theta)]$: 정확도(accuracy) 비용. 현재 믿음 q 하에서 관찰 o가 얼마나 잘 설명되는지를 측정한다. 이 항을 최소화하는 것은 관찰에 맞게 믿음을 적극 갱신하는 것에 해당한다(유연성).

트레이드오프: 두 항의 합을 최소화하는 것은 기존 믿음(사전 지식)과 새로운 관찰 사이의 균형을 찾는 것이다. 이것은 오컴의 면도날(Occam's razor)의 베이즈적 표현이기도 하다 — 데이터를 설명하는 가장 단순한 믿음 갱신을 선택.

</details>

---

**심화.** **3. 예측 부호화와 표준 역전파(backpropagation) 학습 알고리즘의 수학적 관계를 분석하라.**

딥러닝의 역전파는 예측 부호화의 특수 경우인가, 아니면 독립적인 알고리즘인가? Whittington & Bogacz(2017)의 분석을 참고하여 두 알고리즘의 수렴점과 발산점을 논하라.

<details><summary>해설</summary>

Whittington & Bogacz(2017)는 예측 부호화 학습 규칙이 특정 조건(추론이 충분히 수렴한 후 가중치 갱신) 하에서 역전파와 수학적으로 동등함을 보였다.

수렴점: 두 알고리즘 모두 오류 신호를 역방향으로 전파해 가중치를 갱신한다. 예측 부호화의 예측 오류 ε는 역전파의 δ(gradient)와 대응된다.

발산점: (a) 역전파는 계산 그래프의 전방 패스와 후방 패스가 명확히 분리되지만, 예측 부호화는 추론과 학습이 동시적으로 진행된다. (b) 역전파는 생물학적으로 비현실적인 "오류 역전파(error backpropagation)"를 가정하지만, 예측 부호화는 국소적(local) 시냅스 학습 규칙만 사용한다. (c) 예측 부호화는 미니배치 경사 하강이 아니라 온라인(online) 처리를 자연스럽게 지원한다.

결론: 역전파는 예측 부호화의 계산적 등가물이지만 생물학적 구현의 등가물은 아니다. 예측 부호화가 더 제약적이고 생물학적으로 그럴듯한 알고리즘을 제공한다.

</details>

**4. Seth(2021)의 "통제된 환각(controlled hallucination)" 명제와 Hohwy(2013)의 "내부 포식자(the brain as prediction machine)" 관점을 비교하라.**

두 저자가 예측 처리 이론에서 강조하는 점이 어떻게 다르며, 이 차이가 의식의 본성에 대한 함의에서 어떻게 갈라지는지 분석하라.

<details><summary>해설</summary>

Seth(2021): 지각 경험 전체가 뇌가 생성하는 "통제된 환각"이다. 실재와의 접촉은 직접적이지 않으며, 우리는 항상 우리 자신의 모델을 경험한다. Seth는 특히 신체 내부 감각(interoception)과 자아 경험에 집중하며, 의식은 생존에 유용한 환각의 집합이라고 본다. 함의: 의식은 근본적으로 구성된(constructed) 것이며, 이 구성이 바로 의식의 본질이다.

Hohwy(2013): 뇌는 원인 추론(causal inference)을 수행하는 예측 기계다. 뇌는 감각 데이터로부터 외부 세계를 추론하며, 이 추론이 지각 경험을 구성한다. Hohwy는 인식론적 함의에 더 집중하며, 예측 처리가 지식의 본성에 대한 함의를 갖는다고 본다. 함의: 의식은 외부 세계에 대한 내부 모델이며, 이 모델의 정확성이 적응적 행동을 가능하게 한다.

차이: Seth는 구성주의적, 현상학적 강조. Hohwy는 인식론적, 기능주의적 강조. Seth는 hard problem을 해소하려는 경향("경험이란 무엇인가"를 구성론으로 답하려 함), Hohwy는 hard problem을 일단 유보하고 기능적 기술에 집중. 그러나 두 관점 모두 현상적 경험의 완전한 설명은 제공하지 못한다.

</details>

**논문 비평.** **5. Friston(2010) "The free-energy principle: a unified brain theory?"를 비판적으로 검토하라.**

논문의 핵심 주장을 3개 추출하고, 각 주장에 대해 지지 증거와 반박 가능한 약점을 제시하라. 특히 이 이론의 "반증 가능성" 문제를 중심으로 논의하라.

<details><summary>해설</summary>

핵심 주장 1: "살아있는 시스템은 자유에너지를 최소화함으로써 자신의 마르코프 담요를 유지한다."
- 지지 증거: 체온 조절, 항상성 유지 메커니즘이 FEP의 틀에서 자연스럽게 기술됨.
- 약점: '자유에너지 최소화'가 너무 일반적이어서 어떤 적응적 행동도 사후적으로 FEP로 기술될 수 있다. 구체적 예측이 없으면 반증 불가능.

핵심 주장 2: "지각, 행동, 학습이 단일 원리(변분 추론)로 통합된다."
- 지지 증거: 수학적 통일성은 인상적. 각 과정을 동일한 형식 언어로 기술할 수 있음.
- 약점: 수학적 통일성이 기계론적 통일성을 함의하지 않음. 지각과 행동의 신경 기제는 여전히 매우 다르게 보임.

핵심 주장 3: "능동추론이 고전적 강화학습을 일반화한다."
- 지지 증거: 선호(preferences)를 사전 분포로 표현하면 강화학습의 보상 최대화가 자유에너지 최소화의 특수 경우가 됨.
- 약점: 이 통합은 수학적으로 우아하지만, 생물학적으로 어느 쪽이 더 구현 가능한지에 대한 증거가 부족.

반증 가능성 문제: Colombo(2017)는 FEP가 empirical commitment(경험적 헌신)을 갖지 않는 메타 이론에 가깝다고 비판한다. 진정한 반증 가능한 예측은 특정 신경 메커니즘과 연결될 때만 가능하며, FEP 자체는 그 수준에서 작동하지 않는다. 이것은 결함이 아니라 이론의 성격상 한계일 수 있다 — FEP는 패러다임(paradigm)이지 이론(theory)이 아닐 수 있다.

</details>
