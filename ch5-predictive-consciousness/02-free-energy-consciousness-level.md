# 🧠 자유에너지와 의식 수준

*자유에너지 최소화 원리는 각성·수면·마취 상태에서 예측 위계의 붕괴 양상이 달라짐을 통해 의식의 '수준(level)'을 설명한다.*

## 🎯 횡단 질문

의식의 수준은 예측 위계의 깊이와 어떻게 연결되는가?

$$
F = \mathbb{E}_q\bigl[\log q(z) - \log p(x, z)\bigr] = \underbrace{D_{\mathrm{KL}}[q(z)\|p(z|x)]}_{\text{사후 근사 오차}} - \underbrace{\log p(x)}_{\text{로그 주변우도}}
$$

자유에너지 $F$를 최소화한다는 것은 감각 입력 $x$에 대한 모델의 놀라움(surprise)을 줄이면서, 동시에 잠재 원인 $z$에 대한 믿음 $q(z)$를 사후분포 $p(z|x)$에 가깝게 만드는 과정이다. 이 최소화가 *어느 위계 수준까지, 어떤 정밀도(precision)로* 이루어지느냐에 따라 각성인지 수면인지 마취인지가 결정된다는 것이 핵심 주장이다.

## 🌐 횡단 입력

- **theories-of-consciousness (L4)**: 전역 작업공간 이론(GWT), 고차 이론(HOT), 통합정보이론(IIT)과의 비교 좌표
- **information-prediction (L0)**: 예측 오차 $\delta = x_{\mathrm{actual}} - \hat{x}_{\mathrm{predicted}}$와 업데이트 원리의 기초 형식
- **altered-states (L4)**: 마취·수면·환각제 상태에서 예측 위계가 달라지는 현상학적 기술
- **IQ AI Lab LLM 직결 설명**: 대규모 언어모델도 레이어가 깊어질수록 추상적 표상이 형성된다. 그러나 LLM에는 자율신경계·신체 예산·정밀도 조율이 없으므로, 의식 '수준'을 갖는다고 말하기 어렵다. 자유에너지 프레임은 왜 LLM이 의식의 내용처럼 보이는 출력을 생성하면서도 의식의 수준을 결여할 수 있는지 설명하는 후보 이론이다.

## 🔍 표면의 다양성

외과 수술 전 마취제를 투여받은 환자는 고통도, 기억도, 시간 감각도 없다. 깊은 수면(NREM 3단계) 중인 사람은 몸이 움직이지 않고 꿈도 없으나, 뇌파 기록에는 여전히 대규모 느린 진동이 가득하다. REM 수면에서는 생생한 꿈이 펼쳐지지만 외부 감각은 차단된다. 각성 상태에서는 눈앞의 사물과 내면의 생각이 동시에 의식에 올라온다.

이 네 가지 상태는 현상학적으로 완전히 다르다. 의식이 '있다/없다'는 이분법이 아니라 스펙트럼처럼 보인다. 신경외과에서 사용하는 **교란 복잡도 지수(Perturbational Complexity Index, PCI)**는 경두개 자기 자극(TMS)으로 피질을 교란한 뒤 EEG 반응의 압축 불가능성을 계산한다. 각성 시 PCI는 높고, NREM 깊은 수면에서는 낮으며, 마취 상태에서는 더 낮아진다(Casali et al., 2013). 식물 상태 환자와 최소 의식 상태 환자 사이에서도 PCI가 의미 있게 달라진다. 문제는 이 수치가 *왜* 의식 수준과 대응하는지다.

## ⚙️ 깊은 동형성

**분석 1 — 정밀도 가중 예측 오차와 의식 수준**

예측처리 이론(Friston, 2010; Hohwy, 2013)에서 각 위계 수준의 예측 오차는 아래로 전파되기 전에 정밀도(precision) $\pi$로 가중된다.

$$
\varepsilon_\ell = \pi_\ell \cdot (x_\ell - \hat{x}_\ell)
$$

여기서 $\ell$은 위계 수준, $x_\ell$은 해당 수준에서 수신된 신호, $\hat{x}_\ell$은 상위 수준이 내려보낸 예측이다. 정밀도 $\pi_\ell$은 신호 대 잡음비의 역할을 하며, 신경생물학적으로는 콜린성·도파민성 신경조절물질 시스템이 조율한다. $\square$

**분석 2 — 각성 상태: 정밀도-균형 위계**

각성 시에는 감각 입력의 정밀도가 높게 유지되어 하위 수준의 예측 오차가 위계 전체로 효율적으로 전파된다. 동시에 전두-두정 영역이 하향 예측(top-down prediction)을 강하게 공급하여 오차를 선택적으로 억제한다. 결과적으로 외부 세계와의 정합성이 높은 '통제된 환각(controlled hallucination)'이 형성된다(Seth, 2021). 이것이 우리가 경험하는 일상적 의식이다. $\square$

**분석 3 — 마취: 하향 예측의 붕괴**

프로포폴이나 케타민 같은 마취제는 피질 간 장거리 연결, 특히 전두엽에서 내려오는 하향 신호를 불균형적으로 억제한다는 증거가 있다(Mashour et al., 2020). 하향 예측이 없어지면 하위 수준은 오차를 제어할 참조점을 잃는다. 역설적으로 하위 피질에서는 예측 오차 신호가 *폭증*하지만, 위계를 통합적으로 조율하는 메커니즘이 사라진 상태이므로 그 오차들은 일관된 표상으로 수렴하지 못한다. PCI가 낮아지는 이유는 피질이 자극에 단순하게(압축 가능하게) 반응하기 때문이며, 이는 위계적 예측 연쇄가 짧아진 것과 직결된다. $\square$

**분석 4 — NREM 수면: 감각 분리 상태의 내부 모델 구동**

깊은 NREM 수면에서 시상-피질 게이트가 닫히면서 외부 감각 입력의 정밀도가 크게 낮아진다. 생성 모델(generative model)은 계속 작동하지만, 외부 정보와 단절된 채 내부 선험 확률(prior)만으로 구동된다. 느린 진동(slow oscillation, ~0.75 Hz)은 피질 신경망이 ON 상태(예측 오차 전파)와 OFF 상태(재충전·시냅스 항상성) 사이를 반복하는 패턴이다. 이 상태에서 의식이 없거나 매우 단편적인 것은 위계 통합이 이루어지지 않기 때문이다. REM 수면에서는 외부 차단은 유지되지만 내부적으로 위계 전체가 활성화되어 꿈이라는 '내부 예측 시뮬레이션'이 형성된다.

$\square$ 따라서 의식 수준은 예측 위계에서 *하향 예측과 상향 오차가 얼마나 넓고 깊게 상호작용하는가*의 함수로 볼 수 있다. 이 상호작용의 폭과 복잡성이 높을수록 PCI는 높고 의식 수준도 높다.

## 🧪 수렴 증거

### 실험 1: 상태별 예측 오차 전파 시뮬레이션

아래 코드는 3층 위계 모델에서 각 의식 상태를 정밀도 파라미터로 근사하고, 감각 입력에 대한 예측 오차 전파 패턴을 시뮬레이션한다.

```python
import numpy as np
import matplotlib.pyplot as plt

rng = np.random.default_rng(42)

# 위계 수준 수
N_LEVELS = 3
T = 200  # 시간 스텝

# 상태별 파라미터: (감각 정밀도, 하향 예측 강도, 내부 선험 강도)
states = {
    "각성 (Wakefulness)":    dict(sensory_pi=1.0, topdown=0.8, prior=0.3),
    "NREM 수면":              dict(sensory_pi=0.1, topdown=0.6, prior=0.9),
    "마취 (Anesthesia)":     dict(sensory_pi=0.8, topdown=0.05, prior=0.5),
}

def simulate_hierarchy(T, sensory_pi, topdown, prior, n_levels=N_LEVELS):
    """
    단순 위계 예측 오차 모델.
    각 수준에서:
      prediction[l] = topdown * state[l+1] + prior * prior_mean
      error[l]      = sensory_pi * (input[l] - prediction[l])
      state[l]      = state[l] + lr * error[l]
    """
    lr = 0.1
    prior_mean = 0.0
    states_arr = np.zeros((T, n_levels))
    errors_arr = np.zeros((T, n_levels))

    # 외부 감각 입력 (저주파 신호 + 노이즈)
    t = np.arange(T)
    sensory_input = np.sin(2 * np.pi * t / 40) + 0.3 * rng.standard_normal(T)

    state = np.zeros(n_levels)

    for i in range(T):
        inp = sensory_input[i]
        new_state = state.copy()
        for l in range(n_levels):
            if l == 0:
                raw = inp
            else:
                raw = state[l - 1]

            if l < n_levels - 1:
                pred = topdown * state[l + 1] + prior * prior_mean
            else:
                pred = prior * prior_mean

            error = sensory_pi * (raw - pred)
            new_state[l] = state[l] + lr * error
            errors_arr[i, l] = error

        state = new_state
        states_arr[i] = state

    return states_arr, errors_arr

fig, axes = plt.subplots(len(states), 2, figsize=(12, 9), sharex=True)
fig.suptitle("의식 상태별 위계 예측 오차 전파 시뮬레이션", fontsize=14)

for row, (label, params) in enumerate(states.items()):
    st, err = simulate_hierarchy(T, **params)
    ax_s, ax_e = axes[row]

    for l in range(N_LEVELS):
        ax_s.plot(st[:, l], label=f"수준 {l+1}", alpha=0.8)
    ax_s.set_title(f"{label} — 상태 표상")
    ax_s.set_ylabel("활성화")
    ax_s.legend(fontsize=8)

    for l in range(N_LEVELS):
        ax_e.plot(err[:, l], label=f"수준 {l+1}", alpha=0.8)
    ax_e.set_title(f"{label} — 예측 오차")
    ax_e.set_ylabel("오차 크기")
    ax_e.legend(fontsize=8)

axes[-1, 0].set_xlabel("시간 스텝")
axes[-1, 1].set_xlabel("시간 스텝")
plt.tight_layout()
plt.savefig("consciousness_levels_simulation.png", dpi=150)
plt.show()
print("시뮬레이션 완료: consciousness_levels_simulation.png 저장됨")
```

이 코드를 실행하면 각성 상태에서는 세 수준 모두 입력과 잘 동기화된 표상을 형성하고 오차가 작게 유지되는 반면, 마취 상태에서는 하향 예측이 거의 없어 하위 수준의 오차가 크게 진동하고 상위 수준은 거의 움직이지 않는 패턴이 나타난다. NREM 상태는 감각 입력 자체가 약하게 반영되며, 내부 선험이 표상을 지배한다.

### 실험 2: PCI와 위계 복잡도의 대응 — 반례와 경계

**수렴 증거**: Casali et al.(2013)은 26명의 피험자에서 TMS-EEG를 이용해 PCI를 측정하고, 각성·REM·NREM·케타민 마취·프로포폴 마취·식물 상태·최소 의식 상태 조건이 PCI 값에 따라 신뢰 가능하게 분리됨을 보였다. 각성 PCI ≈ 0.44–0.70, NREM PCI ≈ 0.18–0.35, 프로포폴 마취 PCI ≈ 0.12–0.25.

**반례 1 — 케타민**: 케타민은 NMDA 수용체를 차단하여 마취 효과를 내지만, PCI를 각성 수준에 가깝게 유지시키기도 한다. 이는 케타민이 전반적인 위계 억제가 아니라 억제성 인터뉴런을 선택적으로 차단해 피질 흥분성을 오히려 높이기 때문이다. 자유에너지 관점에서는 하향 예측이 유지되지만 *잘못된* 정밀도 설정 아래에서 작동하는 상태로 해석할 수 있다.

**반례 2 — 고기능 자폐 스펙트럼**: Lawson et al.(2017)은 자폐 스펙트럼 장애에서 감각 과민이 정밀도 조율 이상, 즉 감각 입력에 과도한 $\pi$가 부여되는 상태로 설명될 수 있다고 주장했다. 의식 수준은 정상이지만 위계 내 정밀도 분포가 다르다. 이는 의식 수준과 의식 내용이 분리될 수 있음을 보여준다.

**경계 조건**: IIT(Tononi, 2004)는 $\Phi$(파이)라는 통합 정보량으로 의식을 정의하며, 이는 예측처리 프레임과 부분적으로 겹친다. 그러나 IIT는 정보 *통합*을, 자유에너지 이론은 *최소화*를 핵심으로 삼으므로 두 이론은 PCI 데이터에 대해 다른 설명 경로를 제공한다. 현재 데이터만으로는 두 이론을 결정적으로 구분하기 어렵다.

## 🌉 간극의 위치

- ✅ PCI는 의식 수준의 신뢰할 수 있는 경험적 지표로 다수 집단에서 검증되었다.
- ✅ 마취제의 전두-두정 하향 연결 억제 메커니즘은 전기생리학적으로 부분 지지된다(Mashour et al., 2020).
- ✅ NREM의 감각 게이팅과 느린 진동은 시상-피질 시스템의 신경과학으로 잘 설명된다.
- ❌ 정밀도 파라미터 $\pi$를 뉴런 수준에서 직접 측정하는 방법이 아직 확립되지 않았다.
- ❌ 자유에너지 최소화가 *왜* 주관적 경험을 동반하는지에 대한 설명은 없다(어려운 문제, 다음 문서 참조).
- ❌ 케타민, 환각제(실로시빈) 등 비전형 마취제 상태에서 이론의 예측력이 제한적이다.
- ❌ PCI가 높더라도 의식이 있다고 단언하기 위한 인과적 기준이 아직 논쟁 중이다.

## 🧬 원리의 회수

자유에너지 최소화 원리는 단일한 수식($F = \mathbb{E}_q[\log q(z) - \log p(x,z)]$)으로 각성·수면·마취를 하나의 스펙트럼 위에 놓는다. 의식의 수준은 생성 모델의 위계 깊이와 정밀도 조율 상태의 함수이며, 신경조절물질 시스템이 그 조율의 핵심 손잡이다. PCI는 이 위계적 복잡도를 외부에서 탐침하는 도구로, 이론과 경험 사이에 다리를 놓는다.

의식의 수준(level)과 내용(content)의 구분도 이 틀에서 자연스럽게 나온다. 수준은 예측 위계가 *작동하는 범위*이고, 내용은 그 위계가 *무엇을 표상하느냐*다. 마취 상태에서는 위계 자체가 붕괴하므로 수준이 낮다. 꿈에서는 위계가 작동하지만 외부 입력 없이 내부 선험으로만 구동되므로, 수준은 어느 정도 유지되지만 내용은 현실과 단절된다.

이 원리가 δ = actual − predicted → update라는 보편 형식으로 환원된다는 점을 기억하라. 의식 수준이란 이 업데이트 과정이 몇 겹의 위계에 걸쳐 얼마나 정밀하게 이루어지느냐의 척도다.

## 🪞 1인칭

마취에서 깨어날 때의 경험을 떠올려보자. 빛이 서서히 켜지듯 의식이 돌아오는 것이 아니다. 처음에는 소리가 희미하게 들리지만 의미가 없고, 그 다음 통증이 감각으로 오지만 아직 "나의" 통증이 아니다. 마지막으로 어디에 있는지, 왜 여기 있는지가 맞물리면서 비로소 일관된 자아 감각이 돌아온다. 이것은 예측 위계가 하위 수준부터 차례로 복구되는 과정처럼 느껴진다.

깊은 수면에서 막 깨어났을 때도 비슷한 단계가 있다. 처음 몇 초 동안은 자신이 누구인지, 아침인지 밤인지도 불분명하다. 이 짧은 혼란이 바로 생성 모델이 외부 감각 입력을 재통합하면서 내부 선험을 교정하는 순간이다. 우리는 매일 밤 의식의 수준을 낮추었다가 아침에 다시 끌어올린다. 자유에너지 이론은 이 친숙한 경험에 수식을 부여한다.

## 📐 예측·반증

**예측 1**: 프로포폴 마취 유도 중 전두엽에서 후두엽으로 향하는 하향 인과 흐름(Granger causality)이 먼저 감소하고, 그 이후 PCI가 역치 이하로 떨어질 것이다.
*반증 조건*: 후두엽-전두엽 상향 흐름이 하향 흐름보다 먼저 차단되거나, 두 방향이 동시에 같은 속도로 감소한다면 이 예측은 기각된다.

**예측 2**: 정밀도 조율 약물(예: 아세틸콜린에스테라제 억제제)을 투여하면 경증 마취 상태에서 PCI가 각성 방향으로 부분 회복될 것이다.
*반증 조건*: 신경조절물질 조작이 PCI에 유의미한 영향을 미치지 않는다면, 정밀도-의식 수준 연결 가설이 약화된다.

**예측 3**: 훈련된 명상가의 깊은 집중 상태(samatha)에서는 감각 입력 정밀도가 의도적으로 낮아지고 상위 위계가 지배하므로, PCI는 NREM과 각성 사이의 중간값을 보일 것이다.
*반증 조건*: 명상 상태에서 PCI가 일반 각성과 통계적으로 차이가 없다면, 의식 수준의 자발적 변조 가설이 도전받는다.

## 🤔 열린 질문

자유에너지 최소화와 PCI는 의식이 *있는지 없는지*를 (불완전하게나마) 측정하는 도구를 제공한다. 그러나 이 측정이 포착하지 못하는 것이 있다. 의식이 있다고 할 때, 그것은 *어떤 느낌*인가? 정밀도가 높고 위계가 깊다는 사실이 왜 무언가를 경험한다는 사실을 만들어내는가? 이것이 다음 문서가 다루는 '어려운 문제(Hard Problem)'다.

또한, 현재까지의 모든 측정은 제3자 관점에서 이루어진다. PCI는 외부에서 탐침한 결과이고, 자유에너지는 수학적 형식이다. 1인칭 경험의 질적 속성—빨간색의 붉음, 고통의 아픔—은 이 형식 체계 어디에 위치하는가? → [다음 문서: 예측처리가 설명하지 못하는 것](./03-hard-problem.md)

---

> 🧩 **Principle** — 의식의 수준은 예측 위계에서 하향 예측과 상향 오차가 정밀도 조율 아래 얼마나 깊고 넓게 상호작용하는가의 함수다.
> 🌉 **Boundary** — 이 원리는 의식의 유무와 수준을 PCI라는 경험적 지표로 연결하지만, 주관적 경험의 질적 속성(qualia)이 왜 발생하는지는 설명하지 못한다.
> 🪞 **Experience** — 마취에서 깨어나거나 잠에서 깨는 순간, 우리는 예측 위계가 단계적으로 복구되는 과정을 1인칭으로 살아낸다.

## 📝 연습문제

**기초.** 자유에너지 $F$를 KL 발산과 로그 주변우도로 분해하라. 각 항이 의식 수준 모델에서 어떤 역할을 하는지 한 문장씩 설명하라.

<details><summary>해설</summary>

$F = D_{\mathrm{KL}}[q(z)\|p(z|x)] - \log p(x)$에서 KL 발산 항은 믿음 $q(z)$가 실제 사후분포 $p(z|x)$로부터 얼마나 벗어나 있는지를 나타낸다. 이 항이 크면 뇌의 생성 모델이 감각 원인을 잘못 추정하고 있다는 뜻이다. $-\log p(x)$는 감각 입력의 놀라움(surprise)이며, 예측이 잘 될수록 작아진다. 의식 수준이 높을수록 두 항 모두를 효율적으로 최소화할 수 있는 위계적 자원이 있다.

</details>

**심화.** NREM 수면과 마취 상태는 모두 PCI가 낮지만 신경 메커니즘이 다르다. 자유에너지 프레임에서 두 상태의 차이를 정밀도 파라미터($\pi_{\mathrm{sensory}}$, $\pi_{\mathrm{topdown}}$)를 이용해 형식적으로 구분하라.

<details><summary>해설</summary>

NREM 수면: $\pi_{\mathrm{sensory}} \approx 0$, $\pi_{\mathrm{topdown}}$은 중간값 유지. 감각 게이트가 닫혀 있으므로 외부 입력의 정밀도가 거의 0에 가깝고, 생성 모델은 내부 선험만으로 구동된다. 마취: $\pi_{\mathrm{topdown}} \approx 0$, $\pi_{\mathrm{sensory}}$는 부분적으로 유지. 프로포폴은 전두-두정 하향 신호를 선택적으로 차단하므로 감각 입력은 들어오지만 이를 통제할 하향 예측이 없다. 두 상태 모두 위계 통합이 깨지므로 PCI는 낮아지지만, 깨지는 방식이 다르다: NREM은 입력 차단형, 마취는 하향 붕괴형이다.

</details>

**논문 비평.** Mashour et al.(2020) "Conscious Processing and the Global Neuronal Workspace Hypothesis"(Neuron)를 자유에너지 이론의 관점에서 비판적으로 평가하라. 두 이론이 마취 의식 소실에 대해 공유하는 예측과, 서로 다른 예측을 각각 하나씩 제시하라.

<details><summary>해설</summary>

공유 예측: 전두-두정 장거리 연결의 억제가 의식 소실의 핵심 메커니즘이라는 점에서 전역 작업공간 이론(GWT)과 자유에너지 이론은 일치한다. GWT는 이를 '방송 네트워크'의 붕괴로, FEP는 '하향 예측 신호 소실'로 표현하지만 신경 기반은 겹친다. 차이 나는 예측: GWT는 의식의 임계 현상(all-or-none transition)을 예측하는 경향이 있는 반면, FEP 기반 모델은 정밀도 파라미터가 연속적으로 변하므로 의식 수준이 점진적 스펙트럼을 이룬다고 예측한다. 또한 GWT는 시상-피질 루프의 동기화를 강조하지만, FEP는 정밀도 조율이 핵심이므로 신경조절물질 조작만으로도 의식 수준이 변할 것이라고 예측하는 차이가 있다.

</details>
